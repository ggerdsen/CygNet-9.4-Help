Using a Search Box in Run Mode

The Search Box has numerous run-mode options:

  • You can dynamically resize your screen and any controls within. See Resizing Screens for more information about resizing Canvas screens.

To search for a facility

If the search box has been configured to search for a facility, perform the following steps:

  1. Type a full or partial string into the search box to search for a facility in your CygNet system.
  2. Press Enter or single click the (Search) icon.
  3. Canvas will query against the configured Facility service, filter rule, and facility properties. The search results will be presented sorted alphabetically by facility ID.
  4. Click on the desired facility in the search results. Depending on the configured Selection action one of the following actions will occur:
    1. Facility sender — When a facility is selected, the search box functions as a facility sender and sends the selected facility to any controls on the screen configured to receive it.
    2. Navigate — When a facility is selected, the screen will navigate to the selected facility's associated screen. The navigation mode is determined by the Facility navigation mode property setting on the screen displaying the search box.
    3. Script — When a facility is selected, the scripted event will be fired.
To perform a custom search

If the search box has been configured to search a database, .csv file, or other object, perform the following steps:

  1. Type a full or partial search term in the search box.
  2. Press Enter or single click the (Search) icon.
  3. A custom script method is fired to query the specified object, and any matching terms are returned in the search results list.
  4. Click on the desired item in the search results.
  5. A scripted event will be fired: for example, popping a message, passing the selected value to another control(s), linking to another screen, etc.

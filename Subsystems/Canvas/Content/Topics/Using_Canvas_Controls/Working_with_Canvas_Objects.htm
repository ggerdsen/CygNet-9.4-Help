Working with Canvas Controls and Objects

Canvas objects are graphical tools that can be used to view CygNet information, as well as to control actions. Each object can be configured by using the Properties pane and via script.

Adding Objects to the Screen

The objects available to be added to the Canvas screen are shown in the Toolbox pane. There is a wide variety of controls available.

To add an object to the screen you must be in design mode. Most objects can be added to the screen by clicking on the control in the Toolbox pane and dragging it to the screen. Objects on the screen are stacked according to the order in which they were added to the screen. Use the alignment tools in the Settings pane to change the object stacking order.

Once you have added an object to the screen you must configure its properties.

Selecting Objects

In design mode, to configure the properties of an object, move it, resize it, or delete it you must first select it by clicking on the object with the mouse. When a single object is selected its handles are black. Whenever one or more objects is selected the Settings pane icon () will display to the right of the objects.

To select an object
  1. Click on the object or lasso the object using the mouse. The drag area must be completely outside of the boundary of the object.
To select multiple objects
  1. Lasso the objects using the mouse. The drag area must be completely outside of the boundary of all objects.

Note: When selecting multiple controls, non-selected controls will fade to highlight the selected ones.

To assign objects to a layer
  1. Use the Layers property to assign multiple objects to the same layer to show, hide, and edit controls on one layer without affecting controls on another layer.
  2. Use the Layers pane to show, hide, and refresh layers available for a screen.

Manipulating Objects

The following sections describe how to manipulate objects that have been added to the Canvas screen.

To copy and paste objects

When you copy an object all properties are copied except for the Name property. Each object is assigned a unique Name.

To copy and paste an object
  1. Select the object(s) to copy/cut and paste.
  2. Press Ctrl+C (copy) or Ctrl+X (cut) and Ctrl+V (paste), or select (Settings), then use the (copy), (cut) and (paste) buttons.
To delete objects

Any object can be deleted from the screen.

To delete an object
  1. Select the object(s) to be deleted.
  2. Press the Delete key, or select (Settings), then click (delete).
To group objects

Objects can be grouped and ungrouped in design mode. Doing so can ensure that objects you want to keep together are not accidentally repositioned relative to each other. It also lets you treat a set of objects as though it were one object, in the sense that you can select the entire group "object" with one click and then move it, resize it, or delete it.

To group objects
  1. Select all objects you want to group.
  2. Select (Settings), then click (group).
To ungroup objects
  1. Select all objects you want to ungroup.
  2. Select (Settings), then click (ungroup).
To order objects

When you add objects to the Canvas screen, they are stacked in the order they were added. Sometimes you must modify the stack order to get the desired visual results. The Settings pane has options for you to change the z-order of selected objects. front and back send the object to the top or the bottom of the stack, respectively. forward moves the object up one position. backward sends the object down one position. You can use the commands on multiple objects at a time.

To send an object to the back
  1. Select the object you want in the back.
  2. Select (Settings), then click (back).
To send an object backward
  1. Select the object you want to move backward.
  2. Select (Settings), then click (backward).
To bring an object forward
  1. Select the object you want to move forward.
  2. Select (Settings), then click (forward).
To bring an object to the front
  1. Select the object you want in the front.
  2. Select (Settings), then click (front).
To align objects

You must select two or more objects to align or distribute.

To align objects along the right side
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (right). The objects will be aligned along the right side of the right-most object.
To align objects along the center line
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (center). The objects will be aligned along the center line.
To align objects along the left side
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (left). The objects will be aligned along the left side of the left-most object.
To align objects along the top side
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (top). The objects will be aligned along the top side of the upper-most object.
To align objects along the middle line
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (middle). The objects will be aligned along the mid-line.
To align objects along the bottom side
  1. Select the objects you want to align by lassoing around the objects with the mouse.
  2. Select (Settings), then click (bottom). The objects will be aligned along the bottom side of the lower-most object.
To size and position objects

Object size and position are determined by four properties: Height, Width, X, and Y. Height and Width correspond to the dimensions of an object in pixels. X and Y correspond to the distance of an object’s left edge from the left edge of the screen in pixels, and the distance of an object’s top edge from the top edge of the screen in pixels, respectively. All objects include these properties, because all objects can be defined by a boundary rectangle, regardless of shape.

To resize an object

Once you place an object on the screen you can resize it.

To resize an object by dragging
  1. Select the object to be resized.
  2. Position the cursor over one of the object’s side or corner handles. These are shown as black squares along the object’s perimeter. The cursor will change to a double-ended arrow.
  3. Click and drag the handle to resize the object then release the mouse. When you resize the object, it will snap to the grid. The X and Y coordinates will display as you resize the object.
To resize an object one pixel at a time
  1. Select the object to be resized.
  2. Select (Settings), then Size, then:
    • — Click the up-down arrows to expand the size of the object(s) horizontally by one pixel per click or type a new value. The Width property on the Properties pane will change accordingly.
    • — Click the up-down arrows to expand the size of the object(s) vertically by one pixel per click or type a new value. The Height property on the Properties pane will change accordingly.
To move an object

Once you place an object on the screen you can move it.

To move an object by dragging
  1. Select the object to be moved.
  2. Position the cursor over the object’s interior. The cursor will change to a cross with arrow ends.
  3. Click and drag the object to the new location. When you move the object, it will snap to the grid. The X and Y coordinates will display as you move the object.
To move an object
  1. Select the object to be moved.

  2. Use the up, down, left, and right arrow keys to nudge a control by 1 pixel. Arrow moves a control by 1 pixel. Ctrl+Arrow moves a control by 10 pixels. Ctrl+Shift+Arrow moves a control by 100 pixels.

  3. Select (Settings), then Size, then:
    • — click the up-down arrows to move the object(s) along the horizontal axis by one pixel per click or type a new value. The X property on the Properties pane will change accordingly after you click away and then reselect the object.
    • — click the up-down arrows to move the object(s) along the vertical axis by one pixel per click or type a new value. The Y property on the Properties pane will change accordingly after you click away and then reselect the object.

Configuring Object Properties

The Canvas screen and each object on screen has its own set of properties, which define the instance of the object. Properties are displayed in the Properties pane. Most properties are typically defined in design mode; however, they can also be set using script.

Categorizing, Sorting, and Finding Properties

You can click (Categorize), click (Alphabetical within Category) or click (Search) within the properties using the features located immediately above the properties list. The search box is useful to help locate a property when an object contains many properties. For clarity, when you enter a search term, items matching your entry string remain displayed and non-matching items are temporarily hidden. Clear the search box to display the full properties list again.

You can expand and collapse each property category by clicking or next to the group name or simply clicking anywhere in the category heading.

Navigation Pane — Pan and Zoom

The Canvas screen includes a navigation pane which gives you the ability to easily pan around, zoom in and out, and auto fit the Canvas screen in both design and run mode. Pan and zoom are always enabled in design mode. These features are optional in run mode.

Display of the navigation pane in run mode is configurable on the Screen and Object properties pane. If configured to display the pane is also visible in Canvas View.

Navigation pane — pan and zoom

Pan and zoom functionality is controlled by two properties on the Screen and Object controls:

  • Is pan zoom control visible — The Is pan zoom control visible check box indicates whether to display the pan zoom control on the screen in run mode. Pan and zoom are always enabled in design mode; the control is optional in run mode.
  • Is pan/zoom enabled — The Is pan/zoom enabled check box indicates whether the user can pan around or zoom in and out of the screen. Pan and zoom are always enabled in design mode; the control is optional in run mode.Note: Enabling this will negatively impact screen load performance and disable the ability to dynamically add controls via script.
To use the navigation pane
  • Click (Open) to access the navigation pane.
  • Click on the zoom slider to zoom in and out of the area you wish to examine.
  • Use the red Zoom thumbnail to pan around the area.
  • Click (Close) to close the navigation pane.
  • Click (Auto fit) to return to 100% size.

The navigation pane is also available on the Relative Facility Definition Diagram.

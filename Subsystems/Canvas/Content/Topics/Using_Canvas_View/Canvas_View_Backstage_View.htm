Canvas View Backstage View

 

 

The Canvas Backstage view is a set of pages where you can manage and configure your Canvas View experience as a whole, such as opening and saving screens and objects, launching and updating the online help, configuring logging, viewing about and copyright information, and exiting the application.

Click File (or ) to access the Canvas View Backstage view which provides the following options, some of which can also be accessed from the application Toolbar:

  • Open — Open a previously created screen or object from your CygNet folder location, from the Recent Screens list, from a local or network file system, or from a CygNet APPS or BSS folder
  • Save As — Save the current screen to a different local or network file system location, or a CygNet BSS folder, with a new file name
  • Help — Launch the CygNet Help and configure logging settings:
    • Launch Help — Launch the CygNet Help
    • Log files folder — Open the folder where the Canvas log files are stored
    • Logging level — Configure the desired logging level for the Canvas application
  • About — View file, product version, and copyright information about your Canvas View software
  • Exit — Exit the current screen and the Canvas View application.

Click (back) to return to the main Canvas View workspace.

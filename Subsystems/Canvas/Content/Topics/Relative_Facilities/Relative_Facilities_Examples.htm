Relative Facility Definition Examples

The following diagrams demonstrate various relative facility definitions and concepts.

Keep the following in mind:

  • When configuring a relative facility definition, specify a facility attribute for the linked facility
  • All links must resolve to a single facility
  • Multiple links from a single facility/definition are not supported
  • Multiple hops are supported

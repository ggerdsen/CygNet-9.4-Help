Using Canvas

Start the Canvas application and begin to experiment to see what you can do with its components and features. Make a screen, add a couple of controls, script some changes, link in some assemblies, and play with it!

Use Canvas to create and edit customized user screens for controlling, monitoring, and displaying information based on the data and devices in your CygNet system. After screen creation, all functionality presented to a user on a Canvas screen will be available to use as-is, or to copy and design variations.

The Canvas application includes several controls with access to CygNet current values. Another exciting aspect of Canvas is that you can also use its custom control framework to create your own controls to include and use in Canvas.

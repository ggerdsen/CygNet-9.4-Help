Canvas User Interface

The Canvas application contains a variety of user interface components and features to help you create highly customized user screens you can then use to interact with the data managed by your CygNet SCADA system.

The primary features of the Canvas user interface window include the following elements:

  • Toolbar — The main toolbar that helps you easily access all available functionality for Canvas
  • Workspace — The saved window layout containing your screen creation environment and its component panes
  • Backstage View — This view contains a menu of options to access various pages to help you perform commands, and manage and view information about your current Canvas application
  • Status Bar — Shows along the base of the application's window and contains the current user and the ambient domain

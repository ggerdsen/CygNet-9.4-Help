Tag Chooser Pane

On the Canvas toolbar, click (Display tag chooser) to display a Tag Chooser pane in your application window or to hide the currently open pane. The tag chooser displays a hierarchical tree of CygNet facilities and points. Select any item in the tree and drag it to other controls on your screens to populate those control with CygNet data. The tag chooser hierarchy can be organized by facility attributes or group nodes.

Note: The tag chooser is also available as a control which can be added to any Canvas screen. See Configuring the Tag Chooser for more information.

Choosing How to View the Tag Chooser Pane

In the pane title bar, you can select how to display the pane by default (whether to float, dock, auto-hide, or hide it), pin the pane in place, or close the pane. Unpinning (or hiding) a pane when not in use gains extra space for viewing your current screen or another open pane. Closing a pane can also occur from the pane title bar. The following icons are available:

  • Click (menu chooser) to display a context menu from where you can float, dock, auto hide, or hide the pane.
  • Click (pin) on the pane title bar to hide the pane (or "pinned" to the edge of the application window).
  • Click (x) on the pane title bar to close the pane.

See Customizing the Workspace for more information.

Configuring the Tag Chooser Pane by Facility Attributes

The tag chooser hierarchy is organized by the global attributes configured in your CygNet Facility service. You can further limit the tag chooser by any CygNet filter attribute set up for your system.

The Show configuration property must be checked in design mode or order to see the Configuration field in run mode.

  1. Click (Configuration) to open the tag chooser configuration fields.

Tag Chooser — Configure facilities
  1. Specify the appropriate Facility service field. Click to select a Facility service using the Select Service dialog box.
  2. Click or on the Click here to add new item field to expose a drop-down menu of facility attributes. Select a facility attribute from the drop-down menu to organize the tag chooser by that attribute at the highest level.
    1. Continue to add attributes to build the desired hierarchy as many times as necessary.
    2. Once defined, the hierarchy of the tag chooser can be easily reordered by dragging and dropping an item to a new location.
    3. Click to remove any facility attribute from the list.

  3. Use the Facility filter property to limit the data presented in the control to any CygNet filter attribute set up for your CygNet installation. Click to open the Facility Filter dialog box where you can create a custom filter rule. The filter rule is displayed in the property label; you do need to open the Facility Filter dialog box to edit the rule.

Tip: In run mode you must configure the facility filter before configuring the hierarchy levels, in order for the filter to apply to the hierarchy.

  1. The hierarchical tree will dynamically reflect the filtering selected.
  2. Click (Configuration) to close the tag chooser configuration box.

Configuring the Tag Chooser Pane by Group Nodes

The tag chooser hierarchy is organized by the nodes configured in your CygNet Group service. You can further filter the tag chooser by a facility type. This must be a valid facility type in your system.

The Show configuration property must be checked in design mode or order to see the Configuration field in run mode.

  1. Click (Configuration) to open the tag chooser configuration fields.

Tag Chooser — Configure groups
  1. Specify the appropriate Group service field. Click to select a Group service using the Select Service dialog box.
  2. Specify the appropriate group node to use as the Base node for the hierarchy. Click to select the desired node from the Select Group Hierarchy Node dialog box.
  3. Type the name of the facility type on which to filter the hierarchy, in all UPPERCASE letters. The nodes in the hierarchy will correspond to the Facility Type column in the Facility service. This must be a valid facility type in your system.
  4. The hierarchical tree will dynamically reflect the filtering selected.
  5. Click (Configuration) to close the tag chooser configuration box.

Choosing Tags from the CygNet Tag Chooser for a Control

Once you have configured other controls on a Canvas screen, navigate to the desired branch in the navigation tree in the tag chooser, select the desired point, click and drag the point to the receiving control.

Configuring Alarm Colors

Use the Alarm Colors dialog box to configure the color elements of the alarm notifier. The color can be explicitly configured (using <Self>) or it can be sourced from the point state of the associated point (by selecting Point State).

These settings are saved in XML format as part of the global settings file (.gsf) for your Canvas application and applied to all instances of Canvas.

Configure alarm colors

The background color, text color, and the border color for the alarms displayed in the alarm notifier can be shown in a globally set color or shown in their associated point state color.

  1. In the Backstage view, go to Alarm Notifier, and click Alarm colors to open the Alarm Colors dialog box.

Alarm colors configuration
  1. For each color element select the Source and Color.

Background color

The Background color property specifies the color used for the background of the alarm notifier cell. The color can be explicitly configured (using <Self>) or it can be sourced from the point state of the associated alarm.

Options include:

  • Source — Click to reveal two options:
    • <Self> — Select <Self> to explicitly configure the color. When <Self> is selected a Color field and picker is automatically displayed beneath the Source field and must be configured.
    • Point State — Select Point State to specify that the source of the color corresponds to the point state color defined for the point represented in the element. The default point state colors for a CygNet system are defined by the system’s point scheme. When Point State is selected the Color property disappears.
  • Color — Only displays if the Source is <Self>. Click to explicitly specify a color using the color picker.

Text color

The Text color property specifies the color source, or the color used for the text of the alarm notifier cell. The text color is sometimes known as the foreground color. The color can be explicitly configured (using <Self>) or it can be sourced from the point state of the associated alarm.

Options include:

  • Source — Click to reveal two options:
    • <Self> — Select <Self> to explicitly configure the color. When <Self> is selected a Color field and picker is automatically displayed beneath the Source field and must be configured.
    • Point State — Select Point State to specify that the source of the color corresponds to the point state color defined for the point represented in the element. The default point state colors for a CygNet system are defined by the system’s point scheme. When Point State is selected the Color property disappears.
  • Color — Only displays if the Source is <Self>. Click to explicitly specify a color using the color picker.
  1. Once colors are configured, click OK to save the configuration to the global settings file.
  2. Or click Cancel to abort and close the Alarm Colors dialog box.

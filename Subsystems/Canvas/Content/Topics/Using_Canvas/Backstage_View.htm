Canvas Backstage View

Click File (or ) to access the Canvas Backstage view providing the following options, some of which can also be accessed from the application Toolbar:

  • New — Create a new screen or object, and select a script language
  • Open — Open a previously created screen or object from your CygNet folder location, from the Recent Screens list, from a local or network file system, or from a CygNet APPS or BSS folder
  • Save — Save the current screen to a local or network file system location, or to a CygNet BSS folder, with the same file name
  • Save As — Save the current screen to a different local or network file system location, or a CygNet BSS folder, with a new file name
  • Plugins — Add or access Canvas plugin assemblies containing additional controls
  • Settings — Access and configure general and global settings for the Canvas application
    • Settings folder — Click to open the folder where the Canvas settings files are stored
    • Object library — Click to open the Object Library configuration dialog box
    • Facility navigation — Click to open the Facility Navigation configuration dialog box
    • Relative facilities — Click to open the Relative Facilities configuration dialog box
    • Style sheets — Click to open the Style Sheets configuration dialog box
    • Color palettes — Click to open the Color Palettes configuration dialog box
    • General Settings
      • Default script language — Configure the default script language used for new screens
      • Default date format — Configure the default date and time format for timestamps displayed on screens
    • Global settings — Configure the location of your global settings file, or create a new global settings file
  • Alarm Notifier — Access and configure global settings required for the Alarm Notifier
    • Alarm service — Configure the Common Alarm Services (CAS) from where the alarm notifier will retrieve its alarm information
    • Alarm filter — Click to open the Alarm Filter configuration dialog box to create filtering rules to limit the alarms displayed
    • Alarm sounds — Click to open the Alarm Sounds configuration dialog box to configure the alarm priority ranges and associated sound files
    • Alarm colors — Click to open the Alarm Colors configuration dialog box to configure the background text color displayed in the alarm box
    • Alarm attributes — Configure the attributes for the three items that appear in the alarm box on the Alarm Notifier
    • Alarm screen file — Configure the path to the custom alarm screen that will display when the total number of unacknowledged alarms in the alarm banner is clicked
  • Find/Replace — Find and replace text strings in your screen's properties, control properties, script, and/or metadata
  • Help — Launch the CygNet Help and configure logging settings
    • Launch help — Launch the CygNet Help
    • Log files folder — Open the folder where the Canvas log files are stored
    • Logging level — Configure the desired logging level for the Canvas application
  • Feedback — Submit feedback via comments, survey, or log files about your user experience with Canvas
  • About — View file, product version, and copyright information about Canvas
  • Exit — Exit the current screen and the Canvas application. If you have unsaved screens you will see a warning dialog box allowing you to save your screen(s) and reminding you that you need to manually save new screens.

Click (back) to return to the main Canvas workspace.

Value Bounds

Value bounds configuration
  1. Click to reveal a set of options where you can specify the source and value of the lower and upper bound.
  2. For the Lower value bound, select the Source.
  3. If you want to use an explicit value, select <Self> as the source.
    1. Type a value in the Value field.
  4. If you want to use an application-specific attribute of the associated point, select Point Attribute as the source.
    1. Select the attribute from the drop-down list. Options include Indexed 1-3, General 1-3, or Table Drive 1-3.
    2. In CygNet Explorer, use the Point Editor > Application Properties page to configure the value or table, as appropriate.
  5. Repeat for Upper value bound.

CygNet Help: For more information about configuring point application properties, see the Points section in the CygNet Help:

  • Points > Point Editor > Point Application Properties

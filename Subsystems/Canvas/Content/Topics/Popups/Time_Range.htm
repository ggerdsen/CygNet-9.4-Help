Time Range

Use the Configure Time Range dialog box to enable live data updates, set the live update rate, the starting time for a trend, whether to use relative or absolute dates, and the date range.

Live Updates

Configure Time Range
Enable live updates
  1. Click Enable live updates to request new data at a specified interval allowing the chart to trend data at a real-time pace.
  2. Set the Update rate to specify the number of milliseconds to wait before an auto-updated trend refreshes.
  3. Set the Earliest value to indicate the starting date for a trend; that is, the amount of time in the past from when to start the trend.
  4. Select the Earliest units to specify the unit of time in the past from when to start the trend. The options include: Seconds, Minutes, Hours, Days, and Years.
  5. Click Close to close the dialog box and save your settings.
Relative Dates

Configure Time Range
Use relative date
  1. Click Use relative date to use relative dates and time for the chart.
  2. Set the Start date by specifying a wildcard string defining the amount of time by which to adjust the start date and time.
  3. Set the End date by specifying a wildcard string defining the amount of time by which to adjust the end date and time.
  4. Click Close to close the dialog box and save your settings.
Absolute Dates

Configure Time Range
Use absolute date
  1. Uncheck Enable live updates and Use relative date.
  2. Set the Start date by specifying the start date and time for a control for an absolute date and time view window.
  3. Set the End date by specifying the end date and time for a control for an absolute date and time view window.
  4. Click Close to close the dialog box and save your settings.

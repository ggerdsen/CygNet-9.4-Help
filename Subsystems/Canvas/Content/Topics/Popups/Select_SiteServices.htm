Select SiteServices

Use the Select Service dialog box to select one or more services for the search box to use.

Select SiteServices
  1. Select a SiteService from the list of available services in the ambient domain.
  2. Ctrl+Click or Shift+Click to select multiple services.
  3. Click Refresh to update the list of services.
  4. Click OK to accept the selection and close the dialog box.
  5. Click Cancel to close the dialog box without making a selection.

Event Argument Type

Some script events use custom objects to pass in additional information that may be helpful, but Canvas does not currently expose that type of custom object in script. For example, for a Grid.DoubleClickCell event, the EventArgs object can be added to transform the custom object type created. The syntax is as follows:

private void CygNetGrid1_DoubleClickCell(object sender, EventArgs args)

Without that transformation, the user wonâ€™t be able to get at the extra information provided. For example,

private void CygNetGrid1_DoubleClickCell(object sender, EventArgs args)

{

var selChange = args as Canvas.Controls.Grid.CygNetGridCellDoubleClickEventArgs;

}

In the example above the custom object of type 'EventArgs' is turned into an object of type 'Canvas.Controls.Grid.CygNetGridCellDoubleClickEventArgs.'

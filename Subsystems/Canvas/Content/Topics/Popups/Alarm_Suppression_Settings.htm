Alarm Suppression Settings

Use the Alarm Suppression Settings dialog box to configure when alarm suppression will end for all alarms on the selected point.

Alarm Suppression Settings dialog box
  1. Right-click the alarm and select Alarm suppression... to suppress an alarm.
  2. The Alarm Suppression Settings dialog box will list the Point Tag and Point State for the selected alarm.
  3. Click to enable Suppress all alarms for this point.
  4. Click to enable one or both suppression ending conditions.
  5. Configure a date for Condition Two, if selected.
  6. Click Apply.
  7. Click Yes to the confirmation message. Another message box appears giving the status of the suppression settings.
  8. Click Done.

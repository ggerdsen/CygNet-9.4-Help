Chart Series Type

Select the desired series type from the drop-down list. The following table shows examples of the series type options.

Type Example

Bar

A bar chart visualizes data as a set of rectangular bars, with the length of each bar representing the value of each data point. Each bar is the color assigned to the series.

When adding a series showing custom non-CygNet data in a Bar chart type via script, you must set the color at the point level. This allows for points to be colored differently from each other.

Bar chart

Line

A line chart shows information as a series of data points connected by straight lines. This is the default series type for the chart.

Line chart

Line Area

An line area chart is based on the line chart and shows information as a series of data points connected by straight lines, with the area between the X-axis and the line filled with the line color.

Line Area chart

Spline

A spline chart is a line chart where data points are connected by a smooth curve.

 

Spline chart

Spline Area

A spline area chart is based on the spline chart where data points are connected by a smooth curve, with the area between the X-axis and the line filled with the line color.

Spline Area chart

Step

A step chart is a line chart in which data points are connected by horizontal and vertical lines, looking like a series of steps.

Step chart

Step Area

A step area chart is based on the step chart where data points are connected by horizontal and vertical lines, looking like a series of steps, with the area between the X-axis and the steps filled with the line color.

Step Area chart

Mixed Series Example

You can also add multiple different series to the same chart.

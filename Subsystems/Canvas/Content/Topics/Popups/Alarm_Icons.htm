Alarm Icons

Use the Configure Alarm Icons dialog box to configure the icon color source, the associated priority ranges, and the icon image used for each priority.

Sample Configure Alarm Icons dialog box
  1. Select the color source from the Icon color source drop-down menu.
  2. Click New icon to add a new icon to the list of priorities.
  3. Configure the desired priority and associated icon for each range.
  4. Click (Delete row) to remove a selected row as necessary.
  5. Click New icon again to add another priority range and repeat the steps described above.
  6. Click OK to accept the priority order, ranges, icon and close the dialog box.
  7. Click Cancel to close the dialog box without modification.

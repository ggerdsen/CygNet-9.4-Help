Starting Canvas

When you finish installing (or updating) Canvas, it's time to open the application and get started.

Starting Canvas

  • If you intend to display CygNet current values or access screens stored in your CygNet BSS, make sure that your CygNet services are running. If not, this is not necessary.
  • Start Canvas in one of the following ways:
    • You can always start the application using Windows Explorer:

      In the CygNet\Bin folder on your system, double-click the Canvas executable file (Canvas.exe) to open the product.

    • If you have previously placed a shortcut for the application on your desktop, double-click to launch the application.
    • If you have previously pinned the application to your taskbar or start menu, click to launch the application.
    • You can also start Canvas from the command line. Using a command prompt navigate to the folder where the Canvas executable file (Canvas.exe) is located and type canvas. See Open Files via the Command Line for information about opening a Canvas screen from a local or network Windows file-system folder or a CygNet APPS or BSS folder.

Getting Started

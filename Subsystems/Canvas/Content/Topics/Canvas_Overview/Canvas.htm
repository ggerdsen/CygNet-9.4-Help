Canvas

Canvas is CygNet Software's HMI (human machine interface) application. Canvas provides high-quality screen design functionality, utilizing a variety of specialized tools and controls, which you can use to create user screens to interface with your CygNet Software installation and data. The Canvas plugin model allows the creation of custom or third-party control types, and once added to Canvas they will be available for addition to your screens.

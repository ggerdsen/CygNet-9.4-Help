Canvas System Requirements

If you currently have a functioning installation of CygNet Software, version v9.4 or later, the Canvas application is compatible with your system.

Components

The following files are included in the \Bin folder provided for your Canvas application:

Component File(s)

Canvas –.exe, .dll

Executable files:

  • Canvas.exe
  • Canvas.Debug.exe
  • Canvas.View.exe

Multiple dynamic link library files:

  • Canvas.*.dll

CygNet – support files

Multiple dynamic link library files of the following types:

  • CygNet.*.dll

Various third-party support files

Multiple dynamic link library files of the following types:

  • Alternet.*.dll
  • SharpVectors.*.dll
  • Telerik*.dll

Environment

The following basic system requirements generally support CygNet Software and Canvas applications.

Description Requirement
Operating system — Clients
  • Windows 10
  • Windows 8.1
  • Windows 8

Refer to the CygNet System Requirements document for further detailed information.

Software

The following software requirements support your Canvas application. The recommended software is helpful in managing and referencing system assemblies.

Description Requirement

Visual C++ Libraries — Run-time components

This package installs run-time components of Visual C++ libraries and can be used to run such applications on a computer even if it does not have Visual Studio 2017 installed.

  • Microsoft Visual C++ Redistributable for Visual Studio 2017

Refer to Microsoft Visual Studio Downloads for more information and instructions.

Windows Software Developer Pack and Development Kit (SDK)

The .NET Framework helps run mobile, desktop, and web applications that run on Windows PCs, devices and servers.

Not required, but recommended:

  • Microsoft .NET Framework 4.7.1 Developer Pack

Refer to Microsoft Download Center for download information and instructions.

Related Documents

  • For information about any supporting software (such as SQL Server, Microsoft Windows, etc.) required for your CygNet system, refer to available documentation from the specific software provider.

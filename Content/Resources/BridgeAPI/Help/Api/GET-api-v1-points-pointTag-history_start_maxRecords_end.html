GET api/v1/points/{pointTag}/history?start={start}&maxRecords={maxRecords}&end={end}

Retrieves history values for a point

CygNet Access Requirements

    CygNet ACS security: BRDGAPI ACCESS 1

    CygNet Bridge API license type: Base

Request Information

URI Parameters

NameDescriptionTypeAdditional information
pointTag

CygNet point tag

 string

Required
start

Starting datetime

 date

Required
maxRecords

Maximum number of history value records to return

 integer

Default value is 0
end

Ending datetime

 date

None

Body Parameters

None

Response Information

Resource Description

Historical point values for a time range

NameDescriptionTypeAdditional information
historyRecords

List of requested history records

 Collection of HistoryRecordDTO

None
isComplete

Indicates whether this response contains all of the requested history records

 boolean

None
nextStartTime

Timestamp of the next history record when response is incomplete

 date

None

Response Formats

application/json, text/json

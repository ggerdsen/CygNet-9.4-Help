GET api/v1/notes/types?siteService={siteService}

Retrieves the active note types

CygNet Access Requirements

    CygNet ACS security: BRDGAPI ACCESS 1

    CygNet Bridge API license type: Base

Request Information

URI Parameters

NameDescriptionTypeAdditional information
siteService

CygNet site and service (Site.Service) of the Note Service (NOTE)

 string

Required

Body Parameters

None

Response Information

Resource Description

NameDescriptionTypeAdditional information
noteTypes

List of active note types

 Collection of string

None

Response Formats

application/json, text/json

Sample: 
{
  "noteTypes": [
    "NONE",
    "NOTE",
    "WARNING"
  ]
}

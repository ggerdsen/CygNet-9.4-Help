GET api/v1/devices/getCommandInfo?facilityTag={facilityTag}&commandName={commandName}&getDetails={getDetails}

Retrieves command info for a facility and command, optionally with extended details

CygNet Access Requirements

    CygNet ACS security: BRDGAPI ACCESS 1

    CygNet Bridge API license type: Base

Request Information

URI Parameters

NameDescriptionTypeAdditional information
facilityTag

CygNet facility tag

 string

Required
commandName

Name of the command to retrieve info for

 string

Required
getDetails

Optional flag to get extended details; default value = false

 boolean

Default value is False

Body Parameters

None

Response Information

Resource Description

NameDescriptionTypeAdditional information
commandInfoXML

Command info XML string

 string

None

Response Formats

application/json, text/json

Sample: 
{
  "commandInfoXML": "<uisCmdInfoList>\r\n<UisCmdInfo cmdName=\"BASICPOLL\" cmdDesc=\"Basic Poll\" cmdState=\"0\" secApp=\"DDS\" secEvt=\"ACCESS\" facility=\"ELLIOT_RD\" stateTime=\"12/31/1969 16:00:00.000\" stateUser=\"\" stateComment=\"\" verify1=\"0\" verify2=\"0\" verify3=\"0\" verify4=\"0\" verifyTime=\"12/31/1969 16:00:00.000\" verifyUser=\"\" verifyComment=\"\" device=\"ELLIOT_RD\" dbKey=\"0000000020C0000005\"/>\r\n</uisCmdInfoList>\r\n"

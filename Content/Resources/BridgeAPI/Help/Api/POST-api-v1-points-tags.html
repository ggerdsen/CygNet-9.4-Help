POST api/v1/points/tags

Retrieves a list of point tags, contained within a PointTagsResponse object

CygNet Access Requirements

    CygNet ACS security: BRDGAPI ACCESS 1

    CygNet Bridge API license type: Base

Request Information

URI Parameters

None

Body Parameters

Filters the list of point tags returned

NameDescriptionTypeAdditional information
siteService

CygNet SiteService of the point service to use the criteria against

 string

Required
filters

List of filter criteria lists

 Collection of FilterCriteriaListDTO

Required
logicalOperator

Logical operator specifying how to combine the criteria lists

 LogicalOperator

None

Request Formats

application/json, text/json

Sample: 
{
  "siteService": "CYGSITE.UIS",
  "filters": [
    {
      "criteria": [
        {
          "propertyId": "Facility",
          "value": "ANTIER_GM",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        },
        {
          "propertyId": "Facility",
          "value": "BRADY_GM",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        }
      ],
      "logicalOperator": "Or"
    },
    {
      "criteria": [
        {
          "propertyId": "UDC",
          "value": "ANHEP",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        },
        {
          "propertyId": "UDC",
          "value": "ANHEX",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        }
      ],
      "logicalOperator": "Or"
    }
  ],
  "logicalOperator": "And"
}

Response Information

Resource Description

PointTagsResponse containing a list of point tags matching the filter constraints

NameDescriptionTypeAdditional information
tags

List of point tags

 Collection of string

None

Response Formats

application/json, text/json

Sample: 
{
  "tags": [
    "CYGSITE.UIS.00004637:ANTIER_GM_ANHEP:ANTIER_GM.ANHEP",
    "CYGSITE.UIS.00004638:ANTIER_GM_ANHEX:ANTIER_GM.ANHEX",
    "CYGSITE.UIS.00005065:BRADY_GM_ANHEP:BRADY_GM.ANHEP",
    "CYGSITE.UIS.00005066:BRADY_GM_ANHEX:BRADY_GM.ANHEX"
  ]
}

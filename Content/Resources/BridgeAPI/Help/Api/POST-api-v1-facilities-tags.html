POST api/v1/facilities/tags

Retrieves a list of facility tags, contained within a FacilityTagsResponse object

CygNet Access Requirements

    CygNet ACS security: BRDGAPI ACCESS 1

    CygNet Bridge API license type: Base

Request Information

URI Parameters

None

Body Parameters

Filters the list of facility tags returned

NameDescriptionTypeAdditional information
siteService

CygNet SiteService of the point service to use the criteria against

 string

Required
filters

List of filter criteria lists

 Collection of FilterCriteriaListDTO

Required
logicalOperator

Logical operator specifying how to combine the criteria lists

 LogicalOperator

None

Request Formats

application/json, text/json

Sample: 
{
  "siteService": "CYGSITE.FAC",
  "filters": [
    {
      "criteria": [
        {
          "propertyId": "FacilityType",
          "value": "WELL",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        },
        {
          "propertyId": "FacilityType",
          "value": "TANK",
          "comparisonOperator": "Equal",
          "comparisonMethod": "String"
        }
      ],
      "logicalOperator": "Or"
    },
    {
      "criteria": [
        {
          "propertyId": "FacilityTextAttribute00",
          "value": "5",
          "comparisonOperator": "NotEqual",
          "comparisonMethod": "Integer"
        },
        {
          "propertyId": "FacilityId",
          "value": "HOUSTON",
          "comparisonOperator": "NotEqual",
          "comparisonMethod": "String"
        }
      ],
      "logicalOperator": "And"
    }
  ],
  "logicalOperator": "And"
}

Response Information

Resource Description

FacilityTagsResponse containing a list of facility tags matching the filter constraints

NameDescriptionTypeAdditional information
tags

List of facility tags

 Collection of string

None

Response Formats

application/json, text/json

Sample: 
{
  "tags": [
    "CYGSITE.UIS::LEWIS_WM",
    "CYGSITE.UIS::MARTIN_GM"
  ]
}

StartTime

The StartTime element specifies the start time of the flow record for the specified meter or meter group for historical and forecast gas load input data.

Properties

Property Description

Case Sensitive

Yes

Required

Yes

Number of Instances Allowed

1 per FlowRecord

Requires End Tag

Yes

Parent Element

FlowRecord

Child Elements

None

Attributes

None

Data Type

The value of the StartTime element is a dateTime timestamp. See Timestamps in ELF Input Files.

Example

<StartTime>2008-01-01T09:00:00.000-08:00</StartTime>

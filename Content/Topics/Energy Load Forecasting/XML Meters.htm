Meters

The Meters element is the parent node for all meters and meter groups. Nested within the Meters tag are elements that define the individual meters and meter groups, meter information, flow records and flow values for the historical and forecast gas load input data.

Properties

Property Description

Case Sensitive

Yes

Required

Yes

Number of Instances Allowed

1

Requires End Tag

Yes

Parent Element

gasLoadImport

Child Elements

Meter

Attributes

None

Example

<Meters>

<Meter id="GH-000001234:bob_meter">

<Description>Bob's Meter</Description>

<GroupID parent_index="0">GH-000001234:station</GroupID>

<GroupID parent_index="1">GH-000001234:sales</GroupID>

<ValueUnits>MMCF</ValueUnits>

<EnergyContent>0.115</EnergyContent>

<EnergyContentUnits>MMBTU/MMCF</EnergyContentUnits>

<FlowData granularity="HOURLY">

<FlowRecord>

<StartTime>2009-03-12T09:00:00.000Z</StartTime>

<EndTime>2009-03-12T10:00:00.000Z</EndTime>

<FlowValue>234.894</FlowValue>

</FlowRecord>

<FlowRecord>

<StartTime>2009-03-12T10:00:00.000Z</StartTime>

<EndTime>2009-03-12T11:00:00.000Z</EndTime>

<FlowValue>250.734</FlowValue>

</FlowRecord>

</FlowData>

</Meter>

</Meters>

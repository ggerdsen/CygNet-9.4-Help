Using the CygNet ELF Tree Grid Control

The CygNet ELF Tree Grid Control has numerous Run mode options.

Run Mode Options

  • Expanding the Tree
  • Adding a New Child
  • Adding a New Sibling
  • Editing a Node
  • Deleting a Node
  • Searching for a Node
  • Viewing Training for a Neural Network Definition
  • Viewing Archived Neural Network Definitions
  • Approving Training for a Neural Network Definition
  • Approving Training for All Neural Network Definitions
  • Rejecting Training for a Neural Network Definition

Expanding the Tree

Right-click any node and select Expand All to expand the selected node.

Adding a New Child

Right-click any parent node and select New Child to add a new child to the selected branch. See the following links for additional resources:

  • Configuring Neural Network Template Settings
  • Configuring Neural Network Definition Settings
  • Configuring Meter/Meter Group General Settings
  • Configuring Input Owner/Input Owner Group Settings

Adding a New Sibling

Right-click any child node and select New Sibling to add a new sibling to the selected branch. See the resources above.

Editing a Node

Right-click any node and select Edit to edit the selected node. See the resources above.

Deleting a Node 

Right-click any node and select Delete to delete the selected node.

Searching for a Node

Right-click any node and select Jump to to jump to a known facility ID. Type the desired full facility tag. Click OK.

Viewing Training for a Neural Network Definition

Right-click any node and select View Training to see the training results for the selected neural network definition. See Viewing Training Results.

Viewing Archived Neural Network Definitions

If you have selected to include Archived Definitions on the tree you can view all archived neural network definitions by right-clicking the Archived Definitions branch and select Expand All. See Viewing Archived Neural Network Definitions.

Approving Training for a Neural Network Definition

Right-click any node and select Approve Training to approve training for the selected neural network definition.

Approving Training for All Neural Network Definition

Right-click any node and select Approve All Training to approve training for all neural network definitions.

Rejecting Training for a Neural Network Definition

Right-click any node and select Reject Training to reject training for the selected neural network definition.

Viewing Archived Neural Network Definitions

Archived neural network definitions can be viewed as children of the parent neural network definition on the Neural Nets page.

  • To view all archived neural network definitions right-click the Archived Definitions branch of the Neural Network Definitions hierarchy and select Expand All.
  • Each archived version of the parent neural network definition is displayed under the parent. (The parent definition is identical to the definition shown up in the All Definitions branch and can be edited.)
  • Archived neural network definition are numbered sequentially and the Details column indicates the date and time when the definition was archived.
  • Double-click an archive to view its properties.

The Neural Network Definition Archived on: <date and time> dialog box presents the configured properties for the archive in read-only mode.

The properties on this dialog box are identical to those on the parent Neural Network Definition Settings dialog box, except for the following items:

Training Hourly Intervals

The actual training hourly interval information used for training is shown in the archive. These include, depending on definition settings:

  • Template ID
  • Training Interval used
  • Selection Method used
  • Percentage of Interval used
  • Offset and/or Seed Value used

Input Data Items

The input data items used for training for that particular instance are shown in the archive. These include, depending on definition settings:

  • Input Type (Ordinal or Categorical)
  • Input ID (the Site, Service, Facility of the Record Owner ID, and the UDC of the Input Item Type used)
  • Details (for example, Filling Rule for the Input Data Item).

Clicking Tuning Parameters will show the tuning parameters for the neural network definition from the time when the definition was archived in read-only mode. See Configuring Tuning Parameters for more information.

Clicking Training Results will show the training results for the neural network definition from the time when the definition was archived. See Viewing Training Results for more information.

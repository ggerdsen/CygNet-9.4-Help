CygNet ELF Trend Grid Control

The CygNet ELF Trend Grid Control is an ActiveX control used to create custom human-machine interface (HMI) screens in a CygNet Studio application instance to view trends and tabular displays of forecasted data over a configured period of time.

Use the HMI screen to change the time window displayed, change the target facility, reconfigure the series displayed on an ad hoc basis, view or edit any value displayed in the trends.

CygNet ELF Tree Grid Control

The CygNet ELF Tree Grid Control is an ActiveX control used to create custom human-machine interface (HMI) screens in a CygNet Studio application instance to navigate the meters and meter groups, input owners and input owner groups, and neural network templates and definitions configured for your ELF system.

Use the HMI screen to manipulate the tree hierarchy, add new children or siblings to existing meters, input owners or neural networks, edit any node, and jump to any node in the hierarchy. Additionally you can view, approve, approve all, and/or reject training for any neural network.

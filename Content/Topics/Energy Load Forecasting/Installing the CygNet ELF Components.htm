Installing the CygNet ELF Components

Perform the following steps to install the CygNet ELF components: the ELF EIE and the ELF CxHmiElf ActiveX control.

To Install the ELF EIE

  1. Copy the ELF driver file (UisElfDeviceDriver.dll) to the CygNet\Support directory on the host server.
  2. Register the file using regsvr32.exe (located in the Windows\System32 folder, or use Windows\SysWOW64\regsvr32.exe on a 64-bit operating system).
  3. Using CygNet Explorer, upload the editor (CxDdsElfEditors.dll) to the SUPPORT group in the Application Blob Service (APPS).
  4. Using CygNet Explorer, upload the ELF EIE version management file (CxDdsElfEditors.rsp) to the SUPPORT group in the Application Blob Service (APPS).
  5. Download the CxDdsEditors.rsq file from the SUPPORT group in the Application Blob Service (APPS).
  6. Copy the following line to the CxDdsEditors.rsq file:

[SUPPORT]..\SUPPORT\CxDdsElfEditors.rsp
 

  1. Upload the edited CxDdsEditors.rsq file to the SUPPORT group in the Application Blob Service (APPS).
  2. Use the TouchBSS utility found in the CygNet\Utilities directory to update the timestamp for all .rsp and .rsq files in the Application Blob Service (APPS).
  3. Close and reopen CygNet Explorer to perform version management.

To Install the CygNet ELF CxHmiElf ActiveX Control

  1. Copy the Elf ActiveX file (CxHmiElf.ocx) to the CygNet\Support directory on the host server.
  2. Register the file in one of two ways:

  1. Register the file using CVersionInfo.exe (located in the CygNet\Utilities folder).
  2. Register the file using regsvr32.exe (located in the Windows\System32 folder, or use Windows\SysWOW64\regsvr32.exe on a 64-bit system).

  1. Using CygNet Explorer, upload the file (CxHmiElf.ocx) to the SUPPORT group in the Application Blob Service (APPS).
  2. Download the StudSupp.rsp file from the SUPPORT group in the Application Blob Service (APPS).
  3. Copy the following line to the "CygNet ActiveX and Automation Files" section of the StudSupp.rsp file:

[SUPPORT]..\SUPPORT\CxHmiElf.ocx <REGISTER>

  1. Upload the edited StudSupp.rsp to the SUPPORT group in the Application Blob Service (APPS).
  2. Use the TouchBSS utility found in the CygNet\Utilities directory to update the timestamp for all .rsp and .rsq files in the Application Blob Service (APPS).
  3. Close and reopen CygNet Explorer, and/or launch CygNet Studio to perform version management.

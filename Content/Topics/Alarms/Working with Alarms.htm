Working with Alarms

Alarm actions include:

  • Acknowledgment
  • Clear/Force Clear
  • Hide
  • Suppress

These actions can be performed using options on the alarm context menu.

An action can be performed on multiple alarms. To select multiple alarms, use Shift+Click to select alarms listed contiguous in the display or Ctrl+Click to select non-contiguous alarms. To select all alarms in the CAS, right-click in the alarm view and select Select All.

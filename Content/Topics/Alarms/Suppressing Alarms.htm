Suppressing Alarms

When you suppress an alarm, it is suppressed until it returns to normal or a specified period of time has elapsed. Suppression can be set up for a point so that any alarm that is reported is suppressed or can be performed on an ad hoc basis (that is, suppression is an action you can take on the alarm in the alarm view). Suppression applies to all alarms for a point, not individual alarms.

To Suppress an Alarm on an Ad Hoc Basis

  1. Click to select the alarm(s) that you want to suppress. Use Ctrl+Click to select non-contiguous alarms and Shift+Click to select contiguous alarms.
  2. Right-click in the Alarm dialog box and select Suppress.
  3. Click to enable Suppress alarms for the (n) selected points. (The number of selected points is shown in parentheses.)
  4. Click to enable one or both suppression ending conditions.
  5. Click Yes to the confirmation message. Another message box appears giving the status of the suppression settings.
  6. Click Done.

To Configure a Point for Alarm Suppression

  1. Right-click on the alarm in the alarm view and select Properties.
  2. Click Point Configuration to open the PNT Editor.
  3. Click on the page that contains alarm information (Analog, Digital, String, or Enumeration).
  4. Click the Suppress Alarms browse button.
  5. Click to enable Suppress all alarms for this point.
  6. Click to enable one or both suppression ending conditions.
  7. Click Apply.
  8. Click Yes to the confirmation message. Another message box appears giving the status of the suppression settings.
  9. Click Done.

To Unsuppress an Alarm on an Ad Hoc Basis

  1. Set the filter of the alarm view to show suppressed alarms.
  2. Click to select the alarm(s) that you want to unsuppress. Use Ctrl+Click to select non-contiguous alarms and Shift+Click to select contiguous alarms.
  3. Right-click in the Alarm dialog box, and select Suppress.
  4. Click to uncheck Suppress alarms for the (n) selected points. (The number of selected points is shown in parentheses.)

To Remove Alarm Suppression from a Point Configuration

  1. Set the filter of the alarm view to show suppressed alarms.
  2. Right-click on the alarm in the alarm view and select Properties.
  3. Click on Point Configuration.
  4. Click on the page that contains alarm information (Analog, Digital, String, or Enumeration).
  5. Click the Suppress Alarms browse button.
  6. Click to uncheck Suppress all alarms for this point.
  7. Click Apply.
  8. Click OK.

Alarm Suppression and Notifications

If a point has alarm suppression active and one or more of its alarms are configured to send a notification, the notification will not be sent when the alarm is set due to the suppression. However, the Return to Normal message will be sent if: 1) the notification is configured to send a message when the point returns to normal; and 2) the suppression "end condition" is Point is Normal.

If an alarm is suppressed on an ad hoc basis and the point is configured with a notification, the notification message will have been sent. This is because the suppression was not active when the alarm occurred.

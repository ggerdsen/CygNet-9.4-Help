Columns for the View

A variety of column headings from special categories can be displayed in the alarm view. There is no limit on how many selected attributes you want to display in the alarm view, although it will be difficult to view all of them if too many columns are selected. Also, if you do not have a PNT service running, and you select an attribute from a non-running point service, then the column headings will be displayed, but the attributes for them will not; the columns will be blank.

To Customize the Alarm Columns

  1. Right-click in the alarm view and select Columns.
  2. From the Alarm Columns dialog box, select a category attribute (see table below for Available Attributes).
  3. Select the attribute and click Add to move an attribute heading over to the Selected Attributes column. Click Move Up and Move Down to position the column heading to the left or right in the alarm view. Click Remove to remove an attribute column heading from the alarm view. When finished, click OK.
  4. To rename a column heading attribute, double-click on it in the Alarms Columns dialog box. (You cannot add a new site attribute since site attributes are defined in the CygNet Standard Point Scheme; you can only rename an existing column attribute.)
  1. Enter the number of screen pixels (not characters) in the Minimum Column Width.
  2. Enter the new attribute heading in the Override Column Title.
  3. Click OK.
  1. Click OK.

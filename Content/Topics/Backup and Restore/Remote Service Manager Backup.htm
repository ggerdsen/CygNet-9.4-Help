Remote Service Manager Backup

This topic describes automatic backup on CygNet’s Remote Service Manager.

Enabling backup of the RSM will copy the configuration file (Rsm.cfg) and the RSM database file (SvcDef.rsx) to the Backup location specified in the BACKUP_PATH keyword for the RSM.

Backup Keywords

Configure the following keywords with a valid parameter in the RSM service configuration file to enable automatic backup of the above files:

  • BACKUP_PATH
  • BACKUP_TIME
  • COPY_BACKUP_TO

See Backup Keywords and Paths for definitions. See RSM Configuration File Keywords for configuration keyword defaults and options.

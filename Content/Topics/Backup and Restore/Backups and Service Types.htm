Backups and Service Types

Each CygNet service can be configured to automatically backup its important service, database, and configuration files. In CygNet there are two methods for performing a backup:

  1. Enabling and configuring appropriate keywords in the service configuration file for each service:
  • BACKUP_PATH
  • BACKUP_TIME
  • COPY_BACKUP_TO

See Backup Keywords and Paths for definitions of these keywords.

  1. Schedule a backup using the MSS. See Scheduling a Backup with the MSS for more information.

Service Types

CygNet services can be generally classified when it comes to performing operations such as backups. Backup operations, methods, and content differ slightly between different types of service. The following topics describes the backup operations for the different types of CygNet services:

  • DBS-Based Services
  • Value History Service
  • Current Value Services and Common Alarm Services
  • Address Resolution Service
  • Remote Service Manager

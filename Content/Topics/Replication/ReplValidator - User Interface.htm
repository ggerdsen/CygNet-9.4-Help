ReplValidator User Interface

The following topic describes the user interface and the many options available from the ReplValidator utility.

The ReplValidator utility consists of a menu bar, a toolbar, a display grid, a log viewer, and a status bar. The following tables list the options accessible using the menu, toolbar, and context menu.

  • Menu
  • Toolbar
  • Context Menu

Menu

Five tabs are available on the menu bar to access various utility functions, via the File, View, Validation, Tools, and Help menus.

File Menu

The File menu provides access to the following options.

Option Description Shortcut Toolbar

Create Config…

Select to access the Create Configuration dialog box to create .xml configuration file comparisons for validation of replicated services compared to their source service.

See Creating the ReplValidator Configuration File for more information.

 Ctrl+C  

Open Config…

Select to access the Open dialog box to navigate to a previously saved ReplValidation .xml configuration file to select for validation of replicated services. Once selected and opened, the primary and replicated sites will appear in the Replication Sites list.

Ctrl+O

Edit Config…

Select to access the Edit Configuration dialog box to edit comparison values for a previously saved .xml configuration file, for validation of replicated services compared to their source service.

See Creating the ReplValidator Configuration File for more information.

Ctrl+E

  

Recent Files

List of all recently opened ReplValidation configuration files. Click on any file name to change to the settings configured in that file. -  

Exit

Select to close the ReplValidator utility.

Validation Menu

The View menu provides access to the following options.

Option Description Shortcut Toolbar

Start

Click to begin the process of validating the replicated service data against the source service data for the selected Site.Service pair(s).

See Running the ReplValidator Utility for more information.

 -

Stop

Click to discontinue the process of validating the replicated service data against the source service data for the selected Site.Service pair(s).

See Running the ReplValidator Utility for more information.

 -

Select All

Click to select all listed Site.Service pairs for validation of their replicated service data against their source service data.

See Running the ReplValidator Utility for more information.

 -

Deselect All

Click to clear all previous selections of Site.Service pairs for validation.

See Running the ReplValidator Utility for more information.

Tools Menu

The Tools menu provides access to the following options.

Option Description Shortcut Toolbar

Enable Log Viewer

Select to display the log file for the currently selected pair of services in the lower Log Viewer pane.

The Enable Log Viewer Refresh check box will show the selected/cleared status of the option.

See View Validation Logs for more information.

 -

DBS Record Viewer

Select to open the View Primary/Repl Record dialog box to see a side-by-side comparison of the selected database record in the primary and replicated services.

See Comparing Database Records for more information.

-

DBS Resync Keys

Select to open the Resync Records dialog box to submit a list of database keys to be resynchronized on the replicated service.

The Resync option triggers the replicated service to read the specified primary service database keys and change its records to match accordingly. When the resynchronization is complete a success summary is listed in the lower pane.

See Resynchronizing Database Keys for more information.

 -

Help Menu

The Help menu provides access to the following options.

Option Description Shortcut Toolbar

About ReplValidator…

Click to open the About ReplValidator dialog box to view utility information.

Available information includes the following:

  • The application name, version, and release date
  • Copyright information
  • The CygNet Software website address, including a hyperlink to the website
  • The log file location, including a hyperlink to the log file
  • The location of the application's file executable

In addition to information about the application, the About box includes the following buttons:

  • Client Settings — opens the Client Settings dialog box to view and modify client log settings for the application on an ad hoc basis. See Client Log Settings in the Logging section for more information.
  • Diagnostics — access the following options for additional analysis from the drop-down menu:
  • Loaded Modules — use this option to view and search file information for modules loaded in your system. Double-click any result row to view properties for that module. See Loaded Modules for more information.
  • Gather Logs — accesses the Windows file directory where the open log files are located. Double-click to select and view an application log file.
  • User Local Crash Dumps — Local crash dumps are not enabled. See Generating Crash Dumps for more information.
  • Close — click to close the About box.

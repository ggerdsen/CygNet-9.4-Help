VHS Summary Statistics

These statistics are summaries for the entire VHS. Current size statistics are provided as well as the estimates for how current thinning rules would reduce the VHS size, and how much space would be saved as a result.

Field Description

Point Count

Total number of points in the VHS.

Entry Count

Total number of entries in the VHS.

Daily Avg

The average number of new entries added to the VHS per day.

Steady State Potential

An estimate of the number of entries in the VHS if allowed to reach its steady state. The box to the right is the estimated size of the VHS database file if it contained the number of entries listed.

Adj. Potential (Thinning)

An estimate of the new steady-state potential if all the currently defined thinning rules are used. The box to the right is the estimated size of the VHS database file if it contained the number of entries listed.

% Thinning Reduction

The percentage of how much the VHS would be reduced by if the current thinning rules were applied.

Current File Size

The current file size of the VHS database. The VHS database expands a block at a time so this number may reflect a size that is larger than the amount of data that has been written.

File In Use

The amount of file space actually being used by the VHS.

Max File Size

The configured maximum size of the VHS.

Monitoring VHS Data Forwarding

The CygNet Console screen for the VHS has a screen that monitors History Forwarding statistics.

VHS Data Forwarding Info Items

There are several SVCMON UDCs that can be used to monitor VHS Data Forwarding using the CygNet ServiceMon Administration utility. The system info items upon which these UDCs are based can be viewed for each service on the Service Generic page of the CygNet Service Information dialog box.

Note: It is recommended that the SVCMON timer be set for no more than five seconds.

Info Item Description SVCMON UDC

AVG_RESPONSE_TIME

On average, the amount of time it takes for the history receiver to respond to history forwarding messages.

 SVMHSTAVR1

AVG_RESPONSE_TIME_2

On average, the amount of time it takes for the second history receiver to respond to history forwarding messages.

SVMHSTAVR2

AVG_RESPONSE_TIME_3

On average, the amount of time it takes for the third history receiver to respond to history forwarding messages.

SVMHSTAVR3

EXPIRED_COUNT

The number of history forwarding messages that have not been sent to the history receiver due to expiration as configuration parameters specify.

 SVMHSTEXP1

EXPIRED_COUNT_2

The number of history forwarding messages that have not been sent to the second history receiver due to expiration as configuration parameters specify.

SVMHSTEXP2

EXPIRED_COUNT_3

The number of history forwarding messages that have not been sent to the third history receiver due to expiration as configuration parameters specify.

SVMHSTEXP3

FORWARDER_COUNT

The number of history forwarders enabled.

SVMVHSFWDR

FORWARDING_ENABLED

Indicates if any history data forwarding is enabled.

SVMVHSFWDE

FWD_MESSAGE_ERRS

The number of history forwarding messages for which the history receiver has returned an error.

SVMHSTMSG1

FWD_MESSAGE_ERRS_2

The number of history forwarding messages for which the second history receiver has returned an error.

SVMHSTMSG2

FWD_MESSAGE_ERRS_3

The number of history forwarding messages for which the third history receiver has returned an error.

SVMHSTMSG3

FWD_NETWORK_ERRS

The number of history forwarding messages to the history receiver that have returned network errors.

SVMHSTDCL1

FWD_NETWORK_ERRS_2

The number of history forwarding messages to the second history receiver that have returned network errors.

SVMHSTDCL2

FWD_NETWORK_ERRS_3

The number of history forwarding messages to the third history receiver that have returned network errors.

SVMHSTDCL3

IS_FORWARDING

Indicates if history data forwarding has stopped. Possible scenarios might include the following: the forwarder is shut down or is not configured for forwarding, the destination(s) is/are shut down or does/do not exist, or a communication failure exists (network outage, firewall, etc.).

 SVMHSTFWD

LAST_RECEIVE_TIME

The time that the service last received a history forwarding message.

SVMVHSHRCV

LAST_RESPONSE_TIME

The amount of time it took for the history receiver to respond to the last history forwarding message.

 SVMHSTLRS1

LAST_RESPONSE_TIME_2

The amount of time it took for the second history receiver to respond to the last history forwarding message.

SVMHSTLRS2

LAST_RESPONSE_TIME_3

The amount of time it took for the third history receiver to respond to the last history forwarding message.

SVMHSTLRS3

LAST_SENT_TIME

The time that the last history forwarding message was sent to the history receiver.

 SVMHSTLSN1

LAST_SENT_TIME_2

The time that the last history forwarding message was sent to the second history receiver.

SVMHSTLSN2

LAST_SENT_TIME_3

The time that the last history forwarding message was sent to the third history receiver.

SVMHSTLSN3

MESSAGES_RECEIVED

The total number of history forwarding messages received.

SVMVHSRRCV

MESSAGES_SENT

The number of total history forwarding messages sent to the history receiver.

 SVMHSTSNT1

MESSAGES_SENT_2

The number of total history forwarding messages sent to the second history receiver.

SVMHSTSNT2

MESSAGES_SENT_3

The number of total history forwarding messages sent to the third history receiver.

SVMHSTSNT3

PENDING_COUNT

The number of pending history forwarding messages to send to the first VHS.

 SVMHSTPND1

PENDING_COUNT_2

The number of pending history forwarding messages to send to the second VHS.

SVMHSTPND2

PENDING_COUNT_3

The number of pending history forwarding messages to send to the third VHS.

SVMHSTPND3
QUEUED_EDITS

The number of history changes queued for compression and forwarding to the history receiver.

 SVMHSTQED1

QUEUED_EDITS_2

The number of history changes queued for compression and forwarding to the second history receiver.

SVMHSTQED2

QUEUED_EDITS_3

The number of history changes queued for compression and forwarding to the third history receiver.

SVMHSTQED3

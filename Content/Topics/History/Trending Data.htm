Trending Historical Data

CygNet Software supports many ways to graphically display trend data. The following table provides basic information about each way and directs you to additional information. Unless otherwise noted, trend data for each item below originates in the VHS.

CygNet Feature Description

CygNet Studio Trend Tool

 The Trend Tool is an ActiveX control used in CygNet Studio and CygNet Vision screens to provide a customizable means to chart data in a number of useful formats, including line graphs, bar graphs, and area graphs.
CygNet Studio TheFrame/TheView Trend Property

The Default Trend is a property of TheFrame/TheView, but it has much in common with the Trend Tool.
CygNet Studio Tool-Specific Trends

Many CygNet Studio tools include a Trend option in Run mode. Some of these tools are the Button Tool, Text Tool, and Vector Image Tool. Trend options include the Single-Point Trend and Single-Point Trend with Rollups.

Although the Group Grid Tool does not have a Trend option, items included within its grid do.

Although the WatchList Tool does not have a Trend option, items included within its grid do.
Button Tool The Button Tool enables you to, among other things, open a pre-configured default trend. See Button Tool.
ListView Control A Microsoft ListView Control can be used to view multiple Trend Tools simultaneously. See Using a ListView Control to Select Trends to Display.
Remote Device Trends

Several EIEs provide specialized trending capabilities. These EIEs are as follows:

  • FB Net EIE
  • Lufkin MPC/RPC EIE
  • Lufkin SAM EIE
  • Thermo EIE
  • Totalflow EIE
  • TriEnerTech EIE
  • Weatherford WellPilot DLQ/K-Series EIE

Note: Remote devices store historical data in the Device Definition Service (DDS). If you map this data to a UDC, it can also be stored in the VHS.
CygNet Measurement History Graph Control

CygNet Measurement provides trending capabilities. See Using the History Graph Control and CygNet FMS History Graph Control.

Note: CygNet Measurement relies on a SQL Server database for its history.
CygNet ELF Trend Grid Control CygNet ELF provides trending capabilities. See CygNet ELF Trend Grid Control.
GCM Run Mode Options

The General Contract Monitor (GCM) includes trending capabilities implemented using script. See Using a GCM Nomination Grid.

Service Monitoring Utilities

Two Service Monitoring utilities provide trending capabilities. See

  • Trending Service Info Items
  • Trending SVCMON Point Values.
VHS History Values Graph

Entries in the VHS include a View Values option that enables you to view trends on a history record-by-history record basis using the View Graph button on the record.

You can also access this information from the Universal Interface Service (UIS) on a current value point-by-point basis using the Value History button on the point record.
VHS Data Analyzer Utility

The VHS Data Analyzer Utility includes a Trend option that enables you to view VHS point data trends. See Configuring Trend Defaults.

You can also Trend points in the Thinnning Simulator.
CygNet Web Trending

CygNet Web includes trending capabilities. See Configure Application and Customize the Site.

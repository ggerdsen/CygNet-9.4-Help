Error Logging

The VHS Recovery utility can restore data from journal files or database files that were not properly closed. Any errors that are encountered while reading values from a corrupt file will be recorded to the utility’s log file for review.

Point tags are verified against the Point (PNT) service for consistent Short and Long Point IDs. If a Point ID mismatch is detected, the point and its values are not imported to the VHS, but are recorded to the log file for review.

Point deletions are not restored from journal files because of the potential for conflicts. However, this information is recorded in the log file for review.

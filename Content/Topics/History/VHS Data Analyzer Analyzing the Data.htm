Analyzing the Data

Any time the filtering, grouping, or thinning options are changed, you must click the Refresh button to update the statistics in the main section of the VHS Data Analyzer. Several options are available for data analysis and review:

Option Description
Point or Group Statistics

See Point or Group Statistics for an explanation of the data displayed.
Summary Statistics

See VHS Summary Statistics or an explanation of the VHS statistics displayed below the point or group statistics.
Save Statistics

To save out the raw data in a .vda file for future analysis, select Save… on the File menu.
Export Statistics

To export the data analysis displayed in the view to a CSV (comma-separated values) file, select Export to CSV…

On the File menu.

Note: If viewing the export in Microsoft Excel, the view will be limited to the first 65,536 rows. To view more rows a different application must be used.
Thin the Data

To determine how you might thin (delete) data in your VHS select Configure Thinning Rules… on the File menu.

To create a thinning based on a single point, right-click a point and select Create Thinning Rule….

See Thinning Rules for more information.
View History Values

To view all VHS entries for a point, right-click a point and select View Values… to open the History Value dialog box.
View or Edit Point Record

To open the Properties for Point dialog box for a point, right-click a point and select View Point Record…. See PNT Editor for more information.
Trend a Point's History Data

To view a trend for up to 16 points, right-click point(s) and select Trend…. Trend options are set on the Trend menu. See Configuring Trend Defaults.
Add to Simulator

To add a point(s) to the VHS Data Analyzer Thinning Simulator, right-click and select Add to Simulator. The row will be marked orange. See Thinning Simulator for more information.
Remove from Simulator To remove a point(s) from the VHS Data Analyzer Thinning Simulator, right-click and select Remove from Simulator. See Thinning Simulator for more information.
Delete a Point To delete a point and all its entries from the VHS, select the point, right-click and select Delete from VHS.

CygNet Web Browser Requirements

CygNet Software supports the following browsers for viewing CygNet Web:

  • Internet Explorer 9
  • Internet Explorer 10 or Internet Explorer 11 in Compatibility View. Refer to Compatibility View for information about configuring sites to be viewed in compatibility mode.

To Display a Site in Compatibility View

  1. Open Internet Explorer 10 or Internet Explorer 11.
  2. Press the Alt key to display the menu bar.
  3. Click on the Tools menu.
  4. Select Compatibility View settings.
  • Click the check box to display all websites in Compatibility View.

- or -

  • Add only desired websites to Compatibility View.
  1. Close the Compatibility View Settings dialog box.

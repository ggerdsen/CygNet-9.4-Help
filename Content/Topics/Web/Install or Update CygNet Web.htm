Install or Update CygNet Web

To Set Up the Web Server for CygNet Web

  1. Download CygNetWebSetup.exe.
  2. Stop IIS.
  3. If your CygNet Services are running on the same server as IIS, shut down the following services: UIS, HSS, SVCMON, OPCIS. 
  4. Shut down all CygNet clients running on the IIS server, such as CygNet Studio and CygNet Explorer.
  5. Verify the following files are in your Application BLOB Service in the indicated application designations:

[WEB]

  1. For Upgrades only: We suggest deleting the old site except for the "custom" subfolder because there are many files that you no longer need. This is not necessary, but good housekeeping.
  2. Run the CygNet Web setup program, CygNetWebSetup.exe. Choose one of the following:
  1. For Install — follow the prompts and specify the desired directory for the root of the website.
  2. For Update — select the Modify option when prompted by the setup program.
  1. Follow the prompts to finish the setup.
  2. Update the required CygNet files on the IIS server by clicking on the CygNet Web Update icon that should now be on the desktop of your IIS server. This will register the required CygNet dlls on the IIS server.

Note: Remember to make sure the support files on your CygNet BLOB service are up to date so that the files that get copied to your Web server are up to date.

  1. Restart IIS. At this point the Web pages are installed.
  2. Restart any CygNet Services that you shut down in Step 3.
  3. Update the CygNet License manager. See Enter Web License Information.

To Add your Logo (new install only)

To customize your CygNet website with your logo, replace the custom/customerLogo.gif file with your own logo. Keep the file name as customerLogo.gif. The logo will look best if the graphic is 145 pixels wide. The height can be any value.

To Enable Excel Export (new install only)

Some of the CygNet Web pages support exporting data to Microsoft Excel spreadsheets. To employ this functionality you must make the following two changes to your IIS server.

  1. Enable Microsoft Office Web Component (OWC) to support exporting of data from CygNet Web to Excel. On the Microsoft Office Setup Disk, under the Excel option, select and install the Microsoft Office Web Component (OWC). Alternatively you can download it from the Microsoft website.
  2. To enable Excel export from the CygNet Web application, users who are to be granted this functionality must have Modify permission on the Spreadsheets subdirectory.

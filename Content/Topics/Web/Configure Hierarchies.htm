Configure Hierarchies

The Configure Hierarchies option links to a form you can use to structure your individual Web hierarchies. You specify a hierarchy by selecting the Web root from the drop-down menu, and then you specify options for the selected hierarchy. The steps required are detailed below.

To Configure a Hierarchy

  1. From the CygNet Web Console, select Configure Hierarchies.
  2. In the Hierarchy section, select the desired Web Root to display the parameters for the corresponding hierarchy.
  3. Configure the options for the History Report page.
  1. Rollup Units — The measure of time used to calculate the rollup information (days, minutes, hours, or none).
  2. Top of Day — The time to start the rollup calculation.
  3. Rollup Period — The number of Rollup Units to span for the calculation.
  4. UDC List — List of UDCs to display on the History Report page. For each UDC you specify the rollup type that will be used to calculate the history information.
  1. When all of the Hierarchy settings are satisfactory, click the Save Hierarchy button.

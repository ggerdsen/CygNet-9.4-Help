Enter Web License Information

Web License Name

Using the License Manager utility, enter the website and user information. For detailed instructions see License Manager Utility.

Note: The License Manager uses the IP address as registered license name. If your CygNet Web application is set up at the root of your IIS server site, enter only the IP address in the license manager. If you are using virtual websites which appear like subfolders below your IIS server name, then enter the IP address with the virtual site name.

Example

  • 190.161.1.55 (website at root of Default website in IIS)
  • 190.161.1.55/cygnetview (virtual site, “cygnetview,” below Default website in IIS)

WebStatus Command Setting

WebStatus servers allow users to send a single UIS command that can be specified using the License Manager utility. Setting a UIS Command is optional.

CygNet Web Console

You can use the CygNet Web Console (cygAdmin.asp) to manage the various settings for your site. It is organized with links to several pages to help you set up and maintain your site. Some diagnostic information is also provided at the top of the page: website URL, root directory, Web license mode, group service, authenticated user, Web domain, and session ID.

To access the Web Console enter the root address of your new CygNet website followed by /cygAdmin.asp.

Example

myIISserver/myWebsite/cygAdmin.asp

SonicMQ Export EIE

The SonicMQ Export EIE is an export driver that allows users to export CygNet SCADA data in a well-defined XML format for consumption by other business entities. By utilizing SonicMQ Broker technology as middleware, the EIE can export data to business units with only minimal coupling. The CygNet SonicMQ Export EIE can be configured to export to the local file system in the absence of a SonicMQ Broker.

All facility attributes and point attributes are supported by the export.

The EIE works in conjunction with a navigation view root node (stored in the CygNet Group Service Group Nodes). During the export process, the exporter traverses the Root Node and exports data for every subnode with a referenced facility that meets the configured Node Filter attributes. By adding UDCs and point attributes to the configuration, exports will contain the configured point data for all points in the hierarchy that match a given UDC. To export historical point values, a point-in-time can be specified per UDC. Each point-in-time can either be absolute or relative to the current time.

The SonicMQ Export EIE does not support a 64-bit version. See 64-bit Device Drivers for more information.

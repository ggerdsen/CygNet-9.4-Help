Installing the Sonic Queue EIE

To Install the Sonic Queue EIE

  1. Copy the driver file (UisSonicQDeviceDriver.dll) to the CygNet\Support directory on the host server.
  2. Register the file using regsvr32.exe (located in the Windows\System32 folder, or use Windows\SysWOW64\regsvr32.exe on a 64-bit operating system).
  3. Using CygNet Explorer, upload the editor (CxDdsSonicQEditors.dll) to the SUPPORT group in the APPS BLOB.
  4. Upload the Sonic Queue EIE version management CxDdsSonicQEditors.rsp. file to the SUPPORT group in the APPS BLOB
  5. Download the CxDdsEditors.rsq file from the SUPPORT group in the APPS BLOB. Copy the line shown below to the file.

[SUPPORT]..\SUPPORT\CxDdsSonicQEditors.rsp
  1. Upload the edited CxDdsEditors.rsq to the SUPPORT group APPS BLOB.
  2. Use the TouchBSS utility (CygNet\Utilities directory) to update the file time for all .rsp and .rsq files in the APPS BLOB.
  3. Close and reopen CygNet Explorer so that version management is performed.

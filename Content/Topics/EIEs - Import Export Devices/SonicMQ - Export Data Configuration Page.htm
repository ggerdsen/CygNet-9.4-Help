Configuring the Export Data

The SonicMQ Export Data Configuration page defines the information to be included in the export.

The properties for each section of this page are described below.

Property Description

Group Service

The name of the Group Service from which to get group information.

Navigation Root Node

The navigation root to be traversed.

View Root Node

The navigation root's view node to be traversed.

Select Hierarchy Level

The level in the hierarchy.
Export Data for Selected Hierarchy Level

Node Filter

Sets filter criteria on group attributes.

FAC Attrs.

Specifies facility attributes to include in the export.

Add UDCs

Specifies UDCs to include in the export.

Point Attrs.

Specifies point attributes to include in the export.

Export Time

Sets export time for UDC items. Select a UDC item from the list and click Export Time to set the absolute or relative export time.

Delete

Deletes the selected item from the list of data to export.

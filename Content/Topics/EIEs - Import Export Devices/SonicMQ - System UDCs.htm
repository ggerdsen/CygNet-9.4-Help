System UDCs

The following table describes all the system UDCs that keep track of the statistics for each SonicMQ Export device.

UDC UDC Description Long Description Auto Created
SMQXATRE Attrs exported Number of attributes in last XML file Yes
SMQXCHRC XML char count Number of characters in last XML file Yes
SMQXDEST Last arc dest Destination of last disk archive Yes
SMQXDF Last disk arc failed Failure state of last disk archive Yes
SMQXEF Last export failed Failure state of last export Yes
SMQXEND End time End time Yes
SMQXQF Last queue failed Failure state of last queue operation Yes
SMQXQST Queue send time Queue send time Yes
SMQXRFR Resend files Number of files waiting to be resent Yes
SMQXSDEL SMQ delivery suc cnt Total number of successful SMQ deliveries Yes
SMQXSERR SMQ delivery err cnt Total number of failed SMQ deliveries Yes
SMQXSTRT Start time Start time Yes
SYEDFOP Failed ops Failed operations Yes
SYEDIPOP Pending export count Pending export count Yes
SYEDSOP Successful ops Successful operations Yes

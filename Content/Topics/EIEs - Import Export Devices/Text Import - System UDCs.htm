System UDCs

Each Text Import "device" is a facility. This facility has a set of "status tags" that are used to monitor the import statistics, for example, the number of values updated, the number of tags created, values that failed to import, the time the import began/completed, etc. These tags can be used to set alarms, notifications, etc.

The status tags are automatically created the first time the import is run.

You can monitor the statistics of Text Import using system uniform data codes. Status tags are automatically created before an import is performed and will begin monitoring with the first import. The following information is recorded for each import:

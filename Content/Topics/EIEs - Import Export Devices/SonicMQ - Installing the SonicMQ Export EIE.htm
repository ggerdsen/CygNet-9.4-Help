Installing the SonicMQ Export EIE

To Install the SonicMQ Export EIE

  1. Using CygNet Explorer, open the Device Definition Service (DDS).
  2. Click the Import/Export Devices page.
  3. To add a new SonicMQ Export device, right-click in the white space of your DDS pane, select New Import/Export Device, and select SonicMQ Export.
  4. Configure the device as desired. See Configuring the SonicMQ Export for configuration options.
  5. Save any changes by clicking Apply or OK (Apply saves the changes and keeps the dialog box open; OK saves the changes and closes the box).

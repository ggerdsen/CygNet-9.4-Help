Configuring the CygNet CAS View Control

To Configure the CygNet CAS View Tool

  1. Using CygNet Studio in Edit mode, locate the Standard Tools palette, click the ActiveX tool and place on a CygNet Studio screen, select the CygNet CAS View Control from the list and click the OK button.
  1. Right-click the newly added CAS View Control to open its Property Sheet, click its ActiveX page, then locate the (OCXProperties) and click its corresponding Browse button. A CygNet CAS View Control Properties dialog box opens.
  1. Configure settings on the various CAS View dialog box pages as you require.
    1. Using options on the General page, create all required groups, y-axes, and series. For more information, see Configuring General Properties.
    2. Using options on the Grid page, define the date range to apply to the series you created above. For more information, see Configuring Grid Properties.
    3. Using options on the Sort page, start to define the appearance of the trend graph. For more information, see Configuring Sort Properties.
    4. Using options on the Color page, add and define a series legend. For more information, see Configuring Color Properties.
    5. Using options on the Icon page, define the colors of the graph. For more information, see Configuring Icon Properties.
  2. If you are satisfied with the configuration of your CAS View Control's properties, click Apply and OK to save your settings and close the relevant properties dialog box.

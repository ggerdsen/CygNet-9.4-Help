Using the CygNet CAS View Control in Run Mode

The CygNet CAS View Control has many options available on the right-click context menu in Run mode. Some options let you edit, move, sort or delete items during the current view. When the screen is reloaded, the default options appear from the original configuration. See Alarms for more information about acknowledging, clearing, hiding, and suppressing alarms.

Option Description
Acknowledge

See Acknowledging Alarms.
Acknowledge Visible
Clear

See Clearing and Force Clearing Alarms.
Force Clear
Hide

See Hiding Alarms.
Unhide
Suppress

See Suppressing Alarms.

Point Properties

Opens the Point Properties dialog box for the selected point.

Point Configuration

Select to view and/or edit various point configuration record properties for the selected point:

  • Alarm Settings — opens the appropriate Analog/Digital/Enumeration/String page of the PNT editor.
  • Comments — opens the Comments page of the PNT Editor.
  • All — opens the General page of the PNT Editor.

Alarm History

Opens the Alarm History for the selected point.

Alarm Statistics

Opens the Alarm Service Statistics dialog box about the CAS service.

Alarm Auto Configuration

Opens the Alarm Auto Configuration dialog box to configure automatic rules in Run mode.

Select All

Selects all alarms.

Filter

Opens the Alarm Filter dialog box to edit or create filter rules in Run mode.

Columns

Opens the Alarm Columns dialog box to add/remove columns from the view in Run mode. These changes will not be saved to the screen.

Wrap Columns

Enables or disables the word wrap feature in the columns.

Show Alarm Color

If enabled, displays the configured alarm colors. If disabled, displays all grid information in black and white.

Refresh

Refreshes the data on the grid.
Select Print Range Opens the Select Print Range dialog box to select rows or columns to print.

Print Report

Opens the Print Options to print the current screen.

Sound

Opens the Sound Settings options.

CustomContextMenu

If a custom context menu item is configured it will appear at the bottom of the right-click context menu. See Custom Context Menus for more information about this feature.

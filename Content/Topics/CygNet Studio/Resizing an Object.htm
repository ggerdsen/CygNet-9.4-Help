Resizing an Object

Once you place an object on TheView you can resize it.

To Resize an Object by Dragging

  1. Select the object to be resized.
  2. Position the cursor over one of the object’s side or corner handles. These are shown as green squares along the object’s perimeter. The cursor will change to a double-ended arrow.
  3. Click and drag the handle to resize the object then release the mouse. If Snap to Grid (Layout menu, Grid Settings) is enabled, when you resize the object it will snap to the grid.

To Resize an Object One Pixel at a Time

  1. Select the object to be resized.
  2. Hold down Shift and use the arrow keys to resize the object in one-pixel increments. The top left corner remains fixed. This option ignores Snap to Grid (Layout menu, Grid Settings).

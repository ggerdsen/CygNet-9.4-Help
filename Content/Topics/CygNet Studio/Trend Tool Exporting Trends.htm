Exporting Trends

The Trend Tool control provides various export options to export trend data or a trend image.

The Export option enables you to export a trend to your clipboard, a file directory location, or a printer in one of several file formats, most of which are graphics files.

Note: The Export option is unavailable if you are in Auto Extend mode.

The Quick Export option exports the labels and data of each series directly to a temporary comma-separated value file (CSV).

Export options are also available using script methods. The Export method exports the contents of the grid to a temporary CSV file and the ExportAsImage method allows export of an image of a trend.

See the following subsections for more information:

  • Using the Export Option
  • Using the Quick Export Option
  • Exporting via Script

Using the Export Option

  1. From the Trend Tool control, right-click to access the context menu, and then click Export to access the Exporting dialog box to specify the file type, destination, and size for your graph export.

  2. In the Export box, click a button to select the desired export file type. Options are EMF (Enhanced Metafile Format), WMF (Windows MetaFile format), BMP (Bitmap Image File format), JPG (JPEG - Joint Photographic Experts Group), PNG (Portable Network Graphics), or Text/Data.
  3. In the Export Destination box, click a button to select the destination for your export. Options are Clipboard, File, or Printer (although Text/Data exports do not support the Printer option.)
  4. Do one of the following to complete your selected export.
  1. Export a trend image file to the clipboard.
  1. In the Export Size box, enter parameter values.
  2. Click Export.
  1. Export a trend image file to a file directory location.
  1. In the Export Destination box, click Browse to access the Save As dialog box. Navigate to the desired location for the file, and then click Save.
  2. In the Export Size box, enter parameter values.
  3. Click Export.
  1. Export a trend image file to a printer.
  1. In the Export Size box, enter parameter values.
  2. Click Print to access the Printing Chart dialog box to configure print parameters.
  3. Select the desired values for printer, orientation, paper, printing style, and DPI. Optionally click Setup to configure additional properties for the selected printer and then click OK.
  4. Click OK.
  1. Export a trend text/data file to the clipboard.
  1. Click the Export button to access the Export dialog box to configure text/data parameters.
  2. Select the desired options for data subsets and points, data to export, export style, and decimal precision.
  3. Click Export.
  1. Export a trend text/data file to a file directory location.
  1. In the Export Destination box, click Browse to access the Save As dialog box. Navigate to the desired location for the file, and then click Save.
  2. Click the Export button to access the Export dialog box to configure text/data parameters.
  3. Select the desired options for data subsets and points, data to export, export style, and decimal precision.
  4. Click Export.

Using the Quick Export Option

From the Trend Tool control, right-click to access the context menu, and then click Quick Export to export the labels and data of each series directly to a temporary comma-separated value file (CSV).

Exporting via Script

Exporting trend data can be performed via script using the CxTrend object with the following methods:

  • Export — The Export method exports the contents of the grid to a temporary CSV file and opens the file with the default spreadsheet viewer for the user to view and save to disk.
  • ExportAsImage — The ExportAsImage method allows export of an image of a trend.

For more information, see CxTrend Methods.

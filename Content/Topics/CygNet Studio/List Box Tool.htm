List Box Tool

A List Box is similar to a Combo Box Tool in that the List Box displays a list of items; however, a List Box does not have a drop-down component. Depending upon the size of the List Box, all items may be viewable. Scroll bars (both horizontal and vertical) can be used to scroll through the list.

Only a single item can be selected from a List Box.

Script is required for an action to occur when using a List Box object. See Scripting in CygNet Studio for examples.

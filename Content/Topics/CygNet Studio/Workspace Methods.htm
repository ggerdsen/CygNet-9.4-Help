Workspace Methods

The Workspace object contains the following methods:

  • ExitApp
  • GetFormByViewID
  • ListViewIDs
  • RaiseCustomEvent
  • RaiseGroupCustomEvent

ExitApp

Syntax

ExitApp()

GetFormByViewID

Syntax

GetFormByViewID(P1 As Long) As Object

ListViewIDs

Syntax

ListViewIDs() As Variant

RaiseCustomEvent

Note: The EventCustom event runs in response to this method.

Syntax

RaiseCustomEvent(P1 As String, P2 As Variant, P3 As Long)

RaiseGroupCustomEvent

Note: The EventCustom event runs in response to this method.

Syntax

RaiseGroupCustomEvent(P1 As String, P2 As Variant, P3 As Long, P4 As Long)

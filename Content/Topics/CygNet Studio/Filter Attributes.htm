Filter Attributes

The filter attributes can be selected from the Data Item drop-down menu or the Comparison Type drop-down menu for either Reference Attribute or Source Attribute. The available attributes will differ depending on the application, control, or view where you are setting up your filter. Attributes are organized into the following types:

Special Attributes (used in specific applications, controls, and views)

  • Alarm Attributes (CygNet Explorer CAS View, Alarm Filter and CAS View Control, Alarm Filter)
  • Alarm Record Attributes (CygNet Event Summary Control, ELS Alarm, SiteServices Filter)
  • Audit Property Attributes (CygNet Event Summary Control, AUD, SiteServices Filter)
  • Custom Attributes (CAS View Control, Event Summary Control, Facility Navigation Bar Control)
  • Event Information Attributes (CygNet Event Summary Control, ELS Event, SiteServices Filter, ELS Alarm, SiteServices Filter)
  • Group Attributes (CygNet Group Navigation Bar Control, Navigation Root Filter and Level Filter)
  • History Attributes (VHS Recovery Utility, Journal File Recovery, History Data Filter and Database Recovery, History Data Filter)
  • Note Attributes (Trend Tool Note Rule Filter)
  • Notification Attributes (CygNet Explorer GNS Editor, Notification Message, Reference Tag, Facility Rule Filter)

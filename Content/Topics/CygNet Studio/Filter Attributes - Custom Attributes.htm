Filter Attributes - Custom Attributes

The following filter attributes represent special custom attributes available for configuration and use by filter rules in CygNet Software.

CAS View Control Custom Attributes

CAS View Control Custom Attributes are defined on the General page of the CygNet CAS View Control Properties dialog box.

Attribute Description User Values
Custom Attribute 0-19 These attributes are used for filtering on whether the various custom generic attributes 0-19 are set. Yes or No

Event Summary Control Custom Attributes

Event Summary Control Custom Attributes are defined on the Data page of the CygNet Event Summary Control Properties dialog box.

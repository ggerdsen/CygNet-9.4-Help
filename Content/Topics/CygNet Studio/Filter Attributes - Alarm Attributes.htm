Filter Attributes - Alarm Attributes

The following attributes are available for use throughout CygNet Software.

Attribute Description User Values

Alarm Condition

This attribute is used for filtering on the alarm condition of the point; for example, Analog Low Warning, Digital Alarm, Analog High Out-of-Range, etc.

Text

Alarm Hidden

This attribute is used for filtering on an alarm that is either hidden or unhidden.

Yes or No

Alarm Priority Highest

This attribute is used for filtering on the highest numeric priority for the alarm.

0-99

Alarm Priority Highest Category

This attribute is used for filtering on the highest alarm priority category for the alarm, for example, LOW, CRITICAL, MEDIUM, and HIGH.

Text

Alarm Priority Highest Category Description

This attribute is used for filtering on the description for the highest category of a pointâ€™s alarm, for example, Low Alarm Priority, High Alarm Priority. Check spelling and capitalization.

Text

Alarm Priority Highest Since Ack

This attribute is used for filtering on the highest alarm priority for the alarm since it was acknowledged.

0-99

Alarm Priority Highest Since Ack Cat.

This attribute is used for filtering on the highest alarm priority category for the alarm since acknowledgment (for example, LOW, CRITICAL, MEDIUM, and HIGH).

LOW, CRITICAL, MEDIUM, HIGH

Alarm Priority Highest Since Ack Cat. Desc

This attribute is used for filtering on the highest alarm priority category description for the alarm since it was acknowledged, for example, High Alarm Priority, Critical Alarm Priority, etc.

Text

Alarm Suppression State

This attribute is used for filtering on an alarm that is in a suppression state.

Yes or No

Facility Tag (CAS)

This attribute is used for filtering on the facility tag.

Text

Point State

This attribute is used for filtering on the point state of the alarm (for example, Analog Low Warning, Digital Normal, Unreliable, String Normal, etc.)

Text

Reported Time

This attribute is used for filtering on the time when the alarm was last processed.

Numeric (Time)

Set/Reset State

This attribute is used for filtering on the set or reset state of the alarm.

Yes or No

CygNet Studio

CygNet Studio

CygNet Studio is CygNet’s Human-Machine Interface (HMI) application for creating operational screens to control and monitor devices, display data, alarms, and trends. Data can be displayed textually or graphically. Studio contains all of the features of a drawing application, in addition to a collection of CygNet-aware tools and ActiveX controls.

As a COM- and .NET-compatible container, CygNet Studio also supports the use of third-party ActiveX controls, OPC controls, etc. CygNet Studio also supports VB script. This extension allows further customization and manipulation of data. Script can be written for a screen itself or any tool placed on the screen. A variety of events can be scripted. A set of Microsoft common controls such as an option button, check box, and combo box are integrated into the application and can be used with scripting.

CygNet Vision

CygNet Vision is the run-time companion application to CygNet Studio. Screens created using CygNet Studio can be viewed using CygNet Vision. All of the functionality presented to a user on a CygNet Studio screen is available using CygNet Vision, such as trending capabilities, command buttons, etc.

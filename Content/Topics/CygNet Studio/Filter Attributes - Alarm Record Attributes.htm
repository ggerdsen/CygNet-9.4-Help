Filter Attributes - Alarm Record Attributes

The following filter attributes can be used to filter ELS Alarm Record Information on the Data page of the CygNet Event Summary Control.

  • Alarm Record Information
  • Alarm Record Point State
  • Alarm Record Status Bits
  • Alarm Record Changed Items
  • Alarm Record Durations

Alarm Record Information

Attribute Description User Values
Are Change and Duration Set This attribute is used for filtering on the ELS alarm record for the Change and Duration bit. Yes or No
ELS Alarm Acknowledge This attribute is used for filtering on the ELS alarm record for whether the point’s alarm condition is acknowledged. Yes or No
ELS Alarm Alternate Units This attribute is used for filtering on the ELS alarm record for the alarm alternate units. Text
ELS Alarm Alternate Value This attribute is used for filtering on the ELS alarm record for the alarm alternate value. Alphanumeric
ELS Alarm Category This attribute is used for filtering on the ELS alarm record for the point's alarm category. Text
ELS Alarm Category Description This attribute is used for filtering on the ELS alarm record for the description of the alarm category. Text
ELS Alarm Facility ID This attribute is used for filtering on the ELS alarm record for the point's facility ID. Alphanumeric
ELS Alarm Facility Tag This attribute is used for filtering on the ELS alarm record for the point's facility tag. Alphanumeric
ELS Alarm Full Tag This attribute is used for filtering on the ELS alarm record for the point's full tag. Alphanumeric
ELS Alarm Hidden This attribute is used for filtering on the ELS alarm record for whether an alarm is either hidden or unhidden. Yes or No
ELS Alarm Long Point ID This attribute is used for filtering on the ELS alarm record for the alarm's long point ID. Alphanumeric
ELS Alarm Long Tag This attribute is used for filtering on the ELS alarm record for the point's long tag. Alphanumeric
ELS Alarm Point Data Type This attribute is used for filtering on the ELS alarm record for the point data type for the alarm (for example, AI for Analog Input, AO for Analog Output, DO for Digital Output, etc.) AI, AO, DI, DO, EI, EO, SI, SO
ELS Alarm Point Data Type Desc This attribute is used for filtering on the ELS alarm record for the description of the point data type for the alarm (for example, Analog Input, Digital Output, etc.) Text
ELS Alarm Point ID This attribute is used for filtering on the ELS alarm record for the alarm's Point ID. Numeric
ELS Alarm Point Scheme This attribute is used for filtering on the ELS alarm record for the point scheme used for describing point states and alarm conditions. 0 (or user-customized)
ELS Alarm Primary Units This attribute is used for filtering on the ELS alarm record for the point's primary units. Text
ELS Alarm Primary Value This attribute is used for filtering on the ELS alarm record for the point’s primary value. Alphanumeric
ELS Alarm Priority This attribute is used for filtering on the ELS alarm record for the alarm priority. 0-99
ELS Alarm Record Time This attribute is used for filtering on the ELS alarm record for the point's timestamp. Numeric (Time)
ELS Alarm Service This attribute is used for filtering on the ELS alarm record for the alarm's CygNet service. Text
ELS Alarm Set This attribute is used for filtering on the ELS alarm record for whether the alarm is set. Yes or No
ELS Alarm Site This attribute is used for filtering on the ELS alarm record for the alarm's site. Alphanumeric
ELS Alarm SiteService This attribute is used for filtering on the ELS alarm record for the Site.Service for the alarm. Text
ELS Alarm System Name This attribute is used for filtering on the ELS alarm record for the System name for an alarm. Text
ELS Alarm Tag This attribute is used for filtering on the ELS alarm record for the tag for the point. Alphanumeric
ELS Alarm Uniform Data Code This attribute is used for filtering on the ELS alarm record for the point's uniform data code. Alphanumeric

ELS Alarm Blink Mode This attribute is used for filtering on the ELS alarm record for the alarm blink mode. Numeric

ELS Alarm Foreground Color This attribute is used for filtering on the ELS alarm record for the alarm foreground color. Hexadecimal

ELS Alarm Priority Category Desc This attribute is used for filtering on the ELS alarm record for the alarm priority category description. Text
ELS Alarm Single Color This attribute is used for filtering on the ELS alarm record for the alarm single color. Hexadecimal

Alarm Record Status Bits

Attribute Description User Values
Is ELS Alarm Configurable Bit 01-15 Set This attribute is used for filtering on the ELS alarm record for whether a configurable bit is set. Yes or No
Is ELS Alarm External Value Bit Set This attribute is used for filtering on the ELS alarm record for whether an alarm external value bit is set. Yes or No
Is ELS Alarm Suppression Bit Set This attribute is used for filtering on the ELS alarm record for whether an alarm suppression bit is set. Yes or No
Is ELS Alarm Unreliable Bit Set This attribute is used for filtering on the ELS alarm record for whether an alarm unreliable bit is set. Yes or No
Is ELS Alarm User Bit 01-16 Set This attribute is used for filtering on the ELS alarm record for whether an alarm user bit is set. Yes or No

Alarm Record Changed Items

Attribute Description User Values
Has Alarm Priority Changed This attribute is used for filtering on the ELS alarm record for whether an alarm priority has changed. Yes or No
Has Configurable Bit 01-15 Changed This attribute is used for filtering on the ELS alarm record for whether an alarm configurable bit has changed. Yes or No
Has Point Scheme Changed This attribute is used for filtering on the ELS alarm record for whether an alarm point scheme has changed. Yes or No
Has Point Type Changed This attribute is used for filtering on the ELS alarm record for whether an alarm point type has changed. Yes or No
Previous Alarm Priority This attribute is used for filtering on the ELS alarm record for whether a previous alarm priority has changed. Yes or No
Previous Point Scheme This attribute is used for filtering on the ELS alarm record for whether an alarm previous point scheme has changed. Yes or No
Previous Point Type This attribute is used for filtering on the ELS alarm record for whether an alarm previous point type has changed. Yes or No

Alarm Record Durations

Attribute Description User Values
Seconds Configurable Bit 01-15 Has Been Set This attribute is used for filtering on the ELS alarm record for the duration in seconds that a configurable bit has been set. Numeric
Seconds Point Has Been In CAS This attribute is used for filtering on the ELS alarm record for the duration in seconds that an alarm has been reported to the CAS. Numeric
Seconds Point Has Been In Alarm This attribute is used for filtering on the ELS alarm record for the duration in seconds that a point has been in alarm. Numeric

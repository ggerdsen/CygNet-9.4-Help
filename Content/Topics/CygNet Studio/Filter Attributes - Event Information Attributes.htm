Filter Attributes - Event Information Attributes

The following filter attributes can be used to filter ELS Event Information on the Data page of the CygNet Event Summary Control.

Event Information Attributes

Attribute Description User Values
ELS Event Type This attribute is used for filtering on the ELS event type. User values are Service, Operations, Alarms, External.

See Event Logging Service (ELS).
ELS Event Type Description This attribute is used for filtering on the ELS event type description. Text
ELS Source This attribute is used for filtering on the ELS source. Alphanumeric
ELS Category This attribute is used for filtering on the ELS category. Text
ELS Event Time This attribute is used for filtering on the ELS event time. Numeric (Time)
ELS Description This attribute is used for filtering on the ELS description Alphanumeric
ELS Details This attribute is used for filtering on the ELS details. Alphanumeric
ELS Comments This attribute is used for filtering on the ELS comments. Text
ELS User Flag 01-10 This attribute is used for filtering on the ELS user flags. Yes or No

Configuring the Trend Tool

Note: You can change configuration settings for this tool in Run mode. However, these settings usually only persist for the current runtime session. When you close the CygNet Studio screen on which the tool resides, the tool settings may revert to those originally configured in Edit mode.

The Trend Tool has numerous properties available for configuration. The following procedure describes a basic workflow for configuring a Trend Tool. The specific appearance and functionality of the Trend Tool you create depends on your needs and implementation of the tool's properties.

You can embed numerous Trend Tools in TheView, each containing up to a total of 16 y-axes and any number of series (plotted lines). Each series may include as many data points as are available for a given point in the Value History Service (VHS). For more information, see Using a ListView Control to Select Trends to Display.

For a useful example of a Trend Tool implemented in a pre-designed screen, open UDC_TRND.csf in the APPS SCREXAMP folder.

For information about templated trends, see Using Template Screens.

To Configure the Trend Tool

Note: The properties of the Default Trend and Trend Tool are identical except for text in each one's title bar. The process for configuring these tools is therefore nearly identical.

  1. Using CygNet Studio in Edit mode, follow one of these sets of steps:
    • To add a Default Trend, right-click on TheView, locate the [DefaultTrend] property, and click its corresponding Browse button. A Default Trend Properties dialog box opens.
    • To add an embedded Trend Tool, locate the Graphics Tools palette, click the Trend Tool icon, then click on TheView. Right-click the newly added Trend Tool to open its Property Sheet, click its ActiveX page, then locate the (OCXProperties) and click its corresponding Browse button. A CygNet Trend Control Properties dialog box opens.
  1. Configure settings on the various trend property pages as you require.
    1. Using options on the Series page, create all required groups, y-axes, and series. See Configuring Groups, Axes, and Series.
    2. Using options on the Data page, define the date range to apply to the series you created above. See Configuring a Date Range.
    3. Using options on the Graph page, start to define the appearance of the trend graph. See Configuring a Graph.
    4. Using options on the Legend page, add and define a series legend. See Configuring a Legend.
    5. Using options on the Colors page, define the colors of the graph. See Configuring Colors.
    6. Using options on the Font page, define the font of the text items of the graph. See Configuring Font.
    7. Using options on the Toolbar page, define the appearance and behavior of the toolbar. See Configuring a Toolbar.
    8. Using options on the Alarm Limits page, define the alarm limit lines of your graph. See Configuring Alarm Limits.
    9. Using options on the Display Information page, define the display behavior of a trend. See Configuring Display Information.
  2. If you are satisfied with the configuration of your Trend Tool's properties, click Apply and OK to save your settings and close the relevant trend properties dialog box.

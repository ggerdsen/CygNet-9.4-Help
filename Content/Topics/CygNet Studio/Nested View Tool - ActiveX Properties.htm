Nested View Tool - ActiveX Properties

The Nested View Tool is an ActiveX control. As such, it has an additional set of properties that are listed on the ActiveX page on the Property Sheet. Many of these properties can be set in the OCX Properties dialog box, which is accessed by clicking the (OCXProperties) property.

Property Description

AllowUnitsChange

Enables the Units of the screen shown in the Nested View object to be changed. Default is 1 - Yes.

Appearance

Specifies the appearance of the object boundary. Default is 0 - Flat. Other option is 1 - 3D.

BorderStyle

Specifies the border style of the object. Default is 0 - None. Other option is 1 - Fixed Single.

Facility

Specifies the value of the [Facility] property for the screen in the Nested View.

FilePath

Specifies the file path to the screen shown in the Nested View. A user must have at least "read" access to the file.

ScrollbarOverride

Specifies the display of scroll bars on the screen in the Nested View. Default is 1 - No Override, which means that the ScrollBars property of the nested view takes precedence. If set to 0 - No or 1 - Yes, scroll bars on the nested view are enabled or disabled, respectively, regardless of the ScrollBars property of the nested view. If set to 2 - Scrollbars with Flex, scroll bars are shown on the nested view.

SiteService

Specifies the value of the [SiteService] property for the screen in the Nested View.

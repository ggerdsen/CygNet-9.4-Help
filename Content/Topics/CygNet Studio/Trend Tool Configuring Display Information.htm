Trend Tool – Configuring Display Information

Note: You can change configuration settings for this tool in Run mode. However, these settings usually only persist for the current runtime session. When you close the CygNet Studio screen on which the tool resides, the tool settings may revert to those originally configured in Edit mode.

The Display Information page specifies the mode by which to display certain kinds of information on a trend.

Example Trend Tool Display Information Page

Display Information Page Properties

The following table lists and describes Display Information page properties.

Property Description

Mode

Enables you to select from the following information display modes:

  • None: No display information appears.
  • Time Slice Values: Click a plotted point to display a value, date, and time, and to display a vertical line for easy x-axis location. See also AnnotationColor and MonoAnnotationColor.
  • Series Tags: Click a plotted point to display its respective tag string. See also AnnotationColor and MonoAnnotationColor.
  • Values at Cross Hairs: Click a plotted point to place it in cross hairs and display its date, time, and value in the upper left-hand corner of the trend window.
  • Time Slice Value Table: Click a plotted point to display a value, date, and time in table format to the right or the trend, and to display a vertical line for easy x-axis location.
  • Notes: Enables you to view in-scope notes on a trend. See Using CygNet Filter Rules and Notes.

Note Filter

Enables you to create and apply a note filter at design time.

See Using CygNet Filter Rules, Note Attributes, and Notes.

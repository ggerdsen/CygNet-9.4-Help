Filter Attributes - History Attributes

The following filter attributes can be used to filter history data to be recovered in the VHS Recovery utility, Journal File Recovery and Database Recovery, History Data Filter.

Attribute Description User Values
Facility ID This attribute is used for filtering on the facility of a point tag. Text
LongPointID This attribute is used for filtering on the long point ID of a point tag. Text
PointID This attribute is used for filtering on the point ID of a point tag. Text
Service This attribute is used for filtering on the service of a point tag. Text
Site This attribute is used for filtering on the site of a point tag. Text
SiteService This attribute is used for filtering on the site and service of a point tag. Text
Timestamp This attribute is used for filtering on the timestamp of a point tag. Text
Uniform Data Code This attribute is used for filtering on the uniform data code (UDC) of a point tag. Text

Slider Tool - ActiveX Properties

The Slider Tool is an ActiveX control. As such, a Slider Tool object has an ActiveX page on the Property Sheet. Many of these properties are the object’s formatting properties. Values for the ActiveX Properties can be set directly on the ActiveX page or in the OCX Properties dialog box, which is accessed by clicking the (OCXProperties) property.

Property Description Related Properties

AutoCenter

Enables automatic centering of the component within its boundary. Default is 1 - Yes. If set to 0 - No, the component is aligned left in the boundary. OffsetX and OffsetY can be used to move the placement of the component in the boundary.

BackGroundPicture

Note: Although Windows Metafile (.wmf) is listed as a format option, this file format is not supported.

  

BorderStyle

Specifies the border style of the object. Default is 0 - ibsNone. Other options are 1 - ibsRaised and 2 - ibsLowered.

  

CurrentMax

Specifies the maximum value reached by the Position property. It is reset each time the Position value is greater than the CurrentMax value. The only way to access the value is programmatically (i.e., using script).

Position

CurrentMin

Specifies the minimum value reached by the Position property. It is reset each time the Position value is less than the CurrentMin value. The only way to access the value is programmatically (i.e., using script).

Position

Enabled

Enables user interaction with the object. If set to 0 - No, the object is still shown but its context menu is disabled. Default is 1 - Yes.

  

EndsMargin

Specifies the margin (in pixels) between the indicator ends and the object boundary. Default is 10.

  

Height

Specifies (in pixels) the height of the object.

  

KeyArrowStepSize

Specifies the increment applied to the slider position when the arrow keys are pressed. Default is 1.

  

KeyPageStepSize

Specifies the increment applied to the slider position when the Page Up or Page Down keys are pressed. Default is 10.

  

MinMaxFixed

Not applicable for this tool. Specifies whether the minimum and maximum pointers are fixed or will track the range of the Position property. Default is 0 - No.

Position

MouseControlStyle

Specifies how the pointer responds to a mouse click in the track. Default is 0 - ismcsSlideOnly. This option requires the user to slide the pointer to a new location. 1 - ismcsSlidePage allows the user to click in the track. The pointer will move in the direction of the click by the increment specified by the KeyPageStepSize property. 2 - ismcsGoto also allows the user to click in the track and the pointer is moved to the location of the click.

KeyPageStepSize

MouseWheelStepSize

Specifies the adjustment to the slider position when the mouse wheel is moved one increment. Default is 1.

  

OffsetX

Specifies the offset of the object from its x-axis. Default is 0.

  

OffsetY

Specifies the offset of the object from its y-axis. Default is 0.

  

OptionSaveAllProperties

Enables the SaveAllPropertiesToFile method to save just the ActiveX Properties or all object properties. Default is 0 - No, which means that only ActiveX Properties are saved.

  

Orientation

Specifies the orientation of the indicator. The default is 0-ioVertical.

 

OrientationTickMarks

Specifies the location of the tick marks in relation to the indicator. If the object orientation is vertical, the option is the left or right side. If it is horizontal, the option is top or bottom. Default is 0 - iosBottomRight.

  

PointerBitmap

Specifies the bitmap to be shown in place of the pointer. The PointerStyle property must be set to Bitmap. For best results, match the background color of the bitmap to the color of the track.

PointerStyle

PointerStyle

TickMinorCount

TrackStyle

Specifies the track style. Default is 0 - istsBox. Other options are 0 - istsBox, 1 - istsLine, 2 - istsBevelLowered, 3 - istsBevelRaised.

Enables transparency of the object background. Default is 0 - No.

  

UpdateFrameRate

Specifies the rate (in seconds) of repainting the object. Default is 60. Applicable only when AutoFrameRate is set to 1 - Yes.

AutoFrameRate

Specifies the width (in pixels) of the object.

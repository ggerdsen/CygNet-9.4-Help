This section contains examples in a tutorial format, illustrating the use of script with standard tools. The following list may be used as a reference to see which tools are included in which examples.

Not all examples are applicable to all installations. For instance, those using the Group Navigation Bar require a group navigation hierarchy, and many examples are best illustrated with live data.

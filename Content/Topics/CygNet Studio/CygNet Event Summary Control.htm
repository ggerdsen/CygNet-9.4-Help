CygNet Event Summary Control

The CygNet Event Summary Control is an ActiveX control used to view Event Logging Service (ELS) events, alarm history, and Audit service (AUD) information on a CygNet Studio screen. This control is highly configurable so that many types of data can be displayed in a single control. Use this control to filter and run ad-hoc reports for Audit and Logging services.

Because the control is limited to 20,000 records from each source (ELS, ELSALM, and AUD, for a total of 60,000) it is recommended to use the control's filtering capabilities to limit results to stay below these limits. Separate filters can be specified for both ELS (and ELSALM) and AUD data, and returned record results can be printed or exported. See Configuring the CygNet Event Summary Control for more information about filtering.

The control consists of a record grid with date and filter options in a toolbar. One possible use of this control is to create two summary controls: one for ELS records and one for AUD records. The example below shows ELS records.

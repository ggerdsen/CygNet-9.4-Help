Using the CygNet Event Summary Control in Run Mode

In Run mode, the CygNet Event Summary Control is dynamically configurable and allows you to access the properties for individual records, filter the data, export the grid to a spreadsheet, and to print the grid.

By default, the CygNet Event Summary Control includes a toolbar indicating the data range, several buttons, the number of records in the grid.

To View Record Properties

  • In Run mode, right-click in one of the record’s cells, then select Properties. This will launch the properties editor for that record type (ELS or AUD) that is viewable in CygNet Explorer.

To Change Filters, the Date Range, or Service Names

  1. In Run mode, right-click in the grid and select Filters, or click … in the toolbar. This displays the same dialog box used in Edit mode to configure these options. See Configuring Data for more information.
  2. Make the desired changes then click OK.

To Export the Grid to a Spreadsheet

  • In Run mode, right-click the grid and select Export. The exported data will launch in the default spreadsheet editor.

To Print the Grid or a Range of Data

  • In Run mode, right-click the grid and select Select Print Range…. Specify the desired range in the Select Print Range dialog box. Click OK.
  • In Run mode, right-click the grid and select Print. Specify a printer and click OK.

To Page through Date/Time Ranges

  • In Run mode, click the Back button to move back in time or the Forward button to move forward in time. The paging interval is the difference between the configured earliest and latest dates.

To Order Records by a Column

  • In Run mode, click the header of the column by which the records are to be ordered. Click the header again to toggle ascending/descending sort mode.

To Hide/Show or Reposition the Toolbar

  • In Run mode, right-click the grid and select Toolbar and select the desired location for the toolbar. Options include Top, Bottom, or None.

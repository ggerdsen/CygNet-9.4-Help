Filter Attributes - Group Attributes

The following filter attributes can be used to filter the root node or the drop-down level in a Group Navigation Bar object.

  • Group: General Attributes
  • Group: Text Attributes
  • Group: Yes/No Attributes
  • Group: Application Attributes
  • Group: Admin Attributes

Group: General Attributes

Attribute Description User Values

Category

This attribute is used for filtering on the node category for a selected node. A node category indicates the position and role of a node within a hierarchy.

Root Group (R):

Sub Group (S):

Leaf Node (L):

Admin Node (Z):

Description

This attribute is used for filtering on the node description.

Text

Hierarchy Root

This attribute is used for filtering on the selected node's hierarchy root node.

node ID/data key

Node ID

This attribute is used for filtering on the selected node.

node ID/data key
Reference ID

This attribute is used for filtering on the ID of the item referenced by a reference entity.

 Alphanumeric

Reference Site

This attribute is used for filtering on the reference site for a reference entity associated with a node.

Alphanumeric

Reference Type

This attribute is used for filtering on the reference type for a reference entity associated with a node.

None (~); Facility-DDS (F); Facility-CVS (E); Device (D); Point (P); Node Type (N); Node Ref (R); Service (S); Info (I)

Security Application

This attribute is used for filtering on the security application that applies to a node.

Text

Security Event

This attribute is used for filtering on the security event that applies to a node.

Text

Type

This attribute is used for filtering on the node type for a selected node.

Generic (~G); Summary (~U); Facility (~F); Device (~D); Point (~P); Web Root (~W); Navigation Root (~V); Component Def. Head (~L); Component Def. (~B); Fac Type Def. Head (~T); Fac Type Def (~C); Attribute Def. Head (~J); Attribute Def. (~A); Hierarchy Def. Head (~K); Hierarchy Def. (~H); Report Def. Head (~Q); Report Def. (~R); Storage (~S)

Group: Text Attributes

Attribute Description User Values

Attribute 0-21

These attributes are used for filtering on the various node-specific attributes from 0 to 21. These are attributes with alphanumeric string values.

Text

Group: Yes/No Attributes

Attribute Description User Values

Yes/No 0-19

These attributes are used for filtering on the various node-specific yes/no attributes from 0 to 19. These are attributes with only a "Yes" or "No" value.

Yes or No

Group: Application Attributes

Attribute Description User Values
Application 0-5 These attributes are used for filtering on the various node-specific applications from 0 to 5. Text

Group: Admin Attributes

Attribute Description User Values
Auto Maintain Obsolete  
Aux Data CRC This attribute is used for filtering on the CRC of a node's auxiliary data.  
Aux Data Length This attribute is used for filtering on the length in bytes of a node's auxiliary data. Numeric
Aux Data Timestamp This attribute is used for filtering on the timestamp of the last time auxiliary data was edited and saved. Numeric (timestamp)
Has Aux Data This attribute is used for filtering on whether or not a node has auxiliary data.

Yes or No
Node Visible

This attribute is used for filtering on the whether the nodes' records appears in the GRP.

 Yes or No

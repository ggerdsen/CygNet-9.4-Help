Modes of Operation

CygNet Studio has two modes of operation: Run mode and Edit mode. In the Run mode, screens can be viewed but cannot be edited. In the Edit mode, screens can be edited, previewed, and saved. CygNet Vision has only one mode, which is equivalent to Studio’s Run mode.

When switching from Edit mode to Run mode, you will be prompted to save any unsaved changes to the file. If you do not save them they will be discarded.

When creating a new screen, the default mode is Edit. When opening existing screens, the default mode is Run. You can open an existing screen in Edit mode by holding the Shift key when opening the file, or by clicking Open Standard File for Edit or Open Blob File for Edit on the File menu.

Edit mode also contains a Preview option on the View menu. Use this option to view your changes to the screen without having to first save them. The preview shows exactly what the screen will look like in Run mode, with all functionality available. The preview is a one-time snapshot of the screen. It does not update as you add new objects to the screen. You can have more than one preview open to compare two versions of the screen. To close a preview window click the Close button ("X") in the preview window’s title bar.

Run mode also has many built-in features. These include setting temporary changes to properties of TheView (update interval, current value service, and facility), viewing and temporarily modifying the screen’s default trend, showing trends for individual items, accessing a point’s history values and alarm history, and accessing point configuration information.

In both Run mode and Edit mode multiple screens can be open at the same time. This can facilitate switching between different data displays. Use the Window menu to see a list of screens that are currently open.

Switching Modes

To Switch Between Run Mode and Edit Mode

  • On the View menu, click Switch to Run Mode or Switch to Edit Mode.
  • Use the keyboard shortcut Ctrl+E to shift from one mode to the other.
  • On the Layout toolbar, click the Edit button to switch from Edit mode to Run mode. This option is available only in Edit mode.

Using Full Screen

Both Run mode and Edit mode have a full screen option. This can be turned on by clicking Full Screen on the View menu, or toggled on/off by pressing F11 on your keyboard.

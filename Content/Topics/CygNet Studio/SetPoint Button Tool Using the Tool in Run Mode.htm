Using the SetPoint Button in Run Mode

The SetPoint Button Tool has numerous Run mode options. To access available Run mode options, right-click on the button object in Run mode to open its context menu.

Note: You can change configuration settings for this tool in Run mode. However, these settings usually only persist for the current runtime session. When you close the CygNet Studio screen on which the tool resides, the tool settings may revert to those originally configured in Edit mode.

Run Mode Options

The following table lists and describes the controls available in Run mode for the SetPoint Button Tool.

Option Description

Current Value...

Select to view and/or edit the point's Current Value on the Point Properties dialog box.

Note: This option is only available if the Facility and UDC is specified in the button property sheet (not the SetPoint Properties dialog box).

Trend

Select to trend the point's values on a Default Trend. Options include:

  • Single Point
  • Single Point with Rollups
  • Add to Default Trend
  • Remove from Default Trend

See Tool-Specific Trends for configuration details.

Note: This option is only available if the Facility and UDC is specified in the button property sheet (not the SetPoint Properties dialog box).

History

Select to view and/or edit the target point's History Values or Alarm History.

Note: This option is only available if the Facility and UDC is specified in the button property sheet (not the SetPoint Properties dialog box).

Tool Configuration...

Click to open the CygNet SetPoint Button Configuration dialog box where you can view the configuration settings for the button properties, including runtime-resolved (or unresolved) values of several SetPoint specific properties, if configured:

  • SetPoint Default Value — resolved default value
  • SetPoint Default Value Point Tag — default value point tag
  • SetPoint Prompt Analog Maximum Point Tag — prompt analog maximum point tag
  • SetPoint Prompt Analog Maximum Value — resolved analog maximum value
  • SetPoint Prompt Analog Minimum Point Tag — prompt analog minimum point tag
  • SetPoint Prompt Analog Minimum Value — resolved analog minimum value
  • SetPoint Prompt Units — resolved units
  • SetPoint Status Initial Value — resolved status initial value
  • SetPoint Status Point Tag — resolved status point tag
  • SetPoint Target Point Tag — resolved target point tag

Point Configuration

Select to view or edit various point configuration record properties for the point:

  • Comments — opens the Comments page of the PNT Editor
  • Alarm Settings — opens the appropriate Analog/Digital/Enumeration/String page of the PNT editor to access alarm settings
  • All Settings — opens the General page of the PNT Editor
  • Questionable State — opens the Edit Questionable State dialog box (To configure the Questionable/Verified flag use the Application page of the PNT Editor.)
  • Alarm Suppression Settings — opens the Alarm Suppression Settings dialog box (To configure alarm suppression, use the Analog / Digital / Enumeration / String page of the PNT Editor.)

See Configuring Alarm Settings for more information.

Note: This option is only available if the Facility and UDC is specified in the button property sheet (not the SetPoint Properties dialog box).

Facility Configuration...

Select to view and/or edit the point's facility configuration.

Note: This option is only available if the Facility and UDC is specified in the button property sheet (not the SetPoint Properties dialog box).

SetPoint Configuration...

Select to view a read-only (dimmed) view of the original SetPoint Button configuration properties.

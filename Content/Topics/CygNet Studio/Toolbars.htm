Standard Toolbar

The Standard toolbar contains file, clipboard, printing, and help functions found in most Microsoft Windows programs.

Tool Command Description Menu Shortcut

New

Creates a new file.

File > New

Ctrl+N

Open

Opens a Standard or a Blob file.

File > Open Standard File

File > Open Standard File for Edit

File > Open Blob File

File > Open Blob File for Edit

Ctrl+O

Ctrl+Shift+O

Ctrl+B

Ctrl+Shift+B

Save

Saves a file.

File > Save

Ctrl+S

Cut

Adds selected object or text to the clipboard and removes it from the current context.

Edit > Cut

Ctrl+X

Copy

Adds selected object or text to the clipboard.

Edit > Copy

Ctrl+C

Paste

Pastes the contents of the clipboard into the current context.

Edit > Paste

Ctrl+V

Print

Prints the screen or script, depending on view mode.

File > Print

Ctrl+P

View Trend

(Run mode) Displays a new window showing the View’s default trend. See Default Trend for more information.

View > View Trend

  

About

Shows information about the client application.

Help > About CygNet Studio

  

Font Toolbar

The Font toolbar includes commands for editing text and other properties of objects. The availability of each button is determined by the type of object selected. Properties set by these buttons are also available in the object’s property list (see Properties).

Tool Command Description

Font Type

Change the font type of the selected text.

Font Size

Change the font size of the selected text.

Bold

Toggles the "Bold" attribute of the selected text.

Italic

Toggles the "Italic" attribute of the selected text.

Left

Left justifies the selected text.

Center

Centers the selected text.

Right

Right justifies the selected text.

BackColor

Sets the background color of the selected object(s).

ForeColor

Sets the foreground color of the selected object(s).

Pen Width

Sets the pen width of the selected object(s).

Pen Style

Sets the pen style of the selected object(s).

Arrowhead

Sets the arrowhead attribute of the selected line or polyline object(s).

Hatch Style

Sets the hatch style of the selected object(s).

Shadow Style

Sets the shadow style of the selected object(s).

Format Toolbar

The Format Toolbar is used for setting view parameters and formatting objects. It contains commands for zooming, layers, object rotation, and point editing (of polyline objects).

Tool Command Description Menu Shortcut

Zoom Level

Zoom screen in and out to 50%, 100%, 150%, 200%, 300%, or 400%.

Draw > Zoom In

Draw > Zoom Out

F8

Shift+F8

Zoom 1:1

Resets zoom to 100%.

Draw > Zoom > Zoom 1:1

Ctrl+F8

Current Layer

Choose the layer to display.

Layout > Layers

F4

Rotate Angle

Choose the rotation angle for the selected object.

Options are as follows.

  • 0.0
  • +90.0
  • -90.0

Draw > Rotate > Rotate Right

Draw > Rotate > Rotate Left

Ctrl+.

Ctrl+,

Free Rotate

Enables free rotation handles around the selected object.

Draw > Rotate > Free Rotate

Ctrl+R

Edit Points

Enables point editing of selected polyline shape.

Draw > Contour > Edit Points

Script Toolbar

The Script toolbar takes the place of the Layout toolbar when a screen is in Script Mode (that is, when the Script Editor is visible instead of the View layout). This bar includes editing capabilities as well as some common functions from the Layout bar.

The Script toolbar is customizable. Click the Customize button.

Tool Command Description Menu Shortcut

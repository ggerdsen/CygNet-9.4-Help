WatchList Tool - Configuring Notifications

The WatchList notification message is configured in the OCX properties dialog box. The Edit Notification Message dialog box allows tokens to be used to display message information about the items being displayed in the WatchList object. You can select multiple tokens, enter text if desired to assist with the displaying of data.

To Configure a WatchList Notification Message

  1. In Edit mode, show the properties for the WatchList object.
  2. Click the Edit Notification Message button.
  3. Type the notification message in the Edit Text box.

Note: The text in the notification message can be dynamic, static, or a combination. Dynamic text is built using tokens, which can represent point configuration, facility, and CVS point record properties. Data formatting can be specified. See Using Text Tokens for more information about building dynamic text using tokens.

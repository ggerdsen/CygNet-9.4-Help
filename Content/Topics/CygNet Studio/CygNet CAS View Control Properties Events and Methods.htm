CygNet CAS View Control – Properties, Events, and Methods

The following tables list properties, events, and methods for the CygNet CAS View Control.

  • The first table lists normal properties, events, and ActiveX properties as they appear on the tool's Property Sheet.
  • The second table lists normal properties, normal methods, ActiveX properties, and ActiveX methods as they appear in the tool's Script Editor.

See CxHmiCas3 Object in the Scripting section for additional details about Properties, Events, and Methods specific to this control.

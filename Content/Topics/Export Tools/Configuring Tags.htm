Configuring Tags

Reports are generated based on points. The Tags page is used to specify the IDs of tags for which you wish to retrieve data. You can select points by facility/UDC; tag strings; or you can import a list or read the list of points from a file. Keep in mind that only one point selection option is active.

The options available on this page differ depending on the type of report selected on the Values page. The point selection options include:

  • Facility Specifier
  • Facilities and UDCs
  • Tag List
  • Read from File at Runtime

Facility Specifier

Select this option to configure point selection based on UDCs and facility attributes.

To Select Points by UDC

You can select any UDC(s) from the Table Reference Service (TRS).

  1. Click to open the Select UDC by Category dialog box.
  2. Select the desired category from the UDC category drop-down menu.
  3. Choose desired UDCs.
  4. Click Select UDC to add UDCs to the Tags page.
  5. Double-clicking on the Filter cell for the UDC and configure as necessary.
  6. Select the Rollup method from the drop-down menu. For Rollup Type definitions and information about how history values are used in rollups, see History Rollups.
  7. Double-clicking on the Time cell for the UDC and configure as necessary.

To Select Points by Facility Attributes

  1. Click … to select or change the Facility Site Service.
  2. Click … to filter facilities. You can specify up to six facility attributes. Keep in mind that the attributes are combined using an "and" operator, so the more attributes specified the narrower the list.
  3. Select to include/exclude device facilities, as desired.

Facilities and UDCs

Select this option to configure point selection based on UDCs and specific facility IDs.

To Select Points by UDCs

See instructions above to select UDCs.

To Select Specific Facilities

You can select any facility ID(s).

  1. Click to open the Select a Facility dialog box.
  2. Choose desired facility IDs.
  3. Click OK.

Tag List

Select this option to configure point selection based on a list of points. The points can be individually selected or imported from a flat file.

To Select Points

You can select points by Point ID, Long Point ID, or Facility/UDC. When selecting by Facility/UDC, you can select points for a specific facility or all facilities.

  1. Click to open the Select Point dialog box.
  2. Choose desired points from the left pane.
  3. Click Add to add the point tag to the Selected List pane.
  4. Click OK.
  5. Double-clicking on the Filter cell for the UDC and configure as necessary.
  6. Select the Rollup method from the drop-down menu. See History Rollups for more information.
  7. Double-clicking on the Time cell for the UDC and configure as necessary.

To Import a Point List

The list of points must be in a source file (.txt or .csv). Each line in the file must contain a single point identifier. Any valid CygNet tag string format can be used. The tags in the import file must be listed one per row as shown in the example below.

  1. Click to find the import file.
  2. Click Open to import the tags in the import file.

Read from File at Runtime

Select this option to read a list of points from a flat file (.txt or .csv). Each line in the file must contain a single point identifier.

To Read from a File

  1. Click … to specify the path to the import file.
  2. Select the Rollup Type from the drop-down menu. See History Rollups for more information.

The format for the file consumed by the Read from File at Runtime feature includes no header and a different tag on each line, using the tag, long tag, or full tag formats. Any valid CygNet tag string format can be used. For example, files containing any of the following would work:

CYGDEMO.OPCIS.00000031

CYGDEMO.OPCIS:BUCKET_BRIGADE_ARRAYOFSTRING

CYGDEMO.OPCIS.00000034:BUCKET_BRIGADE_INT1

ACME.UIS::TEST001.GOAL

ACME.UIS::TEST001.VGY

ACME.UIS::TEST001.RGAS

ACME.UIS::TEST002.GOAL

ACME.UIS::TEST002.VGY

ACME.UIS::TEST002.RGAS

ACME.UIS::TEST003.GOAL

ACME.UIS::TEST003.VGY

ACME.UIS::TEST003.RGAS

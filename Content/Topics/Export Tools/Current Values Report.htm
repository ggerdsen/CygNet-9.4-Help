Current Values Report

The Current Values report retrieves a single value for each specified point. The value can be the point’s current value or a historical value.

To retrieve a value from the VHS, you must specify the data timestamp using the Time column on the Tags page. For historical data, the system will return the value closest to without exceeding the specified timestamp.

To Set Up a Current Value Service Report

  1. Open CygNet Excel Export.
  2. Click the Configure button.
  3. On the Values page, enable Current Values.
  4. Select the CVS Site Service of the current value service that contains the data. Any type of current value service can be used (UIS, HSS, SVCMON, OPCIS).
  5. On the Tags page, select the points.

    6. Note: Optionally, to retrieve a historical value instead of a current value, double-click the Time cell for the item and enter the timestamp in the Value at Time field. The value closest to without exceeding the timestamp will be returned.

  1. On the Columns page, select the point properties to be included in the report.
  2. On the Format page, set the report format.
  3. On the Export page, specify the name of the .csv file data will be written to and set the file extension option.
  4. Click OK to exit the Setup box.
  5. Click Get CygNet Data to run the report.
  6. Save the file.

System UDCs

Each Line Pack device has a set of status points that are automatically generated when the device is created. The values in these points always pertain to the most recent execution. To see a history of the results, enable trending on the points.

UDC UDC Description Description Point Type
SYLP_ACD Avg curr day calc cnt

The number of Line Pack values that contributed to the Pack: Average Current Day calculation.

 Analog Input
SYLP_ACH Avg curr hour calc cnt

The number of Line Pack values that contributed to the Pack: Average Current Hour calculation.

 Analog Input
SYLP_ACM Avg curr month calc cnt

The number of Line Pack values that contributed to the Pack: Average Current Month calculation.

 Analog Input
SYLP_AD Avg delta avg cnt

The number of delta Line Pack values that contributed to the Pack: Avg Delta Last Hour calculation.

 Analog Input
SYLP_MCD Max curr day calc cnt

This point indicates if a Line Pack value was written to Pack: Maximum Current Day. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYLP_MCH Max curr hour calc cnt

This point indicates if a Line Pack value was written to Pack: Maximum Current Hour. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYLP_MCM Max curr month calc cnt

This point indicates if a Line Pack value was written to Pack: Maximum Current Month. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYLP_XCD Min curr day calc cnt

This point indicates if a Line Pack value was written to Pack: Minimum Current Day. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYLP_XCH Min curr hour calc cnt

This point indicates if a Line Pack value was written to Pack: Minimum Current Hour. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYLP_XCM Min curr month calc cnt

This point indicates if a Line Pack value was written to Pack: Minimum Current Month. 0 indicates no value was written, 1 indicates a value was written.

 Analog Input
SYEDFOP Failed calcs The total number of failed calculations. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYEDIPOP

 Pending calcs The number of calculations pending or in progress. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYEDSOP Successful calcs The number of successful calculations. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPCRT

 Last op calc time The time (in seconds) it took to execute the last calculation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPERR

 Last op error cnt The number of errors during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPRCNT

 Last op rollup cnt The number of points that were set by the rollup during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPRT

 Last op runtime The total runtime (in seconds) of the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPRURT

 Last op rollup time The time it took (in seconds) to roll up the results during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYLPWARN Last op warning cnt The number of warnings during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYMBERR Last op massBal errs The number of Mass Balance related errors generated during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYMBMS Last op missing segs The number of missing segments during last Mass Balance calculation process. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYMBUS Last op unassig segs The number of unassigned segments during last Mass Balance calculation process. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input
SYMBWARN Last op massBal warn The number of Mass Balance related warnings generated during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

Configuring UDC Mappings

There are several required and optional points that need to be configured with UDCs, which are used as part of the calculation and for writing out the actual pack calculation. These "required" and "optional" UDCs are configured on the UDC Map page of the device editor. There are several different types of UDC mapping options, including:

  • required point record configuration input UDCs for each pack segment/sub-segment (status, volume, length, diameter)
  • UDCs for writing out the historicized values of inputs used in the calculations (listed as Last Segment Input:)
  • UDCs for recording calculated intermediate values used in the line pack calculation (listed as Last Segment Calc:)
  • UDCs for writing out average pack values
  • UDCs for the writing out the actual pack calculation (listed as Pack: or MAOP Pack: (current, deltas, minimums, maximums, and averages)).

The points listed in this page differ depending on the contents of the pipeline. See the following topics for descriptions of the required and optional points:

  • Natural Gas UDCs
  • Supercritical CO2 UDCs
  • Petroleum Liquids UDCs

Required segment/sub-segment points

The Required list contains the UDC mappings for all required points.

A Note About Ordinalized Points: The items Last Segment Input: Temperature (Ordinal) and Last Segment Input: Pressure (Ordinal) map the base UDC for these ordinalized points. If multiple temperature and/or pressure inputs have been configured on the Calculations page, then there will need to be a Last Segment Input point for each temperature or pressure input. The form of these UDCs will be the base UDC mapped in the list above with an ordinal appended to the end.

Example

If there are three temperature inputs configured and the UDC mapping for Last Segment Input: Temperature (Ordinal) is LPINTMP then the three UDCs will be LPINTMP1, LPINTMP2, and LPINTMP3.

Optional segment points

The Optional list contains the UDC mappings for the optional points. If UDCs are configured for these items, then the application will calculate and output these statistics. Any items in this list that do not have UDCs defined will not be calculated or set.

Note: Many points/UDCs are depending on other UDC. These are denotes by the asterisk. Hover over the Point Description to see the dependency.

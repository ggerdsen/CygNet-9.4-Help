Deleting Points

You can use CygNet Explorer to delete a single point at a time from the Point service. Alternatively you can use the Point Configuration Manager utility to delete single or multiple points. See Deleting Points in the Point Configuration Manager Utility section for more information about this option.

When you delete a point from the PNT, it is deleted from its current value service, but not from its Value History Service (VHS). The point(s) must be deleted manually from the VHS.

If you delete and re-add a point that you were reporting to the VHS, the point’s previous history records will be orphaned from the subsequent history records unless the Point ID and Long Point ID are the same as they were for the deleted point. If the system generates the new point’s IDs, the Point ID will be different from the original ID since it is auto-numbered by the system. When the new point is written to the VHS, the VHS will find it has two points with the same Long Point ID, but that each point has a different Point ID. In such cases, it will rename the old point’s Long Point ID to its Point ID.

To Delete a Single Point Using CygNet Explorer

Use CygNet Explorer to delete a single point at a time from the Point service (PNT).

  1. From CygNet Explorer, access the points list as follows.
  1. Open the application and select the Point Service (PNT) in the service hierarchy (left pane) to view its contents in the service pane (right pane).
  2. In the PNT service pane (right pane), find and right-click on the point you want to delete to access the context menu and then select Delete to access the Delete Point <point> dialog box (PNT Editor) for the selected point.

  1. In the Delete Point <point> dialog box (PNT Editor), click Delete to remove the selected point.

    Note: See PNT Editor for more information about pages and properties contained in the point editor.

  2. In the message box that appears, click Yes to confirm the deletion and return to the service pane.
  3. Repeat the process for any additional points you want to delete.

Using Points

The Point Service (PNT) contains point configuration records, each of which consists of all configuration information about a specific point, including the name of its current value service (CVS), Facility ID, Uniform Data Code (UDC), point and long point IDs, description(s), engineering units, history retention period, alarm setpoints, scaling factors, notification options, enumeration conversions, etc.

See PNT Editor for more information about creating or editing point configurations.

See Alarms for detailed information about points, point states, alarms and alarm colors.

PNT Editor

Use the PNT Editor (Point Properties dialog box) to create, view or edit configuration properties for a single point. The properties in the Properties for Point <point> dialog box that appears when editing an existing point are the same as those in the New Point dialog box that appears when configuring a new point.

Note: Some point property pages appear only for specific point types.

There are multiple ways to access the PNT Editor (the Properties for Point <point> or the New Point dialog box), either from CygNet Explorer (PNT Service) or via the Point Configuration Manager utility.

To Access the PNT Editor

  • In CygNet Explorer, click the PNT service in the service hierarchy (left pane) to view its contents in the service pane (right pane), and then do one of the following.
    • Right-click in the white space of the service pane to access the context menu and then click New to access an unpopulated New Point dialog box. See Adding Points for more information.
    • Click to select an existing point record in the PNT service pane and then right-click to access the context menu, and click New to access a New Point dialog box initially populated with the General properties of the selected point. See Adding Points for more information.
    • Double-click on an existing point record, or right-click to select an existing point to access the context menu and then click Properties, to access the Properties for Point <point> dialog box. By default the PNT Editor opens in view (or read-only) mode; click Edit to access edit mode and modify properties. See Editing Points for more information.
  • In the Point Configuration Manager utility, select a service and filter the points to include the points to be edited in the results, and then do one of the following.
    • Click Add to access the New Point dialog box. See Adding Points for more information.
    • Click to select a single existing point and then click Copy to access the New Point <point> dialog box initially populated with the General properties of the selected point. See Adding Points for more information.
    • Click to select a single existing point and then click Edit to access the Properties for Point <point> dialog box. This opens the PNT Editor in edit mode for the selected point. See Editing Points for more information.

New Point Properties

  • The properties in the New Point dialog box that appears when configuring a new point are the same as those in the Properties for Point <point> dialog box that appears when editing an existing point. See Adding Points for more information about creating new points.

Existing Point Properties

  • The properties in the Properties for Point <point> dialog box (PNT Editor) that appears when editing an existing point are the same as those in the New Point dialog box that appears when configuring a new point. See Editing Points for more information about editing existing point properties.
  • The Properties for Point <point> dialog box (PNT Editor) opens an existing point record to view or edit its properties. By default the PNT Editor opens in view (or read-only) mode.
  • The Previous and Next buttons are accessible in view mode; in edit mode they are disabled. Click Previous or Next to scroll through sequential configured records in the PNT.
  • Click Audit History to view an audit history trail for the selected point record. See Auditing for more information about how auditing works in CygNet Software.
  • Attempting to Close or Cancel the PNT Editor when uncommitted changes have been made to a point record causes a prompt message to appear, providing you the opportunity to either Save, Discard, or Cancel the edit action and return to the editor.

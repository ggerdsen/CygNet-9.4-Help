Point Scaling Properties

Use the Scaling page to configure scaling properties for a particular Analog point. The Scaling page is shown when the point type is set to Analog Input on the General page. (You cannot scale an output value.)

Values specified for the Raw Low/High and Eng Units Low/High properties are converted into values for the polynomial curve coefficients. As such, if you are importing point information, you must specify the coefficient values. See Scaling Analog Input Point Values for more information.

CygNetStandardMetadata

The CygNetStandardMetadada element is the only element required when running under the default CygNet Standard Point Scheme (Point scheme number 0). The CygNetStandardMetadada element can be placed in an enhanced point scheme to include the standard metadata in the scheme. If used in an enhanced point scheme, ensure there are no conflicts in the XML file between the enhancements and the standard point scheme.

CygNetStandardMetadata element in Enhanced Point Scheme

The following is an example of the CygNetStandardMetadata element placed within an enhanced CvsMetadata.xml file.

<CvsMetadataDefinition formatversion="1.1" modifiedtime="6/01/07 12:00:00" modifiedby="CygNet Engineering">

<CygNetStandardMetadata/>

</CvsMetadataDefinition>

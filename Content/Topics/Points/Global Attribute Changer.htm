Global Attribute Changer Utility

The CygNet Global Attribute Changer (GlobalChanger.exe) utility provides a method to change facility and UDC attributes globally.

The CygNet Global Attribute Changer utility is stored in the CygNet\Utilities directory on your host server.

Note: Changing Facility IDs and UDCs can have unknown effects upon the system. Since backups are overwritten each day, prior to using the Global Attribute Changer utility, it is recommended that you secure and save your backup in a separate location. Upon completion of the changes made, stop and restart all services.

Using CygNet Global Attribute Changer to Rename FMS Facilities

If you use the Global Attribute Changer utility to edit the facility IDs associated with meters or other items that were set up in FMS, you must update the affected IDs manually in FMS.

Note: FMS associates the meters it has configured with the meters in your DDS by data group (FmsConfig or GmrConfig) facility ID. If you want to change those in CygNet, you must also update them in FMS.

Batch Mode Option

If Batch Mode (Read from CSV file) is chosen, the CSV file selected should have two columns with headers. The first column should contain the original Facility Tag or UDC and the second column should contain the new Facility ID or UDC. The same set of services will be used for the entire batch of changes.

Adding Points

You can use CygNet Explorer to add new points to your system. Alternatively you can use the Point Configuration Manager utility to add points. See Adding Points in the Point Configuration Manager Utility section for more information.

To Add a Point Using CygNet Explorer

  1. From CygNet Explorer, access the PNT Editor (New Point dialog box) as follows.
  1. Open the application and select the Point Service (PNT) in the service hierarchy (left pane).
  2. In the PNT service pane (right pane) right-click in the white space to access the context menu and then select New to access the New Point dialog box.
  1. Define the new point properties on the General page.
  1. In the Site.Service box, type the Site.Service name or click … to select the appropriate service.
  2. In the Facility ID box, type the Facility ID name or click … to select the appropriate facility.
  3. In the Uniform data code box, type the UDC name or click … to select the appropriate UDC. You can only select a UDC if the selected facility has UDCs mapped to it in the DDS, or if you are selecting a system UDC.

    Note: Once you provide valid values for Site.Service, Facility ID, and Uniform data code, the system will automatically provide values for the Point ID, Long point ID, Short description, and Long description fields if you do not enter values for those fields. If you prefer to manually enter text into the Long point ID, Point ID, Short description, and Long description fields, those values will prevail and are not overwritten by the system.

  4. Select the appropriate Point scheme. See Point Schemes for more information.
  5. Select the appropriate Point type. See Point Types for more information.
  6. Select the appropriate Primary units and Alternate units. See PNT Engineering Units for more information.
  7. Define additional properties, as applicable to the point type and your requirements.
  1. Define the new point properties on each applicable additional page. See PNT Editor for more information about the properties contained in each page.
  2. Click OK when all properties are complete.
  3. Click Refresh for new points matching the filter to appear in the utility's listed point results.

Tables

The Table Reference Service (TRS) stores selections for table-driven fields and Uniform Data Codes (UDC). Some table fields are system-defined. Others are user-defined.

System tables are those tables for which the entries are system defined. They are generally associated with configuration properties, such as the Point Type property in the Point Service.

User-defined are those tables for which the CygNet system administrator defines the entries, such as a table of facility attributes, enumeration conversions, or the application specific fields on the Application page in the PNT Editor. Some system tables can also store user-defined entries. For example, the CMT1 table is used by the PNT for OPCIS point entries, yet can also be used to store user-defined attributes to aid in user entry consistency.

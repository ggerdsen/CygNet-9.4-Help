System UDCs

System UDCs enable the monitoring of system-level facility, communication, and device performance, statistics, and configuration data.

Some system UDCs are auto-generated (for example, SYSFCCAT), while many others can be added manually using the Point Service (PNT). However, system UDCs cannot be modified. To add a system UDC, see Adding Points to the PNT.

To monitor your system at a broader level (for example, host, network, and machine), use Service Monitor (SVCMON) info items, which make use of special SVCMON UDCs.

The following system UDC topics are for more specialized CygNet applications:

  • Energy Load Forecasting System UDCs
  • Enterprise Operations System UDCs
  • Line Pack System UDCs
  • Sonic Queue System UDCs
  • Text Import System UDCs

For general information about UDCs, see Uniform Data Codes.

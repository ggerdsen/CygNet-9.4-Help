Setting Up 20-Minute Statistics

The 20-minute statistics system UDC (SYCSST2M) is a special rollup of one-minute values for the 20 minutes prior to its execution. SYCSST2M (Activate 20 Min Stats) helps control the strain that constant recalculation would ordinarily put on a large system. 20-minute statistics are only applicable to communication and remote devices.

20-minute statistics are not represented as current period-last period pairs; there is no previous statistic to compare against the current one. For each minute that passes, the rollup is recalculated with the oldest value dropped off and the most recent value added on. There is no dynamic processing of the 20-minute statistic in between the one-minute processing intervals even if the statistic's point processing is forced by the device.

Note: Activating a SYCSST2M for your UIS and selected devices does not enable mapping for the SYCSST2M UDC. To enable mapping, you must add specific points using the appropriate communication statistics system UDC(s) along with the required time period suffix.

To Set Up 20-Minute Statistics (SYCSST2M)

The 20-minute statistics are not enabled by default due to the strain that constant recalculation can put on a large system. They can be enabled for a UIS, or just for select devices in a UIS. Optionally, you can create points for the 20-minute statistics. The suffix for the 20-minute statistic is 2M.

  1. In CygNet Explorer, open your Point Service (PNT) in the service hierarchy.
  2. Right-click in the white space of the PNT service pane and click New.
  3. In the Site.Service box, select a UIS using the Browse button to open a drop-down menu.
  4. Type a Facility ID in the following format: SITE_SERVICE. For instance, MYSITE_UIS. The service you choose here must be a UIS.
  5. In the Uniform Data Code box, either type or select SYCSST2M.
  6. In the Point Type box, use the Browse button to select String Input from the drop-down menu.
  7. You have two options at this point for activating 20-minute statistics for devices:
    • If you want to enable 20-minute statistics for all devices, type ALL in the Initial Value box, then click OK.
    • If you want to enable 20-minute statistics for particular devices, type SELECT in the Initial Value box, then click OK. Next, create new SYCSST2M points for each required device. Type ON in the particular device's Initial Value box.
  8. Restart your UIS, open an affected device in the UIS, and confirm that the Last 20 Min. column appears in its Device Information dialog box.

To Disable All 20-Minute Statistics

  1. In CygNet Explorer, open your Point Service (PNT) in the service hierarchy.
  2. Open the UIS-level SYCSST2M point.
  3. Click Edit, change the Initial Value to NONE, then click Save.

To Disable Specific 20-Minute Statistics

  1. In CygNet Explorer, open your Point Service (PNT) in the service hierarchy.
  2. Open the device-specific SYCSST2M point.
  3. Click Edit, change the Initial Value to OFF, then click Save.

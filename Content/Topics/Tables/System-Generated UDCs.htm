System-Generated UDCs

Some archive-type data groups can be assigned a Base UDC.

Base UDCs

Data groups with elements that represent related aspects of the same thing can use a Point Value Base UDC to represent the data group as a whole. When a Point Value Base UDC is assigned to a data group, the system generates other child UDCs based on that Point Value Base UDC.

Example

The device has a data group that tracks daily history for static pressure (Daily Archive Data). The data group represents static pressure as a whole, but elements of the data group represent hourly values, daily average, daily maximum, and daily minimum. As such, if you assign PSTATIC as the Point Base Value UDC, and the system auto-generates other UDCs based on that UDC, for example, PSTATICH, PSTATICD, PSTATICX, and PSTATICN.

To Assign a UDC as the Base UDC

  1. Configure the desired data group (for example, Daily Archive Data) on the Data Group page for a remote device. See Configuring a Data Group for more information.
  2. View data for the data group.
  3. Browse for the base UDC to assign as the Point Value Base UDC. Click Assign UDC.

  1. Click Close.
  2. Click Map UDCs to see that the base UDC has been prefixed to other related data group elements, as shown below.

Note: This data group does not have an element with PSTATIC assigned to it. That is because none of the elements are for instantaneous static pressure.

Note: System-generated UDCs are not added to the TRS and do not have a description. You can manually add these UDCs to the TRS.

Administrative Services

An Administrative service is a generic name for a service used for system administration.

The Administrative services in CygNet Software are listed in the following table.

Service Description Function

ACS

Access Control Service

The ACS handles CygNet security, that is, permissions and authorization levels for individuals, groups, and applications.

Note: The ACS is also a DBS-based service.

APPS

BLOB Storage Service

The APPS is an instance of a BSS that stores CygNet client application files.

Note: The APPS is also a DBS-based service.

ARS

Address Resolution Service

The ARS is a CygNet directory service that resolves network addresses.

AUD

Audit Service

The AUD is a CygNet service that stores audit records of all service events.

Note: The AUD is also a DBS-based service.

BSS

BLOB Storage Service

The BSS is a CygNet service that stores files.

Note: The BSS is also a DBS-based service.

RSM

Remote Service Manager

The RSM is a CygNet service used to control all other CygNet services. It is used to start and stop individual services, and handles CygNet Redundancy.

Database Services

A Database service is a generic name for a service that stores information used by other services. See DBS-Based Services for more information.

All database services are DBS-based services. The Database services in CygNet Software are listed in the following table.

Service Description Function

DDS

Device Definition Service

The DDS database stores remote device, communication device, and text import device records.

DBS

(Generic) Database Service

The DBS is a generic customizable database that can be used for any purpose.

FAC

Facility Service

The FAC database stores all facility information.

GRP

Group Service

The GRP database stores component and navigation hierarchies based on facility and device attributes.

MSS

Master Scheduling Service

The MSS database stores scheudling information for UIS command tasks, FMS command tasks, set info keyword tasks, and setpoint tasks.

PNT

Point Service

The PNT is the the CygNet point database.

TRS

Table Reference Service

The TRS database stores table-driven fields, list box entries, and Uniform Data Codes (UDC).

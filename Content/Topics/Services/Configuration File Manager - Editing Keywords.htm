Editing Keywords

Edit keyword values in the Config File Manager as required. Text strings are case-sensitive. You can manually change any cell value, or use one of the mass change options described below. Anytime a cell is edited (either manually or via the mass change options), the cell background color changes to green and the text is italicized. Final changes will not be committed to the configuration files, until all edits are reviewed and saved.

For more information, see the following subsections below:

  • Import and Export Keywords
  • Find and Replace Keywords
  • Undo Changes
  • Context Menu
  • Editing Special Action Keywords

Import and Export Keywords

Keywords and values can be imported from or exported to a comma-separated value (CSV) file. You might want to create a standard configuration file definition that can be imported whenever you create a new CygNet site. Alternately, you can modify keyword values using Microsoft Excel and then import.

To Import from a CSV File

  1. Click Import/Export to view the Import and Export dialog box.
  2. Select Import to import the contents of a CSV file to the keyword grid.
  3. Specify the file path from where to read the import file or click … to browse to a location.
  4. Click OK. Any changed values will be displayed in green to indicate they have been modified.
  5. Once you have made imported and reviewed the grid, click Finish to review and save changes.

Note: The following notes and conditions apply when importing from a CSV file:

  • Wildcards in column headers are allowed. For example, I have a column in my CSV titled "*.ACS". Those settings will be used for any service name that matches.
  • If there is a column in the CSV that doesn't match any column in the manager, that column is ignored.
  • If multiple columns in the CSV match a single column in the manager, the import is aborted, and an error messages is displayed.
  • If there are keywords in the CSV that aren't in the manager, those keywords are ignored and not imported.
  • A CSV import file may contain fewer keywords than are in the manager.
  • Disabled keywords must be handled with a "#" preceding the value in the CSV file.
  • N/A keywords must be listed as blank in the CSV file.
  • If the import file contains a keyword value that should be N/A, the value is ignored.

To Export to a CSV File

  1. Click Import/Export to view the Import and Export dialog box.
  2. Select Export to export the contents of the keyword grid.
  3. Specify the file path where to write the export file and a file name or click … to browse to a location.
  4. Click Save. Only the keyword and values for all configuration files will be exported.

Note: The Summary and Special Action columns do not export.

Find and Replace Keywords

You can find all keyword values that match a search string, replace all with a new value, and preview a grid showing Keyword, Service, Old Value, and New Value.

  1. Click Replace (or Ctrl+F) to view the Find and Replace with dialog box.
  2. Type a value in the Find and Replace with text boxes and click Preview.
  3. All keywords that match the criteria will be displayed in the grid.
  4. Clear any check box to not replace the value for that keyword and service.
  5. You can sort any column by clicking the column heading.
  6. Click OK to approve the change.
  7. Once you have made all desired changes, click Finish to review and save changes.

Undo Changes

  • Click Undo to undo all changes made to keyword values. The button is only active if there are actions to undo. You can undo separate changes, including mass change operations.

Context Menu

The right-click context menu contains the following options for editing individual keyword values:

Command Description Shortcut

Cut

Removes a keyword value and puts it on the clipboard.

Ctrl+X

Copy

Copy a keyword value to the clipboard.

Ctrl+C

Paste

Paste a keyword value from the clipboard.

Ctrl+V

Set row to value

Select to set every cell in the row to the value in the selected cell, including the Enable/<Disabled> state.

Ctrl+R

Enable

Select to Enable a <Disabled> keyword. You can highlight multiple cells and enable all at once.

Ctrl+E

Disable

Select to Disable an Enabled keyword. You can highlight multiple cells and disable all at once.

Ctrl+D

Encrypt

Select to encrypt an unencrypted keyword. See Encrypted Keywords for more information about encrypted keywords.

  

Decrypt

Select to decrypt a previously encrypted keyword.

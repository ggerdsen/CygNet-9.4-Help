File Management Using BSS Explorer

  • To Update Rsp and Rsq File Timestamps

  • To Move a File to a New Folder
  • To Check Out Files for Editing

  • To Configure Client Settings for BSS Explorer

Note: You can use Ctrl+Click and Shift+Click to multiselect files whenever necessary.

To View BLOB Service Files

  1. Open the BSS Explorer.
  2. Click Service and then Connect…, or click the connection icon , to access the Select Service dialog box.

  3. In order to connect to a BLOB Storage Service, it must be running. Verify that the desired service is running and then click the service name to select it.
  4. Click OK to display folders from the selected service in the left pane.
  5. Click on a folder name in the left pane to display its contents in the right pane.

To Upload File(s) to the BLOB Service

  1. Follow the procedure to view BLOB Service files.
  2. Click a folder name in the left pane to select it and then click the upload icon to access the Open dialog box, or right-click the folder name to access the context menu, and then select Upload Files… to access the Select Blob Folder dialog box.

  3. Select the desired folder from the drop-down menu, or type a new folder name if necessary.
  4. Click OK to access the Upload Files to Blob Folder dialog box.

  5. Browse to the file(s) desired and select it (them).
  6. Click Upload to upload the selected files to the previously selected Blob folder.

    7. Note: You can also drag one or more files from the desktop or other file system location into a folder.

To Download File(s) from the BLOB Service

  1. Follow the procedure to view BLOB Service files.
  2. Right-click a file name in the right pane to access the context menu, and select Download….
  3. Browse to the directory where you want to put the file and select it.
  4. If necessary, click Make New Folder to make a new folder in which to place the file.
  5. Click OK.

    6. Note: You can also drag one or more files from a folder to the desktop or other file system location.

To Delete Files from the BLOB Service

  1. Follow the procedure to view BLOB Service files.
  2. Select the file(s) you want to delete and click the delete button , or right-click and select Delete.
  3. Click Yes to confirm the deletion as shown below. There is NO "undo" for this action.

    4. Note: To delete a folder, select all of the files it contains and delete them.

To Rename a File

  1. Select the file you want to rename and right-click and select Rename.
  2. Type the new file name.

    3. Note: You can only rename one file at a time.

To Update Rsp and Rsq File Timestamps

  1. To update all .rsp and .rsq file times in the currently viewed service, click Update File Times.
  2. Click Yes to confirm the update. This will update the Modified Date timestamp for all .rsp and .rsq files to the current date/time on your computer.

    3. Note: See Timestamps in the Data section for more information about timestamps.

To Move a File from One Folder to Another

  1. Right-click on the file you want to move and select Properties.
  2. In the Properties for Entry <file name> dialog box select the Folder in the BLOB File Settings field where you want to move the file.

  1. Click OK.

    2. Notes:

    To copy a file from one folder to another, drag the file(s) from folder X to folder Y. A copy of the file will be made in folder Y.

    Click Refresh to see the copied file(s).

To Move a File to a New Folder

  1. Right-click on the file you want to move and select Properties.
  2. In the Properties for Entry <file name> dialog box type the new folder name in the Folder list box.
  3. The new folder will be created when you click OK.

    4. Note: Click Refresh to see the new folder name in the Folders pane.

To Check Out Files for Editing

  1. Right-click the desired folder in the left pane and select Enable Librarian to allow file check in and check out. See Librarian Mode.
  2. Right-click the file you want to edit and select Check Out. Your User ID and the check out time are displayed in columns in the right pane.
  3. Edit and save the file as necessary.
  4. When ready, check in the file by selecting Check In from the right-click menu.
  5. To turn off librarian mode, select the folder again and right-click Enable Librarian.

    6. Note: CygNet Administrators can undo another user's check out by selecting Undo Check Out from the right-click menu.

To Change Security Settings for a Folder and File(s)

  1. Right-click the desired folder or file(s) and select Set File Level Security.
  2. Select the desired Application and Event in the Choose Security Setting dialog box as shown below.

  1. Click OK.

    2. Note: See BLOB Storage Service BSS and APPS for more information about security on the BSS.

To Configure Client Settings for BSS Explorer

  1. Select Help > About CygNet BSS Explorer.
  2. Click Client Settings on the About CygNet BSS Explorer dialog box to view the Client Settings dialog box.
  3. From this dialog box you can view network traffic and configure Application Log Settings and/or Network (DCL) Log Settings.
  4. Once the desired log settings are selected you can choose one or both of the following options:
  1. Click Use New Settings to apply your selections.
  2. Click Restore to Default to return to the default settings for the utility.
  1. Click Close.

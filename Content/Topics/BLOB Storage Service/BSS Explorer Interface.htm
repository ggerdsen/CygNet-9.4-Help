BSS Explorer User Interface

The BSS Explorer user interface includes a Menu Bar, a Toolbar, Context Menus, and a Status Bar.

BSS Explorer Interface showing right-click menu in right pane

Menu Bar and Toolbar

Menu options are contained in the following menus: Service, Edit, View, and Help.

Service Menu

The Service menu contains options for connecting to a service and exiting the BSS Explorer.

Menu Option Toolbar Icon Description Shortcut

Connect

Opens the Select Service dialog box to select the desired BLOB Storage Service. Only those services on the domain on which the BSS Explorer is running will be displayed. The services must be running to appear in the list of available services.

Alt S+C

Exit

Exits BSS Explorer.

Alt S+X

Edit Menu

The Edit menu contains options for selecting, copying, and pasting files.

Menu Option Toolbar Icon Description Shortcut

Undo

  

Undoes the previous action.

Ctrl+Z

Copy

Copies selected file(s) to the clipboard.

Ctrl+C

Paste

Pastes file(s) from the clipboard to the selected folder or to a selected file system location.

Ctrl+V

Select All

  

Selects all files in the pane.

Ctrl+A

View Menu

The View menu contains options for customizing the appearance of the BSS Explorer.

Menu Option Description Shortcut

Toolbar

Hide or Show the toolbar.

Alt V+T

Status Bar

Hide or Show the status bar. The status bar displays the following information:

  • Number of files in the currently selected folder
  • File Type of the currently selected file
  • Modified Date of the currently selected file
  • Size of the currently selected file
  • Username
  • Service being explored
  • Domain

Alt V+S

Help Menu

The Help menu opens Help and About information.

Menu Option Toolbar Icon Description Shortcut

Help

  

Displays the Online Help for CygNet Software.

F1

About CygNet BSS Explorer

Displays an About CygNet BSS Explorer dialog box to view application information.

Available information includes the following:

  • The application name, version, and release date
  • Copyright information
  • The CygNet Software website address, including a hyperlink to the website
  • The log file location, including a hyperlink to the log file
  • The location of the application's file executable

In addition to information about the application, the About box includes the following buttons:

  • Client Settings — opens the Client Settings dialog box to view and modify client log settings for the application on an ad hoc basis. See Client Log Settings in the Logging section for more information.
  • Diagnostics — access the following options for additional analysis from the drop-down menu:
  • Loaded Modules — use this option to view and search file information for modules loaded in your system. Double-click any result row to view properties for that module. See Loaded Modules for more information.
  • Gather Logs — accesses the Windows file directory where the open log files are located. Double-click to select and view an application log file.
  • User Local Crash Dumps — Local crash dumps are not enabled. See Generating Crash Dumps for more information.
  • Close — click to close the About box.

Alt H+A

Context Menus

The BSS Explorer provides right-click menus in each pane.

Folders Pane (Left)

Right-click anywhere in the left pane to access the following options.

Right-Click Menu Option Description Toolbar Icon Shortcut

Upload Files

Uploads file(s) from a file system location to the selected folder.

 

Enable Librarian

Enables Librarian mode for a selected folder. When Librarian is enabled for a folder, CygNet Studio will lock a file when it is opened for editing. This prevents other users from editing the same file while it is locked. See Librarian Mode.

Note: A BSS Security Settings.~ file is added to the folder, and a Check Out User and a Check Out Time column are added to the right pane.

 

 

Security Settings

Changes folder level security settings for the selected folder.

 

 

 

File Pane (Right)

Right-click any file in the right pane to access the following options.

Note: The right-click options differ depending on the type of file being viewed and the actions taken.

Right-Click Menu Option Description Toolbar Icon Shortcut

Open

Opens or executes the selected file.

 

 

Download

Downloads file(s) from the folder to a selected file system location.

 

Update File Time

Updates the Modified Date timestamp for all .rsq and .rsp files.

Note: This option is only shown for .rsq and .rsp files.

 

CVision Preview

Opens a preview of a selected .csf file in CygNet Vision. Not available for non-csf files.

 

 

Copy

Copies selected file(s) to the clipboard.

Ctrl+C

Paste

Pastes file(s) from the clipboard to the selected folder or to a selected file system location.

Note: If a file is pasted into the same folder from which it was copied, a copy of the file will be made.

Ctrl+V

Delete

Deletes selected file(s) from the folder.

 

Rename

Renames the selected file. If no file extension is given, the previous extension will be used.

 

 

Set File Level Security

Changes file level security settings for the selected file.

 

 

Check Out

Check the selected file for editing.

Note: This option is available only when Librarian has been enabled for the folder.

 

 

Check In

Check in the selected file.

Note: This option is available only when Librarian has been enabled for the folder.

 

 

Undo Check Out

Force the undo of a checked-out file.

Note: This option is available for CygNet Administrators only.

 

 

Properties

Opens the Properties for the selected file. From Properties you can perform the following:

  • Change file settings
  • View file details
  • Set file level security
  • Set folder level security
  • Enable the Librarian for the folder
  • View the Audit History.

Auxiliary Info Page

Use the Auxiliary Info page of the Group Service editor to include auxiliary information with a specific node. Auxiliary information may be useful in a node that is used as a file in a GRP file system, a node used by the Energy Load Forecasting (ELF) application, or for some other user-defined purpose. It can be used in conjunction with GrpClient methods like HasAuxData, ReadAuxData, and WriteAuxData, among others.

Information in the Data field may be plain text or XML. If you choose to view text as XML, the XML is formatted and then validated for well-formedness. A dynamically generated report at the bottom of the page informs you about the validity of the XML.

Auxiliary Info Page Properties

The following table lists and describes the Auxiliary Info page properties of the GRP editor.

Property Description
View as XML Check this box to enable the text in the Data field to be validated as XML. Such text must be marked up according to standard XML guidelines. A dynamically generated report at the bottom of the page informs you about the validity of the XML.
Data This field displays text or XML. To enable editing, click Edit Node.
Time Stamp This field reflects the last time text or XML in the Data field was edited and saved.
Data Length This field dynamically displays the size of the data in the Data field. The number of bytes correlates to the number of characters.

Saving and Printing Cards

Several methods exist for retaining card information, in addition to the stored transactions in the DDS. Cards can be saved as reference cards in the system and device, saved (and exported) as .txt or .dat files, and/or printed.

Saving Cards

Saving cards in the Dynagraph Viewer can mean saving the card as a reference or exporting the card data. Reference cards are saved in the CygNet system, and optionally to the device. CygNet can store an unlimited number of reference cards.

To Save a Card as a Reference

  1. Make sure that the card is selected. On the Card menu (or right-click context menu), click Save As Reference.
  2. In the dialog box, enter a description for the card.
  3. Click Save to save the card to the CygNet system only, or click Send to RTU to save to both the CygNet system and the device.

Cards can be exported as text or data files for reporting. Currently, two formats are available for export. These are a comma-separated text file (Load Position CSV), and a data file (Lufkin Automation DIAG). The formats of each type can be previewed in the Export dialog box.

To Save and Export a Card

  1. With the card selected, do one of the following:

  • In the Card menu, click Export.

  • On the right-click context menu, click Export Card.

  • Click the Export Card button on the toolbar.

  1. In the dialog box, choose a file format. A read-only preview of the format is displayed in the File Format Description dialog box.
  2. Click OK.
  3. In the Save As dialog box, choose a location and file name for the exported file.
  4. Click Save.

Printing Cards

Use the print option to print the current plot alongside the information from the details panel. The printed card plot will appear identical to the card plot viewed on screen; overlaid cards appear overlaid, units are displayed on the axes, and labels for all card sets are printed with the selected card highlighted.

To Print the Current Plot

  1. Do one of the following:

  • Click Print on the toolbar.

  • On the View menu click Print or right-click in the graph and select Print from the context menu.

  • Press Ctrl+P on the keyboard.

  1. In the Print dialog box, select the printer and use the Properties button to specify printer properties (i.e. landscape or portrait mode). Select the number of copies in the Copies box.
  2. Click OK to send the document to the printer.

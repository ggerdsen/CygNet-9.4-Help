Toolbar

All actions available in the toolbar are also available in the main menu, and many are also available in the context menu. The following table lists these actions, with keyboard shortcuts where applicable.

Icon Action Shortcut Description

Select UIS

Ctrl+S

Opens a dialog box for selecting a Universal Interface Service (UIS).

Select Facility

Ctrl+F

Opens a dialog box for selecting a device in the current UIS.

Previous Facility

Down Arrow

Selects the previous device in the current UIS (device order is alphabetical by device ID).

Next Facility

Up Arrow

Selects the next device in the current UIS (device order is alphabetical by device ID).

Get Current Card

N/A

Gets the current dynagraph card from the device.

Get Shutdown Card

N/A

Gets the shutdown card from the device (see Card Types).

Get Pumpup Card

N/A

Gets the pump-up card from the device (see Card Types).

Get Shutdown and Pumpup Card

N/A

Gets a shutdown card and a pump-up card (see Card Types).

Get Start Card

N/A

Gets the start card from the device (see Card Types).

Get Reference Card

N/A

Gets a reference card for the device (see Card Types).

Start/Stop Card Trace

N/A

Starts or stops a card trace.

Select Cards

N/A

Displays all stored card transactions, allowing you to select a set to display.

Clear View

N/A

Clears the current view set.

Print

Ctrl+P

Opens a print dialog box for printing the current card and status information.

Card Details

N/A

Displays card information and load/position pairs in a dialog box.

Export Card

N/A

Displays options for exporting the card to a text or data file.

Previous Card

Left Arrow

Moves to the previous card in the view set.

Next Card

Right Arrow

Moves to the next card in the view set.

About CDynagraph

N/A

Displays information about the CygNet Dynagraph Viewer application.

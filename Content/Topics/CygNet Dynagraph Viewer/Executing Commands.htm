Executing Commands

UIS commands can be executed on the current facility from the Dynagraph Viewer interface. All commands are available for which the current user has security permissions. Although commands can be executed through the Dynagraph Viewer, they cannot be created or configured through this interface.

To Execute a UIS Command

  1. On the Facility menu, click Execute Command…. The Select Command to Execute dialog box appears, showing all available commands.
  2. Click on a command to select it.

    3. Note: Double-clicking a command will send the command immediately.

  1. Specify any command parameters, if necessary, in the Command Parameters field.
  2. Use the Status Point box to provide optional status point information for the command. Leave Long ID checked if the point name is a long ID (otherwise, enter a short ID). Select Monitor Status Point to optionally monitor the status point.
  3. Click Execute to send the command.

CygNet Dynagraph Viewer > Using The Viewer Window > Context Menu

Context Menu

Right-click anywhere in the card plot area to access the application’s context menu. Many commonly used actions are available through this menu.

Menu Option(s) Description

Select Site/Service…

Select Site/Service…

Access the Select Service dialog box to select a UIS Site.Service

Select Facility…

Select Facility…

Opens a dialog box for selecting a facility in the current UIS.

View

Details Panel

Displays the details panel.

Mark Points

Displays the mark points along the plot.

Net Stroke

Displays the net stroke value.

Fluid Fill Setpoint

Displays the fluid fill setpoint, if applicable.

Initial Cards for Device…

Accesses the Initial Cards dialog box to select display of previously retrieved cards, when navigating to a new device.

Overlay Cards

Overlays card views within the current view set.

Clear Cards

Clears the current view set.

Select Card Set…

Displays all stored card transactions, allowing you to select a set to display.

Max Cards to Display…

Accesses the Enter Value dialog box to select the maximum number of cards to display. Valid values are 5 - 50.

Surface Cards Only

Displays a card plot for surface cards only.

Downhole Cards Only

Displays a card plot for downhole cards only.

Surface and Downhole

Displays card plots for both surface and downhole cards.

Print

Print

Opens a print dialog box for printing the current card and status information.

Card Details…

Card Details…

Accesses the Dynagraph Card Details dialog box, displaying card information and position/load pairs.

Save as Reference…

Save as Reference…

Accesses the Save as Reference Card dialog box, allowing you to save the selected card as the most recent reference card in your CygNet system.

Export Card…

Export Card…

Displays options for exporting the card to a text or data file.

Remove Card

Remove Card

Deletes the card.

Get from Device

Status Data

Gets status data in the details panel.

Pumpoff Config

Gets pumpoff configuration data.

Fluid Fill Setpoint

Gets the fluid fill setpoint from the device.

Current Card…

Gets the current dynagraph card from the device.

See Card Types for more information.

Shutdown Card…

Gets the shutdown card from the device.

See Card Types for more information.

Pumpup Card…

Gets the pump-up card from the device.

See Card Types for more information.

Reference Card

Gets a reference card for the device.

See Card Types for more information.

Start Card

Gets the start card from the device.

See Card Types for more information.

Alarm Card

Gets the alarm card from the device.

See Card Types for more information.

Failure Card

Gets the failure card from the device.

See Card Types for more information.

Edit Data

Pumpoff Config

Accesses the Pump-Off Configuration dialog box to edit values and send the edits back to the RTU.

Fluid Fill Setpoint

Accesses the Fluid Fill Setpoint (%) dialog box to edit load and position values and send the edits back to the RTU.

Refresh

Status Data

Refreshes the display of status data in the details panel.

Pumpoff Config

Refreshes the pumpoff configuration display.

Fluid Fill Setpoint

Refreshes the fluid fill setpoint display.

Auto Refresh Status…

Accesses the Status Point Automatic Refresh dialog box to disable/enable and set a refresh rate (in seconds) for status points

Send to Device

Pumpoff Config

Sends the pumpoff configuration to the device.

Fluid Fill Setpoint

Sends the fluid fill setpoint to the device.

Card Trace

Start Trace

Starts a Card Trace

Setup

Sets up a Card Trace

Using the Viewer Window

The following image shows a CygNet Dynagraph Viewer window as it looks with a card displayed. Scroll over areas of the image to get details.

Sample CygNet Dynagraph Viewer Window

The standard Windows main menu includes all toolbar commands. The status bar provides feedback about actions performed, such as card requests, as well as context-sensitive help for toolbar and menu commands. The details panel displays current information about the selected device, the current card, and setpoint positions. The card plot area shows a load vs. position graph of a dynagraph card.

The Viewer window can be resized, and its constituent windows (i.e., the details panel and card plot area) will resize accordingly. Standard maximize and minimize buttons are available in the title bar for working full screen or hiding the window. The Dynagraph Viewer retains its size and position information when closed.

Customizing the Window

You can toggle visibility of the following elements using the View menu:

  • Toolbar
  • Status Bar
  • Details Panel
  • Mark Points
  • Net Stroke
  • Fluid Fill Setpoint
  • Initial Cards for Device

When one of these elements is visible, a check mark appears next to the option in the View menu. Click the option to toggle the element’s visibility. Visibility settings are preserved across Dynagraph Viewer sessions. See Card Viewing Options for information about toggling visibility of card information.

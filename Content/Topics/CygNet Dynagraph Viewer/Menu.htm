CygNet Dynagraph Viewer > Using The Viewer Window > Menu

Menus

CygNet Dynagraph Viewer contains eight menus: UIS, Device, Poll, Edit, Refresh, View, Card, and Help. Click one of the menus to access the following options.

Menu Options (Keyboard Shortcut) Description

UIS

Select Service… (Ctrl+S)

Accesses the Select Service dialog box to select a UIS Site.Service

[Lists recently selected Services]

Lists recently selected Site.Service options. Click to select.

Exit

Exits the Dynagraph Viewer.

Facility

Select Facility (Ctrl+F)

Accesses a dialog box for selecting a facility in the current UIS.

Next Facility (Up Arrow)

Selects the next facility in the current UIS (facility order is alphabetical by facility ID).

Previous Facility (Down Arrow)

Selects the previous facility in the current UIS (facility order is alphabetical by facility ID).

Device Specific Data…

Accesses the Select Data Group dialog box, allowing you to select a data group from the list of all data groups configured for the device.

Execute Command…

Accesses the Select Command to Execute dialog box, allowing you to select a command from the list of all available commands.

Poll

Get Status Data

Gets status data to display in the details panel.

Get Pumpoff Config

Gets pumpoff configuration data from the device.

Get Fluid Fill Setpoint

Gets the fluid fill setpoint from the device.

Get Current Card

Gets the current dynagraph card from the device.

See Card Types.

Get Shutdown Card

Gets the shutdown card from the device.

See Card Types.

Get Pumpup Card

Gets the pump-up card from the device.

See Card Types.

Get Card Pair

Gets a shutdown card and a pump-up card.

See Card Types.

Get Reference Card

Gets a reference card for the device.

See Card Types.

Get Start Card

Gets the start card from the device.

See Card Types.

Get Alarm Card

Gets an alarm card for the device.

See Card Types.

Get Failure Card

Gets a failure card for the device.

See Card Types.

Get Current Cards…

Accesses the Request Current Cards dialog box to define a card set to request.

Get Shutdown Cards…

Accesses the Request Shutdown Cards dialog box to define a card set to request.

Get Pumpup Cards…

Submits a Pumpup Card request.

Get Alarm Cards...

Submits an Alarm Card request.

Get Failure Cards...

Submits a Failure Card request.

Get Custom Card Pair…

Submits a Shutdown Card and a Pumpup Card request.

Send Pumpoff Config

Sends the pumpoff configuration to the device.

Send Fluid Fill Setpoint

Sends the fluid fill setpoint to the device.

Start card trace

Starts a Card Trace.

See Card Requests.

Setup Trace…

Accesses the Card Trace Setup dialog box to set up a card trace.

See Card Requests.
Abort Current Poll

Aborts the current poll.

See Card Requests.

Abort All Polls

Aborts the current poll plus polls in the queue

See Card Requests.

Show Last Poll Error

Displays the last poll error.

Clear Last Poll Error

Clears the last poll error.

Edit

Edit Pumpoff Config

Accesses the Pump-Off Configuration dialog box to edit values and send the edits back to the RTU.

Edit Fluid Fill Setpoint

Accesses the Fluid Fill Setpoint (%) dialog box to edit load and position values and send the edits back to the RTU.

Refresh

Refresh Status (Alt+F5)

Refreshes the status data display.

Refresh Pumpoff Config (Ctrl+F5)

Refreshes the pumpoff config display.

Refresh Fluid Fill Setpoint

Refreshes the fluid fill setpoint display.

Refresh Cards

The RefreshCards method enables you to manually refresh card data and display to see the latest changes. Useful when a dynagraph card depends on another system for calculations, which might create a small lag time in communication between the systems.

Auto Refresh Status…

Accesses the Status Point Automatic Refresh dialog box to disable/enable and set a refresh rate (in seconds) for status points.

Auto Refresh Cards…

Enables you to set up a time interval by which to automatically refresh card data and display to see the latest changes.

Useful when a dynagraph card depends on another system for calculations, which might create a small lag time in communication between the systems.

View

Toolbar

Hide or Show the toolbar.

Status Bar

Hide or Show the status bar.

Details Panel

Select/clear display of the details panel.

Mark Points

Select/clear display of the mark points along the plot.

Net Stroke

Select/clear display of the net stroke value.

Fluid Fill Setpoint

Select/clear display of the fluid fill setpoint, if applicable.

Initial Cards for Device…

Accesses the Initial Cards dialog box to select/clear display of previously retrieved cards when navigating to a new device.

Overlay Cards

Overlays card views within the current view set.

Clear Cards

Clears the current view set.

Select Card Set…

Accesses the Choose Cards to View dialog box, displaying all stored card transactions, allowing you to select a set to display.

Max Cards to Display…

Accesses the Enter Value dialog box to select the maximum number of cards to display. Valid values are 5 - 50.

Surface Cards Only

Displays a card plot for surface cards only.

Downhole Cards Only

Displays a card plot for downhole cards only.

Surface and Downhole

Displays card plots for both surface and downhole cards.

Filter Downhole Cards

Enables you to filter downhole cards based on relevant calculation type(s) used in a single transaction. Options only appear if they are valid for the specified transaction.

Options are as follows:

  • Device
  • Everett-Jennings
  • Gibbs

Check only those options you want to display. Graphs for selected card options overlay.

Click Clear Filters to remove any filtering applied. This results in the display of all available card types.

Print… (Ctrl+P)

Opens a print dialog box to select parameters for printing the current card and status information.

Card

Card Details…

The ShowCardDetails method displays the Dynagraph Card Details dialog box, displaying card information and position/load pairs.

Save as Reference…

The SaveReferenceCard method displays the Save as Reference Card dialog box, allowing you to save the selected card as the most recent reference card in your CygNet system.

Export…

Displays options for exporting the card to a text or data file.

Remove (Del)

Deletes the card.

Next Card (Right Arrow)

Moves to the next card in the view set.

Prev Card (Left Arrow)

Moves to the previous card in the view set.

Help

About CDynagraph…

Displays an About CygNet Dynagraph Viewer Application dialog box to view application information.

Available information includes the following:

  • The application name, version, and release date
  • Copyright information
  • The CygNet Software website address, including a hyperlink to the website
  • The log file location, including a hyperlink to the log file
  • The location of the application's file executable

In addition to information about the application, the About box includes the following buttons:

  • Client Settings — opens the Client Settings dialog box to view and modify client log settings for the application on an ad hoc basis. See Client Log Settings in the Logging section for more information.
  • Diagnostics — access the following options for additional analysis from the drop-down menu:
  • Loaded Modules — use this option to view and search file information for modules loaded in your system. Double-click any result row to view properties for that module. See Loaded Modules for more information.
  • Gather Logs — accesses the Windows file directory where the open log files are located. Double-click to select and view an application log file.
  • User Local Crash Dumps — Local crash dumps are not enabled. See Generating Crash Dumps for more information.
  • Close — click to close the About box.

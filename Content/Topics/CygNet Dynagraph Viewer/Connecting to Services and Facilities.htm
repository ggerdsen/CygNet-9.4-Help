Connecting to Services and Facilities

Once your system is configured properly, you will be able to use the Dynagraph Viewer to display cards from all available Pump-Off Control (POC) facilities in your CygNet system. Only one Universal Interface Service (UIS) and one facility within the selected UIS can be selected at a time. The current, or selected, service name is shown in the application title bar. The current, or selected, facility name is shown at the top of the details panel.

When you open the Dynagraph Viewer, it will automatically select the last facility viewed, if there was a facility selected when the application was last closed.

When a facility is selected in the Dynagraph Viewer for the first time, no card will be displayed in the card plot area until you submit a card request. See Viewing Cards for information about card requests and view sets.

When you switch to view a different facility in the Dynagraph Viewer, the card view set for the current facility will be lost.

Selecting a Service

To Select a Service

Use one of the following options to select a UIS Service.

  • On the toolbar, click Select UIS .
  • On the UIS menu, click Select Service….
  • On the keyboard, press Ctrl+S.

Utilizing any of these options displays the Select Service dialog box. The dialog box lists all running UIS services in the system. Select the desired service name, then click OK.

The first facility in a selected UIS is automatically selected in the Viewer. You can see what facility is selected by looking at the top few lines in the details panel. These lines show the current facility ID, description, and device ID.

You can select a facility by either choosing it from a list or by paging through the facilities using Previous Facility and Next Facility. The available facilities are limited to Pump-Off Control (POC) facilities in the current UIS. The facility list is organized alphabetically by facility ID.

Selecting a Facility

To Select a Facility from the List

Use one of the following options to select a facility from the list.

  • On the toolbar, click Select Facility .
  • On the Facility menu, click Select Facility….
  • On the keyboard, press Ctrl+D.

Utilizing any of these options displays the Choose Pump-Off Control Facility dialog box. The dialog box lists only POC facilities for the selected service. Select the desired facility ID, then click OK.

To Move to the Next Facility in the List

Use one of the following options to move to the next facility.

  • On the toolbar, click Next Facility .
  • On the Facility menu, click Next Facility.
  • On the keyboard, press the UP arrow.

This option is disabled if only one POC facility is configured in the UIS.

To Move to the Previous Facility in the List

Use one of the following options to move to the previous facility.

  • On the toolbar, click Previous Facility .
  • On the Facility menu, click Previous Facility.
  • On the keyboard, press the DOWN arrow.

This option is disabled if only one POC facility is configured in the UIS.

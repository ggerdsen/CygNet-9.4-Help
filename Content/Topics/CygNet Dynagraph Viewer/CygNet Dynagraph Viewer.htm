CygNet Dynagraph Viewer

This section describes the CygNet Dynagraph Viewer user interface. The CygNet Dynagraph Viewer interfaces with your CygNet system to provide a window into rod-pump well performance.

The following EIE types can operate with the Dynagraph Viewer.

  • AutoCom EIE
  • eProd EIE
  • Lufkin MPC/RCP EIE
  • Lufkin SAM EIE
  • Modbus EFM EIE
  • OPC Lufkin EIE
  • OPC Weatherford EIE
  • TriEnerTech EIE

Many of the features described require that the devices be properly configured in the Device Definition Service (DDS) to work with the CygNet Dynagraph Viewer (for example, UDC mapping).

Note: This document is intended as a guide to using the CygNet Dynagraph Viewer application; it is not intended as an introduction to analyzing stroke information. The reader is assumed to have prior Pump-Off Control (POC) operating experience in order to utilize the functionality appropriately.

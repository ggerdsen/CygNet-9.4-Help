Card Colors and Abbreviations

Card Colors

Device-specific well states are mapped to the following categories by the device driver. Colors are fixed for uniform presentation.

Category Color
Alarm Red
Alert Yellow
Pump-up Green
Normal White
Pumpoff Turquoise
Control other than RTU Magenta
Runtime violation Blue
Unable to determine Gray

Card Abbreviations

C

Current Card

S

Shutdown Card

P

Pump-up Card

R

Reference Card

T

Start Card

A "*" next to a label indicates that there are no data points for that card. The numbers next to current cards and shutdown cards relate to the number of these cards requested in one command. If you use the Get Current Card or Get Shutdown * command, a "1" will be appended to the C or S in the card label because only one card was requested. This is used to distinguish the requested card from a set of cards requested using Get Current Cards or Get Shutdown Cards, which allow you to retrieve multiple cards. In these cases, the card labels show the card number above the total number requested, such as "1/5" to indicate the first card in a set of five.

Configuring and Accessing Device Data Groups

Supported devices must be configured in the Device Definition Service (DDS) to work with the Dynagraph Viewer, in order to view screens in the Dynagraph Viewer or to add dynagraph controls to CygNet Studio screens. Once they are configured, device data groups can also be accessed from the Dynagraph Viewer.

Configuring Device Data Groups

At a minimum, the following data groups must be configured.

  • Downhole Configuration - Required for generating downhole cards (CygNet Software only; not to/from device).
  • Pump-off Configuration - Required for generating shutdown cards (or pumpoff cards) when the pump is turned off.
  • Dynagraph Cards - Required to view real-time surface and downhole cards. A CygNet Dynagraph Viewer control can be added to a screen only when this is configured.

To Configure Device Data Groups

  1. Using CygNet Explorer, open the Device Definition Service (DDS), and click Remote Devices to view a list of devices.
  2. Double-click the remote device you want to configure.
  3. Select the Data Groups page to add data groups and map UDCs.
  4. To add more data groups, click Add.
  5. The Select Data Group to Add dialog box appears, showing all data groups configured for the device.
  6. Click a data group name to select it, then click View Data to display the associated Dynagraph Card.

See Card Types for more information about card types generated (all card types may not be available on all device types).

See specific device types in the Remote Devices section for more information about specific data groups available for your device type. Device types supported include AutoCom EIE, eProd EIE, Lufkin MPC/RCP EIE, Lufkin SAM EIE, Modbus EFM EIE, OPC Lufkin EIE, OPC Weatherford EIE, and TriEnerTech EIE.

Accessing Device Data Groups

It is possible to view and edit facility data through data groups, just as you would in CygNet Explorer. This feature can be useful when you want to change a facility parameter or view data group information without first switching to CygNet Explorer, or if you are in an operational environment where CygNet Explorer may not be available. Since this data is pulled from the Device Definition Service (DDS), any changes that you make will be saved back to the DDS as expected.

To Access Device Data Groups from the Dynagraph Viewer

  1. On the Dynagraph Viewer Facility menu, click Device Specific Data. The Select Data Group dialog box appears, showing all data groups configured for the device.
  2. Click a data group name to select it, then click View Data to display the associated Dynagraph Card.

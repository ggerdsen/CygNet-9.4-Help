Configuring CygNet Groups

To manage CygNet Groups in the ACS you must be a Security Supervisor.

To Add a CygNet Group

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the Groups page or right-click in the service window and click View Groups.
  3. Right-click in the white space and click New Group to open the New Group dialog box.
  4. In the Group text box, type an ID for the group. The group name can be up to 64 characters. If possible, use A-Z, 0-9, dash, and underscore.
  5. In the Description text box, type a description for the group.
  6. Add the group members:
  1. In the Group Members area, click New to add a Security ID for the group.
  2. Select the Security ID Type: AD - Active Directory, CG - CygNet Group, or US - User.
  3. In the Group or User text box, type the group or user ID. Wildcards can be used only for a US - User type.
  1. Click OK.

To Add an ID to a CygNet Group

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the Groups page or right-click in the service window and click View Groups.
  3. Right-click on a member of the group and click New Group Member to open the Group Member Properties dialog box.
  4. Select the Security ID Type: AD - Active Directory, CG - CygNet Group, or US - User.
  5. In the Group or User text box, type the group or user ID. Wildcards can be used only for a US - User type.
  6. Click OK or click View Group… to see the Properties for Group dialog box.

To Delete a Member of a CygNet Group

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the Groups page or right-click in the service window and click View Groups.
  3. Right-click on the member to be deleted and click Delete Group Member.
  4. Click Yes to confirm the deletion.

To Delete a CygNet Group

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the Groups page or right-click in the service window and click View Groups.
  3. Right-click on a member of the group and click Delete Group.
  4. Click OK in the Delete Group dialog box.
  5. Click OK to confirm the deletion.

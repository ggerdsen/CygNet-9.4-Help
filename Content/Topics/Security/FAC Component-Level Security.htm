FAC Component-Level Security

Component-level security applied to a FAC record does two things:

  • It determines who can edit the record.
  • It establishes an Application Override for all of the facility’s points in their target services.

You must have at least Level 2 authorization for the service’s ACCESS event to change the properties of a record.

To Configure FAC Component Security

Be sure you understand Application Override before configuring FAC component-level security.

  1. In the ACS, create an Application for the Facility record. Add Events that are the same name as the target services’ Events and assign permissions for them.
  2. In the FAC, right-click on the facility and click Properties.
  3. On the General page, click … next to ACS Appl box and click the Application created in step 1.
  4. Click OK to apply the selection.
  5. Click OK to save the change to the facility.

Configuring Application Administrators

To Add an Application Administrator

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the App Admins page or right-click in the service window and click View App Admins.
  3. Right-click in the App Admins view and select Apps List.
  4. Click on the name of the Application for which you want to add the Application Administrator and click Edit.
  5. Click on the App Admins page.
  1. Click New.
  2. In the Type box, select US - User.
  3. In the User text box, type the full login of the person.
  4. Click OK to return to the Properties for Application dialog box.
  5. Click OK to return to the Applications List dialog box.
  1. Click Close to close the Applications List dialog box.

To Edit an Application Administrator

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the App Admins page or right-click in the service window and click View App Admins.
  3. Right-click on the Application Administrator and click App Admin Properties.
  4. Edit as necessary.
  5. Click OK or click View App… to open the Properties for Application dialog box.
  6. You can also view Audit History… if desired.

To Delete an Application Administrator

  1. Using CygNet Explorer, click on the ACS in the service hierarchy.
  2. Click the App Admins page or right-click in the service window and click View App Admins.
  3. Right-click on the Application Administrator and click Delete App Admin.
  4. Click Yes to confirm the deletion.

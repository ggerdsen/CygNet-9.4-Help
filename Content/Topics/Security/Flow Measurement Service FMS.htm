Flow Measurement Service (FMS) Security

Security for the FMS service (CygNet Measurement) is administered by the Access Control Service (ACS). As with other CygNet services, security is set on an application and event basis. The application name of the FMS service is defined in the service configuration file (CygNet\Services\FMS\fms.cfg) using the security keyword ACS_APPLICATION. The default value is "FMS." See FMS Configuration File Keywords for more information.

See Security for more information about security and CygNet Software.

Database Utilities

CygNet Software provides the following command-line and executable utilities for managing your database services. All CygNet utility files are found in the CygNet\Utilities folder on your host server.

CygNet Utility Name Utility File Name Description Supported Command-Line Only Utility
CygNet Database Utilities

CAPRI To DAT

CapriToDAT.exe

Translates platform-neutral database files (CAPRI) to CygNet database files (DAT). See also DAT To CAPRI.

DAT To CAPRI

DATToCapri.exe

Translates CygNet database files (DAT) to a platform-neutral format (CAPRI) for transportation between operating systems or to send to CygNet for support. See also CAPRI To DAT.

Database Copy

Dbscopy.exe

Copies or moves queues from one database service to another, or deletes queues from a database service.

Database Delete Queue

Dbdelq.exe

Deletes a specific queue and optionally dumps the results to the screen.

Database Dump

Dbdump.exe

Not supported.

Reads a database queue and dumps the records (in hex) to a file.

 

Database Unlock

Dbunlock.exe

Unlocks a specified queue (record).

DBS Consistency Check

DbsConsistencyCheck.exe

Performs consistency checking between the database records and index records. Run this utility to check for any logical errors.

 

DBS Restore Backup

DBSRestoreBackup.exe

Restores a database backup in the event of a media failure, or recovers CygNet data from a previous backup following an event such as an accidental deletion.

DBS TxLog Management

DbsTxLogManagement.exe

Changes the value for the DBS_TRANSACTION_LOG_SIZE keyword in the service configuration file. The DBS_TRANSACTION_LOG_SIZE keyword specifies the size (in megabytes) of each log file in the txlogs directory, the directory where the service transaction log files are stored.

File Check ESE

FileChkESE.exe

Performs internal logical consistency checks for the ESE-based datastore file (.dat.edb). It primarily checks for several different types of errors including, record length inconsistencies, two types of orphaned records, and database record count mismatches.

ReIndex ESE

ReIndexESE.exe

Rebuilds the ESE Index file (inx.edb).

Other Database Utilities

ESENTUTL

Esentutl.exe

Located in the Windows\System32 folder (or Windows\SysWOW64 folder if running a 64-bit operating system).

This is not a CygNet utility.

This Microsoft utility performs database maintenance and recovery operations of ESE-based datastores.

Note: If a CygNet utility has been provided for a particular purpose then use that appropriate utility and do not use ESENTUTL. For example, use DBS Restore Backup instead of the recovery option of ESENTUTL.

Filtering the Service Hierarchy

You can filter the services listed in the service hierarchy. The filter options are site name, service types, and service status. Any combination of these options can be used.

To Filter Services

  1. Click in the service hierarchy.
  2. Click View > Filter > Add Filter or right-click Filter > New or click Add Filter ( ).
  3. Click Enable Filter.
  4. Select the Site Names or Service Types or Service Status you want to view. Use All or None to aid in selection.
  5. Give the filter a name.
  6. Click OK.
  7. To edit an existing filter, select the filter from the drop-down menu, and click Edit Filter ( ) to view the Filter Options dialog box.
  8. To delete an existing filter, select the filter from the drop-down menu, and click Delete Filter ( ).
  9. The saved filters can be selected from the drop-down menu.

Note: Each set of filters is specific to each user and CygNet domain.

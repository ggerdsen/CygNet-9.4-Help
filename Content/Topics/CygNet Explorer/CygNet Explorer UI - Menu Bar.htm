CygNet Explorer User Interface – Menu Bar

CygNet Explorer contains three fixed menu items to access options for performing various CygNet tasks: File, View, and Help.

File Menu

The File menu contains options for managing and printing service records, and exiting the application.

Option Description

New

Add a new record to the selected service. The appropriate dialog box will open for the selected service. For example, the Permissions Properties dialog box for the ACS, the Device Editor for the DDS, the Note Editor for the NOTE, or the New RSM Entry dialog box for the RSM.

The option is dimmed if not applicable, for example, CAS, MSS tasks, UIS.

Delete

Delete the selected record from the selected service. All deletions will require confirmation. Shortcut is DEL key.

Properties

Show or change the properties for the selected item. The dialog box displayed will depend on the item selected:

  • If you have selected a service in the service hierarchy, the SITE.SERVICE Properties dialog box will display.
  • If you have selected a record in the service pane, the appropriate dialog box will open for the selected service.

Print

Opens the Print dialog box to print the active document. Shortcut is Ctrl+P.

Print Preview

Shows a print preview.

Print Setup

Opens the Print Setup dialog box to change print options.

Exit

Closes CygNet Explorer after prompting to save documents.

Execute Now

Select this option to execute an MSS task immediately. For one instance only, Execute Now issues the selected task to its respective service. The task's priority determines when the task will initiate an actual poll. After the task executes, polling returns to the task's scheduled routine.

Note: You must either have administrative access to the service or have "update" level access to the security event EXECNOW to run this task.

Help Menu

The Help menu contains options to link to application help and application version information.

Option Description

Help

Displays an online Help window. Shortcut is F1.

About CygNet Explorer

Displays an About CygNet Explorer Application dialog box to view application information.

Available information includes the following:

  • The application name, version, and release date
  • Copyright information
  • The CygNet Software website address, including a hyperlink to the website
  • The log file location, including a hyperlink to the log file
  • The location of the application's file executable

In addition to information about the application, the About box includes the following buttons:

  • Client Settings — opens the Client Settings dialog box to view and modify client log settings for the application on an ad hoc basis. See Client Log Settings in the Logging section for more information.
  • Diagnostics — access the following options for additional analysis from the drop-down menu:
  • Loaded Modules — use this option to view and search file information for modules loaded in your system. Double-click any result row to view properties for that module. See Loaded Modules for more information.
  • Gather Logs — accesses the Windows file directory where the open log files are located. Double-click to select and view an application log file.
  • User Local Crash Dumps — Local crash dumps are not enabled. See Generating Crash Dumps for more information.
  • Close — click to close the About box.

CygNet Explorer User Interface – Toolbar

The toolbar contains new, delete, properties, refresh, and help functions, as well as filtering options.

Tool

Command

 Description

Menu

New

Add a new record to the selected service. The appropriate dialog box will open for the selected service. For example, the Permissions Properties dialog box for the ACS, the Device Editor for the DDS, the Note Editor for the NOTE, or the New RSM Entry dialog box for the RSM.

The option is dimmed if not applicable, for example, CAS, MSS tasks, UIS.

File > New

Delete

Delete the selected record from the selected service. All deletions will require confirmation. Shortcut is DEL key.

File > Delete

Properties

Show or change the properties for the selected item. The dialog box displayed will depend on the item selected:

  • If you have selected a service in the service hierarchy, the SITE.SERVICE Properties dialog box will display.
  • If you have selected a record in the service pane, the appropriate dialog box will open for the selected service.

File > Properties

Refresh

Refresh the service or record information. Shortcut is F5.

File > Refresh

Add Filter

Click to create a new service filter. Shortcut is Ctrl+Shift+F.

View > Filter > Add Filter

Edit Filter

Click to edit an existing service filter. Shortcut is Ctrl+F.

View > Filter > Edit Filter

Delete Filter

Click to delete an existing service filter. Shortcut is Ctrl+Alt+F.

View > Filter > Delete Filter

Enable Filter

 Click to enable the filter displayed in the filter drop-down menu. Shortcut is F9.

View > Filter > Enable Filter

About

Shows information about the client application. Shortcut is F1.

Help > About CygNet Studio

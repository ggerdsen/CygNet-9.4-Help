AUD, ELSALM, and ELS XML Properties

The SetAudFilterAsXml, SetElsAlmFilterAsXml, and SetElsFilterAsXml methods require an XML filter which allows all types of service-specific properties to be filtered. The following are lists of property names for AUD, ELSALM, and ELS services.

Note: Property names are case-insensitive, and a property number can be used in place of a property name.

  • AUD XML Properties
  • ELSALM XML Properties
  • ELS XML Properties

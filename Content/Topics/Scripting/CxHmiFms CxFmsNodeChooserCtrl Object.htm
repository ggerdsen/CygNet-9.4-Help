CxFmsNodeChooserCtrl Object

The CxFmsNodeChooserCtrl object is used to display and manipulate the CygNet FMS Node Chooser Control.  The CxFmsNodeChooserCtrl object is an ActiveX control that must be placed in an ActiveX control container such as CygNet Studio.

Declare the object using the following code:

CreateObject("CxHmiFms.CxFmsNodeChooserCtrl.1")

The CygNet FMS Node Chooser Control displays a stand-alone chooser used to select an FMS service, display a filtered list of Nodes in the selected service, and select one or multiple Nodes.  The CygNet FMS Node Chooser Control provides the same functionality as the Node Chooser control in FMS Explorer. 

See Using the Node Chooser Control (in FMS Explorer) and CygNet FMS Node Chooser Control (on CygNet Studio screens) for more information about using this control.

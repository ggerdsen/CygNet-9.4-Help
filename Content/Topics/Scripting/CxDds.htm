CxDds

The CxDds is the scripting reference library for objects related to the Device Definition Service (DDS).

The DDS is a DBS-based service used to store device configuration information and to track events, communication modes, and data associated with remote and communication devices. This interface allows access to device information and provides methods for changing data where applicable.

CxDds contains the following objects:

  • DdsClient Object
  • DGEMapping Object

Note: Both 32-bit and 64-bit versions of this library are available; library files are located in your CygNet source files at CygNet\Support\CxDds.dll and CygNet\Support64\CxDds64.dll respectively.

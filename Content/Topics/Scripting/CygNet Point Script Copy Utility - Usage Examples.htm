CygNet Point Script Copy - Usage Examples

The following examples describe several ways to use the CygNet Point Script Copy utility.

  • Edit script offline
  • Copy script from one point to another existing point
  • Copy script from one point to multiple existing points
  • Copy script from one point to a new point
  • Install group rollup code
  • Update group rollup template and points

To Edit Script Offline

  1. Use the utility to copy the scripts to a text file.
  1. Source: From All Points or From Point.
  2. Target: To Script File.
  1. Edit only the script portion of the file.
  2. Use the utility to copy the edited script back to the point(s).
  1. Source: From File.
  2. Target: To Default Service Point in Source.
  1. Test and perform copy.

To Copy Script from One Point to Another

  1. Select the source point to be copied (Source: From Point).
  2. Select the target point (Target: To Point in Service).
  3. Verify that the HyperPoint property of the target point is enabled.
  4. Test and perform copy.

To Copy Script from One Point to Multiple Existing Points

  1. Create a text file that lists the points to which the script is to be copied.
  2. Verify that the HyperPoint property of each target point is enabled.
  3. Select the source point to be copied (Source: From Point).
  4. Set the target to To Points in File.
  5. Test and perform copy.

To Copy Script from One Point to New Points

  1. Create a text file that lists the points to which the script is to be copied.
  2. Select the source point to be copied (Source: From Point).
  3. Set the target to To Points in File.
  4. Test and perform copy.

To Install Group Rollup Code

  1. Obtain the group rollup code from CygNet Software.
  2. Select the source as From File and the Scripts to *All Scripts in this File*.
  3. Set the target to Default Service Point in Source.
  4. Test and perform copy.

To Update Group Rollup Template and Points

  1. Obtain the updated group rollup template code from CygNet Software.
  2. Update the group rollup template.
  1. Select the source as From File and the Scripts to *All Scripts in this File*.
  2. Set the target to Default Service Point in Source.
  3. Test and perform copy.
  1. Update the points that are based on the group rollup template (i.e., group rollup instances).
  1. Select the source as From Point and select the group rollup template point.
  2. Set the target to To Group Rollup Instances.
  3. Test and perform copy.

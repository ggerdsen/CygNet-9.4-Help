CxHmiFms

The CxHmiFms is the standard library for human-interface objects related to CygNet Measurement (FMS) Service controls.

CxHmiFms contains the following objects:

  • CxFmsAlarmLogCtrl
  • CxFmsBalanceCtrl
  • CxFmsClosePeriodCtrl
  • CxFmsConfigCtrl
  • CxFmsDashboardCtrl
  • CxFmsExceptionsCtrl
  • CxFmsHistoryGraphCtrl
  • CxFmsHistoryGridCtrl
  • CxFmsJobsCtrl
  • CxFmsNodeChooserCtrl
  • CxFmsPPACtrl
  • CxFmsRawDeviceDataCtrl
  • CxFmsReportCtrl

See the following topics for more information about the declaration, properties, methods, events, and usage examples for each FMS control object.

Note: The library file (32-bit version) is located in your CygNet source files at CygNet\Support\CxHmiFms.ocx.

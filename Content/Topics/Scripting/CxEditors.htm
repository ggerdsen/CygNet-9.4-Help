CxEditors

The CxEditors is the standard library for service-editing objects. The objects contained in CxEditors are dialog box-based editors for CygNet services and service-specific items.

CxEditors contains the following objects:

  • CxAcsEditor Object
  • CxArsEditor Object
  • CxAudEditor Object
  • CxBssEditor Object
  • CxCasEditor Object
  • CxCvsEditor Object
  • CxDbsEditor Object
  • CxElsEditor Object
  • CxFacEditor Object
  • CxGnsDbEditor Object
  • CxGrpEditor Object
  • CxMssBlackoutEditor Object
  • CxMssCmdEditor Object
  • CxMssFmsCmdEditor Object
  • CxMssInfoKeywordEditor Object
  • CxMssSetPointEditor Object
  • CxNoteEditor Object
  • CxPntEditor Object
  • CxRsmEditor Object (Part of CxHmiRsm)
  • CxTrsEditor Object
  • CxUisEditor Object
  • CxVhsEditor Object

Note: The library file (32-bit version) is located in your CygNet source files at CygNet\Support\CxEditors.dll.

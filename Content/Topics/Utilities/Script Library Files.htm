Script Library Files

CygNet Software provides a set of example script system library files that can be installed using the CygNet Point Script Copy Utility.

Since many system library HyperPoints include methods that accept a HyperPoint object as a parameter, this helps you to run the same code to update multiple points. These points can be incorporated into a working system, or used as a basis for further customization. See System HyperPoints for more information.

The following script library files are available in the Utilities\SCRIPTLIBRARY folder on your host server.

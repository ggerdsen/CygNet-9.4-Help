SCADA Utilities

CygNet Software provides the following utilities for managing your CygNet installation. Click on the file name for more information about each utility. All CygNet utility files are found in the CygNet\Utilities folder on your host server.

Browse by letter:     A     B     C     D     F     G     H     L     M     O     P     R     S     T     U     V

Utility Name Utility File Names Description Command-Line Only Services Supported Checks Security

ARS Diagnostic Tool

ArsDiags.exe

The ARS Diagnostic Tool provides administrative statistics on Address Resolution Services (ARS) for advanced administrators, integrators, and CygNet support. Available diagnostic information includes service validation, iterator test, site consistency, cycle times, and CVS metadata file synchronization.

  

ARS

 

ARS Modify Service

ARSModifyService.exe

The ARS Modify Service command-line utility adds, updates, or deletes a record from an ARS.

ARS

 

BSAP Configurable Data Group Migration

BSAPMigration.exe

The BSAP Configurable Data Group Migration utility converts any "Device Specific Data Group" to the new "Configurable Data Group." UIS commands using the previous device-specific data groups are also updated.

  

EIE

 

BSS Explorer

BssExplorer.exe

The CygNet BSS Explorer client can be used to view, edit, upload and download files from a BSS.

Business Object Builder

CygBob.exe

An Enterprise Objects (EIS) utility used to produce Business Object Package files.

CAPRI To DAT

CapriToDAT.exe

The CAPRI TO DAT command-line utility translates platform-neutral database files (CAPRI) to CygNet database files (DAT). See also DATToCapri.exe below.

DBS

  

Client Monitor

CClientMonitor.exe

The CygNet Client Monitor utility checks for updates for specific CygNet applications on the Application Blob Service (APPS) and notifies users if there are new versions.

  

BLOBs

 

Comm Director

CommDirector.exe

The CygNet Comm Director utility re-directs TCP/IP traffic and directs communications between various ports.

  

DDS

 

 

Config File Manager

ConfigFileMgr.exe

The Config File Manager is a configuration file management tool used to centrally manage all configuration files in your CygNet system (from both local and remote servers) even if you aren’t currently running the application on the primary host server. The tool will allow you to edit (via manual and mass-modification options) and validate your configuration files in one central location.

  

All

 

Connect ARS

ConnectARS.exe

The Connect ARS command-line utility connects one Address Resolution Service (ARS) to another ARS.

CRC Generator

CrcGenerator.exe

The CygNet CRC Generator utility is a protocol analyzer used to troubleshoot field device communication responses.

DAT To CAPRI

DATToCapri.exe

The DAT To CAPRI command-line utility translates CygNet database files (DAT) to a platform-neutral format (CAPRI) for transportation between operating systems or to send to CygNet for support. See also CapriToDAT.exe above.

DBS

  

Database Copy

Dbscopy.exe

The Database Copy command-line utility copies or moves queues from one database service to another, or deletes queues from a database service.

DBS

 

Database Delete Queue

Dbdelq.exe

The Database Delete Queue command-line utility deletes a specific queue and optionally dumps the results to the screen.

DBS

 

Database Dump

Dbdump.exe

The Database Dump utility reads a database queue and dumps the records (in hex) to a file.

DBS

 

 

Database Unlock

Dbunlock.exe

The Database Unlock command-line utility unlocks a specific queue (record).

DBS

 

DBS Consistency Check

DbsConsistencyCheck.exe

The DBS Consistency Check utility performs consistency checking between the database records and index records. Run this utility to check for any logical errors.

  

DBS

  

DBS Restore Backup

DBSRestoreBackup.exe

The DBS Restore Backup command-line utility restores a database backup in the event of a media failure, or recovers CygNet data from a previous backup following an event such as an accidental deletion.

DBS

  

DBS TxLog Management

DbsTxLogManagement.exe

The DBS TxLog Management command-line utility changes the value for the DBS_TRANSACTION_LOG_SIZE keyword in the service configuration file. This keyword specifies the size (in megabytes) of each log file in the txlogs directory, the directory where the service transaction log files are stored.

DBS

  

DDS Check

DdsCheck.exe

The DDS Check utility performs basic validity checking and repairs for common Device Definition Service (DDS) record inconsistencies.

  

DDS

 

DDS Command Copy

DDSCommandCopy.exe

The CygNet DDS Command Copy utility copies UIS command(s) from one remote device to other remote device(s).

  

DDS

DDS Data Group Copy

DDSDataGroupCopy.exe

The CygNet DDS Data Group Copy utility copies data group(s) from one remote device to other remote device(s).

  

DDS

DDS Import Export

DDSLoad.exe

The CygNet DDS Import/Export utility imports and exports device header information (Device ID, Facility ID, and Ordinal) in CSV file format. Options are available to force synchronization of the (DDS) with the Facility Service (FAC) and to update Facility ordinals.

  

DDS

 

DDS Transaction Viewer

DdsViewer.exe

The DDS Transaction Viewer utility provides DDS device and transaction information.

  

DDS

 

 

Device Copy

DeviceCopy.exe

The Device Copy utility can be used as a bulk device copy utility. The source device (that is, the device to copy) must be in the DDS. The source file (information for the new devices) must be a text file. The utility has an option to create a source text file from the existing device(s).

  

DDS

 

Device Template Manager

DtfMgr.exe

The CygNet Device Template Manager verifies XML structure of device template files for Modbus and other templated EIEs. It can be used to upload device template files to the DDS. For overview information about device template files, see Device Template Files.

  

DDS

 

Directory Sync

CDirectorySync.exe

The CygNet Directory Sync utility compares file versions and replaces files based on selected criteria.

This utility does not compare file sets. As such, use the DOS "Comp" command to compare file lists.

File Check ESE

FileChkESE.exe

The File Check ESE command-line utility performs internal logical consistency checks for the ESE-based datastore file (.dat.edb). The utility primarily checks for several different types of errors including record length inconsistencies, two types of orphaned records, and database record count mismatches.

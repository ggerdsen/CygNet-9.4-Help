64-bit Utilities

CygNet Software supports the following 64-bit utilities, which can handle larger data sets associated with 64-bit services.

Utility Name Utility File Name Description Command-Line Parameters Supported Checks Security

Group Manager

GrpMgr64.exe

The Group Manager utility is used to builds hierarchies in the Group service.

  

Point Configuration Manager

PointCfgMgr64.exe

The Point Configuration Manager utility modifies in batch, if desired, fields and deletes points based on filters, such as Facility ID, Site.Service, UDC, etc. The utility also has an import and export point function, to and from a CSV file.

VHS Check

VhsCheck64.exe

The VHS Check utility perform a data integrity check on the VHS database.

    

VHS Import/Export Utility

VhsImportExport64.exe

The VHS Import/Export utility exports data from one VHS to another.

Installation

The 64-bit utilities need to be installed from the CygNet Client Installer.

To Install a Utility

  1. Open the CygNet Client Installer (Start Menu > All Programs > CygNet > CygNet Client Installer). The CygNet Client Installer/Launcher dialog box will be displayed.
  2. Select the APPS BLOB Service.
  3. Select the UTILS folder.
  4. Select Run as administrator and confirm UAC prompting.
  5. Select the utility name. You can also install multiple applications at once. Use the Shift or Control key for the multi-selection.
  6. Optional. If you want to add a shortcut on your desktop, click Add shortcut to desktop. This ensures that you have the version-management settings, which are automatically built into the shortcut properties.
  7. Click Install. The utility will be installed to the CygNet/Utilities directory.

Also see 64-bit Services and 64-bit Device Drivers for more information.

Using CygNet Console

Use the CygNet Console to change from one CygNet Service to another.

Selecting or Changing a CygNet Console Service

When you open CygNet Console, you can easily select a Site.Service view and then quickly switch between service console views as desired.

To Select a CygNet Console Service

  1. When you open CygNet Console, the Select Service dialog appears and you will see a list of sites and services.
  2. To open a service console screen, double-click the name of the site and service you want, or select the site and service name and then click OK, to view the service console for the selected Site.Service. The service console screen for that Site.Service appears.

To Change a CygNet Console Service

To change to (open) another service from the current service console screen:

  1. Click the Service > Open or click to access the Select Service dialog box.
  2. Double-click the name of the site and service you want, or select the site and service name and then click OK, to view the service console for the selected Site.Service. The service console screen for that Site.Service appears.

CygNet Console Service Views

Each service has various views. Use the function keys listed at the bottom of the console screen to switch between views.

Note: When you are viewing a CygNet Console service screen, you are seeing the same view as all other users who may be viewing it; you are all remotely connected to the same screen. This means that if you change views, all other users will also see the change.

CygNet Console Messages

Message Description

DCL error (-9) 'The CygNet service 'SITE.SERVICE' is not in service.'

The CygNet service is stopped or has been shut down.

Service console is not available - MM/DD/YYYY hh:mm:ss

The CygNet service console is unavailable.

Enterprise Integration Suite

The CygNet Enterprise Integration Suite (EIS) facilitates data distribution throughout the enterprise using Enterprise Objects and various industry standard interfaces including XML, OPC, ODBC and COM. These interfaces ensure the timely exchange of data between the CygNet Software platform, third-party applications, and other business systems.

Enterprise Integration Suite Components

The Enterprise Integration Suite includes the following components.

  • Enterprise Objects
  • CygNet COM API
  • ODBC

In addition, the Enterprise Integration Suite can be enriched with the following.

  • OPC Integration Tools

See Enterprise Objects for more information about using Enterprise Objects for data integration.

UIS Overview

The UIS is a current value service focused on messaging, current point values, system statistics, and device statistics. Device configuration takes place in the Device Definition Service (DDS). Point configuration takes place in the Point Service (PNT). And scheduling of polls takes place in the Master Scheduling Service (MSS).

Context Menu Options

The following table lists and describes the context menu options that display when you right-click in the UIS service pane.

Option Description Shortcut

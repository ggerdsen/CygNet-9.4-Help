Failed Comm Transition Period

The failed comm transition period is the amount of time that communications can be in the Exception state. The default transition period is 8 hours (480 minutes). You can change the failed comm transition period using the system UDC SYCOMSTFP. The change can be applied to a communication device, a remote device, and/or an entire UIS. See Failed comm transition period for more information.

To Change the Default Period in the UIS

  1. In CygNet Explorer, open the Point Service (PNT), right-click in the white space of the service pane, and select New.
  2. At the Site.service box, click the Browse button and select a UIS.
  3. Enter the SITE_UIS in the Facility ID box.
  4. Enter the Uniform data code SYCOMSTFP.
  5. Select the Point type of Analog Input.
  6. Set the point's Initial value type to Specified Value.
  7. Set the Initial value equal to the desired transition period (in minutes) and click OK.

To Change the Default Period in a Communication Device

Note: The setting of an individual communication device takes precedence over the UIS setting.

  1. In CygNet Explorer, open the Point Service (PNT), right-click in the white space of the service pane, and select New.
  2. At the Site.service box, click the Browse button and select a UIS.
  3. Enter the communication Device ID in the Facility ID box.
  4. Enter the Uniform data code SYCOMSTFP.
  5. Select the Point type of Analog Input.
  6. Set the point's Initial value type to Specified Value.
  7. Set the Initial value equal to the desired transition period (in minutes) and click OK.

To Change the Default Period in a Remote Device

Note: The setting of an individual remote device takes precedence over the UIS setting.

  1. In CygNet Explorer, open the Point Service (PNT), right-click in the white space of the service pane, and select New.
  2. At the Site.service box, click the Browse button and select a UIS.
  3. Enter the remote Device ID in the Facility ID box.
  4. Enter the Uniform data code SYCOMSTFP.
  5. Select the Point type of Analog Input.
  6. Set the point's Initial value type to Specified Value.
  7. Set the Initial value equal to the desired transition period (in minutes) and click OK.

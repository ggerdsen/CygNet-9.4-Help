CygNet Software Downloads

Obtain downloads for CygNet software products and updates via the CygNet Software Download Site on the Weatherford software support portal at https://softwaresupport.weatherford.com/hc/en-us (login required).

Downloading CygNet Software

The CygNet software download site is available online for authorized users to access software, support, and training resources for a variety of CygNet software products.

Access CygNet software downloads as follows.

To Download CygNet Software

  1. Access the Weatherford.com website.
  2. From the website home page, click the Menu button to access several options, and then click Customer Portal (in the bottom right corner) to access the customer login portal.
  3. Expand the Production option and select Software Support Site to enter the Weatherford software support portal.
  4. On the Weatherford support portal home page, do both of the following to access the CygNet Software page.
    • In the Product Knowledge Base section, select CygNet Software.
    • Click Sign In to enter your email and customer password. Authorization is required to access full content. If you do not yet have sign-in credentials, follow instructions provided on the screen to sign up or get a password
  1. From the CygNet Software page, click available links to access current Patch Notes, Patch Composites for various software versions, documents, announcements, and a variety of useful information and resources.
  2. On the CygNet Software page, click CygNet Software Download Site to access the Software Downloads page. Re-authorization is required via customer login to proceed to the CygNet Download website. Enter credentials as prompted.
  1. When the CygNet Download welcome page appears, click the Software link in the text to access the Software Downloads page.
  2. On the Software Downloads page, click the desired option (e.g. Applied Engineering Products, SCADA and Measurement Products, Windows Update Certs) to view a list of available software downloads.
  1. Read and familiarize yourself with installation documents provided for the product(s) you are choosing.

    Note: Installation instructions are commonly contained in the CygNet Help OR are provided separately in the file download options, depending on the product you are installing and its status in your system. For example, an "Upgrade Procedure" document is provided for information about upgrading an existing CygNet system to a newer version and a "readme" document is provided to address installing patches to a current CygNet product.

  2. Select, download, and install your product files.

Additional Support

From the software support portal pages you can also make support requests. Click on available Support Request or Submit a Request or Submit a Support Ticket links to create a support request ticket.

See Additional CygNet Resources for more information about other CygNet resources available to help you optimize your software usage.

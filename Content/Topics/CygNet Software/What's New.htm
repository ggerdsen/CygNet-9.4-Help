What’s New in the CygNet v9.4 Help

The v9.4 Online Help includes many documentation updates to describe the new features and enhancements contained in the CygNet Software v9.4 release.

Note: See the CygNet v9.4 Release Notes for a more complete list of enhancements and modifications made to CygNet SCADA and CygNet Measurement.

Changes to the v9.4 Online Help

The following documentation items have been added, removed, or revised in the CygNet Softwarev9.4 Online Help. Changes are listed in order of the Online Help document sections in which they appear.

Address Resolution Service — The ARS Diagnostic Tool utility (ArsDiags.exe) topic has been added to the CygNet Help.

Alarms — Enhanced Alarm Configuration topics have been added to the CygNet Help.

Canvas — See the What's New in the v9.4 Canvas Help topic in the Canvas chapter for more information.

Clustering — The "Clustering" and "Failover Clustering on Microsoft Windows Server 2008 R2" topics have been removed from the CygNet Help. For information about using clustering and failover for your CygNet environment, see CygNet Redundancy.

CygNet Broadcast — Documented new GNS keyword REPORT_TO_AUDIT to report all CygNet Broadcast messages to the AUDIT service.

CygNet Dispatch — Refer to CygNet Measurement notes for documentation additions and updates in support of improved CygNet Dispatch functionality, including the following:

  • a new "Accessory" Node type for scheduling non-meter-based jobs
  • a new "Job" category of FMS validation rules
  • a new "Auto-calibration" job type
  • new Jobs control features
  • new Job report options
  • new FMS Event data items to support automatic calibration values
  • FMS Dispatch Data items — added a topic describing FMS Dispatch data item enumerations used by the FMS service to support CygNet Dispatch job data functionality

CygNet Measurement — several new and updated features are documented in CygNet Measurement including the following areas:

  • FMS System/Service
    • Balancing — updated descriptions throughout the chapter to be able to include Liquid Device Nodes with periodic history support in the balancing process
    • Batch data — added support for "flow time" in several locations to expand viewing, reporting, and exporting of batch flow time data
    • Custom Device Data — added "Mass-weighted average" to the aggregation method options available for custom liquid metering process variables when Configuring Custom Device Data in the FMS Explorer Administrative Options
    • Data purging — added the option to specify a "purge end date" when Purging Device Data to retain recent good data if desired
    • FMS Data items — added a topic describing FMS Dispatch data item enumerations, used by the FMS service to support CygNet Dispatch and FMS Explorer, and added several FMS Event data items to support calibration job functionality
    • Nodes — added information about the new Accessory Node type that can be used in conjunction with CygNet Dispatch to schedule and report jobs for non-meter-specific components
    • Nodes — updated Nodes and Group Node Membership Requirements and descriptions to describe allowing Liquid Device Nodes as members of Virtual Station Group Nodes, to be included in balancing and reporting
    • Node properties — updated Creating Liquid Device Nodes to reflect newly added normalization options for Liquid Device Nodes supporting periodic history data, to support publishing data to SCADA via normalization views and to be included in balancing
    • Normalization — updated Administrative Options documentation to describe changes when Configuring Normalization Views to now include Gas and Liquid Device Nodes with support for periodic metering data, and to describe the enhanced functionality in the configuration dialogs used to create new normalization views
    • Validation — documented the new "Job" category of FMS validation rules, and several newly available rule types including Calibration Drift Threshold, Calibration Drift Range, and Number of Calibration Point
  • FMS Commands
    • Command logs - added new columns to view Submitted Date/Time, Executed Date/Time, and Completed Date/Time
    • Export: Device Data CSV - updated Export: Device Data CSV command to include new data item tokens to support including gas analysis liquid content values in the export, and updated sample export definition file information
    • Export: Normalization View Data CSV - updated and renamed the (previous) Export: Station Data CSV command to become the (current) Export: Normalization View CSV command and provided updated sample file information to describe the ability to include data for gas and liquid device Nodes that support periodic history data
  • FMS Controls
    • Balance control - added and updated information to describe the ability to include Liquid Device Nodes with support for periodic metering data in the Balance control
    • Jobs control - added information describing new job status values and added abilities to delete jobs, multi-select jobs for control actions, view automatic calibration values, review calibration events, and address event-triggered exceptions and potential data changes based upon calibration events, directly from the Jobs control grid
  • FMS Reports
    • Changed Device Data report- added parameters to the report template to provide support for showing Mass value changes when they exceed a specified tolerance
    • Job report - added parameters to the report template to allow the use of automatic calibration values ("Auto-calibration" job type) in the report, either in row or column format, if preferred; added the option to use separate post-calibration test standards if desired
    • Normalization View report - updated and renamed the (previous) Station report to become the (current) Normalization View report and provided updated sample template information to describe the ability to include data for gas and liquid device Nodes that support periodic history data

CygNet Studio / Vision — Several topics and subsections have been updated including the following areas:

  • Added a better description and code example to the ImportScriptFile method on TheView.
  • Added a better description and code example to the EventPreToolTip event on all tools and controls.
  • Revised formatting and clarity for Interprocess Communication topic and added a section about Nested Views and shared messages.
  • Added a new topic, Using the The Frame/The View in Run Mode, which includes information on the Point List dialog box.
  • Made some minor edits for special controls in Customizing Point State Colors.

Devices - Remote Devices

  • Added new conversion methods to cvt Conversion Methods.
  • Added new FMS Calibration Events for the Emerson ROCPlus EIE.

Notifications

  • Added a new topic, Getting Started with VoIP, to aid in configuring the Gns.cfg file to use SIP as the VoIP protocol.

Points

  • New command-line parameters (/update and /delete) have been added to the Point Configuration Manager utility.

Security

  • A PNT Configurable Bit Security Events topic has been added.
  • New security events for Enhanced Alarm Configuration Security have been added.

Scripting — the following sections have been updated:

  • Updated CxScript.FileSystemObject.ReadBlobString to include a script example showing a cross domain read.
  • Corrected and updated CxScript.Points.AddPoint to describe the five optional tag string parameters.
  • Add section for CanSendUISCommand to CxUis.UisClient.UisMethods.
  • Added new ScriptJob mode and output log file options for the JobRunner: UseSynchronousOutput, WriteOutputToLogFile, and WriteErrorToLogFile
  • Removed reference to Microsoft Script Debugger, which is now deprecated.
  • Added new methods to Job Runner to kill script processes: KillAllScriptJobs and KillScriptJob.

Service Configuration Keywords

  • Updated the cache size options for the VHS keywords "VHS_MAX_CACHE_SIZE_MB" and "VHS_MIN_CACHE_SIZE_MB" in the VHS Configuration File Keywords topic.

System Administration — Update external link in Generating Crash Dumps and added recommended DumpType value.

Tables — Updated the description for the SYRSPEND UDC in UIS System UDCs.

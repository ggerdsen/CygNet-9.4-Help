CygNet Release Documents

Each version of CygNet includes the following release documents, which contain additional information about the current release.

Access the following documents for assistance:

  • v9.4 Release Notes — a comprehensive list of enhancements, modifications, and fixes contained in the current CygNet Software release version
  • v9.4 Upgrade Procedure — instructions for upgrading existing installations of CygNet Software to the current version
  • v9.4 System Requirements — hardware and operating system requirements for running CygNet Software
  • v9.4 Quick Start Guide — instructions for first-time installations of CygNet Software
  • v9.4 Memory and Disk Calculator — an Excel calculator provided to help estimate memory and disk space requirements for a typical CygNet system
  • 64-bit Installation Guide — instructions for installing 64-bit components on your CygNet Software system

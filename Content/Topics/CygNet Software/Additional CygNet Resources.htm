Additional CygNet Resources

The CygNet Help document explains the breadth and complexity of CygNet products in great detail, however questions or circumstances not covered in the Help document can still arise. The following resources provide additional avenues to help you use and make the most of your CygNet Software product, including source file, online, phone, and email access.

CygNet Release Documents

Each version of CygNet includes the following release documents, which contain additional information about the current release.

Access the following documents for assistance:

  • v9.4 Release Notes — a comprehensive list of enhancements, modifications, and fixes contained in the current CygNet Software release version
  • v9.4 Upgrade Procedure — instructions for upgrading existing installations of CygNet Software to the current version
  • v9.4 System Requirements — hardware and operating system requirements for running CygNet Software
  • v9.4 Quick Start Guide — instructions for first-time installations of CygNet Software
  • v9.4 Memory and Disk Calculator — an Excel calculator provided to help estimate memory and disk space requirements for a typical CygNet system
  • 64-bit Installation Guide — instructions for installing 64-bit components on your CygNet Software system

 

CygNet .NET API Help

The CygNet API Help is a Microsoft Help file (CygNet.API.chm) describing the CygNet .NET APIs.

  • The CygNet API Help is included with the CygNet product, in the /Bin folder.

CygNet Bridge API Help Page

CygNet Bridge API provides methods for secure access to CygNet system data for integration into desktop, mobile, or web applications.

  • The CygNet Bridge API Help is a set of auto-generated HTML pages included with the CygNet Bridge product in the /Docs folder.
  • An offline version of the CygNet Bridge API Help is provided here and will open in a new tab.

CygNet Bridge API Sample Web Application

The CygNet Bridge API comes with a sample web application providing source code that demonstrates using the API to retrieve data from your CygNet system. The sample application source code is located in the CygNet Bridge product in the BridgeAPISampleApp/sample folder. The source code is for an Angular web application and needs to be compiled first in order to run it in a web browser.

Other CygNet Help Files

CygNet Reports

CygNet Reports is an application that uses a Crystal Reports run-time engine that allows easy access and formatting of CygNet data into a variety of dynamic reports. Each report has a template with parameters that defines the format of the report. You define the parameters for the report, such as what types of data to include, date ranges, sorting, etc. Completed reports can then be exported into many different file formats, including comma-separated values (CSV), character-separated values, Adobe Portable Document (PDF), and others. Reports can also be scheduled using a third-party scheduler such as Microsoft Scheduled Tasks, available from the Windows Control panel.

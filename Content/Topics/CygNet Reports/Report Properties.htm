Report Properties

CygNet Reports have two types of properties you can configure. Report Parameters properties define the report type and its data content, and Export properties define how and where to send the report file.

When a report template file (.rpt) is open, you can access the CygNet Reports Properties pages in one of the following ways.

  • On the CygNet Reports Report menu, click Properties
  • On the CygNet Reports toolbar, click the Properties icon

Once the Properties dialog box is open, two property pages will appear. Configure both types of properties to define how to produce and deliver the report.

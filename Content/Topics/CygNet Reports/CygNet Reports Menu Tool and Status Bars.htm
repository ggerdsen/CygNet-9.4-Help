CygNet Reports Menu, Tool, and Status Bars

Menu Bar

Menu Bar Description

File

Options are as follows.

  • New - create a report parameters file
  • Open - open an existing report parameters file
  • Save - save the report parameters file
  • Saves As… - save the report to a different name or location
  • Recent File - access recent files
  • Exit - close the application

Report

Options are as follows.

  • Run Report - generate the report
  • Print Report - print the report to a Microsoft Windows default printer
  • Export Report - export the report to a directory destination with a chosen file format
  • Export As - export the report in a number of different formats, such as an Acrobat PDF, character-separated values file, comma-separated values file, a Data Interchange Format (DIF) file, an HTML 3.2 or 4.0 file, ODBC data source file, Lotus 1-2-3, Microsoft Word, Crystal Reports, text, Rich Text, Report Definition file, an XML file, etc., with a choice of file location, application, Lotus Domino, Microsoft Exchange folder, or Microsoft email (MAPI)
  • Properties - view report properties

View
  • Toolbar - hide or show the toolbar
  • Status Bar - hide or show the status bar
  • Full Screen (or press F11) - maximize/restore the report view window

Help
  • Help - access the CygNet Help (press F1)
  • About CygNet Reports - open the About CygNet Reports dialog box to view application information.

Available information includes the following:

  • The application name, version, and release date
  • Copyright information
  • The CygNet Software website address, including a hyperlink to the website
  • The log file location, including a hyperlink to the log file
  • The location of the application's file executable

In addition to information about the application, the About box includes the following buttons:

  • Client Settings — opens the Client Settings dialog box to view and modify client log settings for the application on an ad hoc basis. See Client Log Settings in the Logging section for more information.
  • Diagnostics — access the following options for additional analysis from the drop-down menu:
  • Loaded Modules — use this option to view and search file information for modules loaded in your system. Double-click any result row to view properties for that module. See Loaded Modules for more information.
  • Gather Logs — accesses the Windows file directory where the open log files are located. Double-click to select and view an application log file.
  • User Local Crash Dumps — Local crash dumps are not enabled. See Generating Crash Dumps for more information.
  • Close — click to close the About box.

Toolbar

Toolbar Description

Creates a new report parameters file for completing

Opens a report parameters file (.rpx). Select the already-created report parameters file from the Standard File System or the BLOB File System

Saves the current report parameters file to either the Standard File System or the BLOB File System

Runs the report

Prints the report to a Microsoft Windows default printer

Exports the report

Opens the Properties dialog box

Navigation Bar

Except for the Page Indicator and the Zoom option, you can hover over each icon to see a description.

Navigation Bar Description

Toggle Group Tree presents a split screen, with the right pane displaying the report

Go to first page of the report

Go to previous page of the report

Go to next page in the report

Go to last page in the report

Page Indicator; the first number is the current page; the last number is the total number of pages

To go to a different page number, type the number and press Enter.

Stops loading the report

When you begin loading the report, the icon displays a red “X”, indicating that at this point if you click the icon, the report will stop loading.

Searches the report. Brings up a Search dialog box.

Page Size

The magnification percentage area is used for zooming in and out; both Page Width and Whole Page options automatically size the page within the report window, respectively. You can select a percentage or enter a new percentage.

After the report is run and opened in CygNet Reports, you can also right-click in the report view and from the context menu select navigation options as described above, such as First Page, Previous Page, Group Tree, etc.

DBS Tables

The DBS Tables page on the CygNet Service Information dialog box displays other relevant service information available for DBS-based services. For each DBS-based service, this page shows a list of database tables that the service provides. Each row shows the total record count for that table, index, record, header, segment, etc. These items do not have associated SVCMON UDCs.

Examples

  • Table Row Count: ACS_EVENT_SEG (TABLEROWCNT:ACS_EVENT_SEG) displays the record count for the Event Segment table for the ACS.
  • Table Row Count: AUD_AUX_INFO_RECORD (TABLEROWCNT:AUD_AUX_INFO_RECORD) displays the record count for the Aux Info Record table for the AUD.
  • Table Row Count: DDS_AD_INDEX (TABLEROWCNT:DDS_AD_INDEX) displays the record count for the AD index for the DDS.
  • Table Row Count: FAC_HEADER (TABLEROWCNT:FAC_HEADER) displays the record count for the FAC header for the FAC.
  • Table Row Count: GNS_ADDR_RECORD (TABLEROWCNT:GNS_ADDR_RECORD) displays the record count for the Addr Record table for the GNS.

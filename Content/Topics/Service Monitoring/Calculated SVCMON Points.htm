Calculated SVCMON Points

To Create a Calculated SVCMON Point

  1. In the Calculated Points view, select a UDC from the Defined Calculated UDCs list.
  2. Select the source point on which to perform the calculation from the Source Points list.
  3. Click the Add button (>>) to add the information to the Points to Create list.
  4. To apply the template of the Calculated UDC Point template, click the Use Templates option.
  5. Click Create Calculated Points. The new points will be shown in the Explore view and will be assigned to the same timer as the source point.

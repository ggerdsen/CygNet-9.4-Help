Timer Configuration

To Add a Timer

  1. On the Timer Config view, click Configure Timers to open the Timer Configurations dialog box.
  2. Click Create New Timer to open the Create New Timer dialog box.
  3. Type a name for the timer. Valid characters are A-Z, 0-9, dash (-) and underscore (_).The name cannot contain spaces.
  4. To apply the timer point templates (for timer state, cycle, and errors), click the Use template for support points option.
  5. Click Create.
  6. Click OK.

 

To Set a Timer Refresh Rate

  1. On the Timer Config view, click Configure Timers to open the Timer Configurations dialog box.
  2. Select the timer in the Timer to configure drop-down menu.
  3. Enter the refresh rate in the Seconds text box.
  4. Click Set Refresh Rate to apply the value.

To Delete a Timer

Before deleting a timer it is recommended that you first remove any SVCMON points assigned to the timer. If you do not remove the points, they will be listed under No Assigned Timer.

  1. On the Timer Config view, click Configure Timers to open the Timer Configurations dialog box.
  2. Select the timer in the Timer to configure drop-down menu.
  3. Click Delete Selected Timer. The utility will display a prompt asking if you want to delete the timer values from the VHS. These are the history values associated with the timer tags.
  4. Click OK.

Disabling a Timer

A timer can be disabled to stop updates.

To Disable a Timer

  1. On the Timer Config view, click Configure Timers to open the Timer Configurations dialog box.
  2. Select the timer in the Timer to configure drop-down menu.
  3. Click to clear the Enabled option.
  4. Click OK.

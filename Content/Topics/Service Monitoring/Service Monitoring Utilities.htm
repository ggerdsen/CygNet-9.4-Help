Service Monitoring Utilities

CygNet Software provides the following service monitoring utilities. All CygNet utility files are found in the CygNet\Utilities folder on your host server.

CygNet Utility Name Utility File Name Description Supported Checks Security
Service Diagnostics SvcDiags.exe Shows service, site, and machine statistics.  
ServiceMon Administration SvcMonAdmin.exe Creates Service Monitoring Service (SVCMON) points.

See Utilities for a complete list of SCADA utilities.

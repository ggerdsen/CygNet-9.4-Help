Table Notes:

  • CVS includes HSS, OPCIS, SVCMON and UIS.
  • DBS includes ACS, APPS, AUD, BSS, DBS, DDS, ELS, ELSALM, FAC, GNS, GRP, MSS, NOTE, PNT, and TRS.

SVMBGVALID Note:

  • For the DDS, the SVMBGVALID count represents a subset of the DDS check errors.
  • For the PNT, the SVMBGVALID count represents points whose GNS IDs, referenced points, table driven fields, or primary or alternate units are invalid. For the PNT, the service log lists each point that failed the first time it encounters the error. The point record will be modified to change the GnsIdEntryError, GeneralConfigError, EnhAlmConfigError, or KeyFieldEntryError field depending on which one is invalid. These fields can only be viewed via ODBC, CxPnt, CygNet Studio, and CygNet Group Grid.

Adding a SVCMON Point

Two methods can be used to add a SVCMON point.

Adding a SVCMON Point Using the CygNet ServiceMon Administration Utility

See the CygNet ServiceMon Administration utility to add a SVCMON point using the utility.

Adding a Non-Templated SVCMON Point Using an Existing Point

Any service info item can be mapped to a new SVCMON point by using an existing SVCMON point as a template.

To Add a SVCMON Point Using an Existing Point as a Template

  1. Right click on an existing SVCMON point in the PNT and click New.
  2. Choose a new unique UDC for the point.
  3. Select the Application page in the PNT Editor.
  4. In the General data fields section, replace the data in the General 2 field with the info item you wish to monitor.
  5. Select the Point reference page in the PNT Editor and check to make sure the Long point ID field is pointing to the SVCMON timer to be used.
  6. Save the point. A new SVCMON point containing the data of the specified info item as its current value is created.

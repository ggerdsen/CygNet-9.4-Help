Host Activation Warnings

CygNet provides warnings when Host Activation issues are discovered in your system.

Host Activation Notification Bar

When you use a CygNet client, a yellow notification bar appears if a host activation issue exists for services with which the client has communicated, for example activation is disabled, expired, or in error.

Click the yellow bar to access the CygNet Activation/Licensing Issues dialog box, which provides detailed information about the activation issue(s) encountered. Additional details are included in the dialog box, such as when the host activation expired. See Host Activation Verification for more information.

If you encounter this notification bar, take action to activate the referenced CygNet host.

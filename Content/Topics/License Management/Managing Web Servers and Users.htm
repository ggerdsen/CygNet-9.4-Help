Managing Web Servers and Users

CygNet has three types of web servers: Web Control, Web Status, and Web View. Web Control users are named users. Web View and Web Status users are concurrent users.

Each web license type has its own authorized limit. The License Manager view shows the number of authorized components for each type of web license.

The Status column in the various web views shows the status of the server/user license record.

Status Description

E

The license record is enabled.

D

The license record is disabled.

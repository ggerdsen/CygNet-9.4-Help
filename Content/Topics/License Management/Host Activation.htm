Host Activation

Note: CygNet hosts do not shut down automatically when they fall out of activation compliance. You will, however, receive warnings in various CygNet client applications (CygNet Explorer, CygNet Studio, CygNet Vision, and FMS Explorer) if a host is not activated. Best practice recommends using the Service Monitoring Service (SVCMON) to monitor status and set up notifications.

All CygNet hosts require activation (activation is separate from licensing). Host activation is a software key that must be obtained from CygNet Support.

The Status column in the Host Activation view shows the status of the host.

Status Description
DISABLED The host has been deactivated by a CygNet administrator.

ENABLED

The host is activated.
ERROR The activation key was not created for the system.
EXPIRED The 30-day trial has expired and the host was not activated.

TRIAL

The host is in trial mode.
UNKNOWN Host activation is in an unknown state.

See Host Activation Verification for more information.

To Request an Activation Key

  1. Make sure the CygNet services are running.
  2. Click the Host Activation icon in the left panel.
  3. Select the computer for which you want to obtain an activation code. You can request keys for multiple computers by selecting multiple lines. If necessary, expand the dialog box or scroll to view the Computer Name column.
  4. Click Request Keys.
  5. Copy the information shown in the dialog box into an email and send it to CygNet Support. Be sure to replace the placeholders of Contact Name, Contact Phone, and Contact Site Name with the information for your site.

To Activate the Host

  1. Click the Host Activation icon in the left panel.
  2. Select the computer for which you want to activate and click Activate Host. You can activate only one host at a time. If necessary, expand the dialog box or scroll to view the Computer Name column.
  3. Enter the key supplied by CygNet Support. If you have requested several keys, be sure to enter the key for the selected computer.
  4. Click OK.

Deleting Host Activation Records

The Delete Record button is for use by CygNet Support only. The button will not function without a CygNet Authorization Key provided by CygNet Support.

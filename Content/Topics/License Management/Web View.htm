Web View

A Web View server is licensed for concurrent users. Server ordinals are groups of concurrent users that must be assigned to a web server when it is enabled.

Web View Server

The Web View Server view shows the detailed records of your Web View servers.

To Create a New Web View Server License Record

  1. Click the Web View Server icon in the left panel.
  2. Click the Create New button.
  3. Type in the name of the Web Server.
  4. Click Create New Record.
  5. With the newly created record still selected, click the Edit button.
  6. Click the Enable button to enable the web server.
  7. Click to select the Server Ordinal and the associated Maximum User Count that will be assigned to the server.
  8. Click Set Ordinal.
  9. Click Done.

To Clear Server Ordinals for a Web Server

  1. The server must be in the enabled status to clear Server Ordinals.
  2. Select the server record you wish to un-assign concurrent users from.
  3. Click the Edit button.
  4. Click the Change button located beneath the Max Users box.
  5. Click the Clear Ordinal button.
  6. Click Yes to confirm the clearing of the ordinals.
  7. Click Done.

To View the Number of Users per Web View Server

  1. Click the Web View Server icon in the left panel.
  2. Click the Column Selection button.
  3. Click to check the Web Max Users option
  4. Click Done.

To view the number of users currently online, you must use the web server .asp admin page.

Web Control

The License Manager views that are specific to web components show the number of licensed web components. These web records must be added manually.

Note: Although you may add more web server records than you are licensed for, you will only be allowed to enable the number of servers you are licensed for.

You cannot use License Manager to add components such as additional web servers or web users to your license file, only to edit the existing licenses. In addition, enabling or disabling a record does not add another undefined record to the list.

Contact your Account Manager or CygNet Sales to purchase additional components.

Web Control Servers

The Web Control Server view shows the detailed records of your Web Control servers.

To Create a New Web Control Server License Record

  1. Click the Web Control Server icon in the left panel.
  2. Click the Create New button.
  3. Type in the name of the Web Server.
  4. Click Create New Record.
  5. With the newly created record still selected, click the Edit button.
  6. Click the Enable button to enable the web server.
  7. Click Done.

To Edit a Web Control Server License Record

  1. Click the Web Control Server icon in the left panel.
  2. Select a server record in the list.
  3. Click the Edit button (or double-click the record) to show the record’s properties.
  4. In the Name box, type the IP address and, if necessary, the virtual sub-site name (for example, 192.164.1.35/Web Control).
  5. Click Change to save the changes to the record.
  6. Click Done.

Web Control Users

The Web Control User view shows the detailed records of your Web Control users.

To Create a New Web Control User License Record

  1. Click the Web Control User icon in the left panel.
  2. Click the Create New button.
  3. Enter the Web Control User in the format domain\username.
  4. Click Create New Record.
  5. With the newly created record still selected, click the Edit button.
  6. Click the Enable button to enable the web user.
  7. Click Done.

To Edit a Web Control User License Record

  1. Click the Web Control User icon in the left panel.
  2. Select a "webuser" license and click the Edit button.
  3. In the Name box, type the domain\userID of the user (for example, ACME\Alfred.E.Neuman).
  4. Click Change to save the changes to the record.
  5. Click Done.

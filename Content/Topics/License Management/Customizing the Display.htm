Customizing the Display

Filtering Records

You can filter the list of licenses based on the status of the license. The check boxes in the top of the window are the status filter options.

When a filter is enabled (checked), the display lists only those licenses that meet the criteria.

Columns

The columns in the display can be modified using the Column Selection button above the record list. Column visibility settings are specific to each view type.

Exporting License Manager Contents

Each view of License Manager can be exported to a CSV file. This file can then be opened with another application, formatted and printed.

To Export License Manager Data

  1. Click on the desired view in the left pane.
  2. Click the Export Data button.
  3. Click … to select the file you wish to export to. If the file does not already exist it will be created during the export process.
  4. Select the desired Records to Export option.
  5. Select the desired Columns to Export option.
  6. Click the Export to CSV button.
  7. Click the Close button.

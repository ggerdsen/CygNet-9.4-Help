Configuring Facility Attributes

To Configure Facility Attributes

  1. Right-click in the Facility Service pane and click Configure Attributes….
  2. Click the Edit button to begin configuring attributes for the facility service.
  3. Click the In Use box to enable an attribute.
  4. Type in the Attribute Description up to 20 characters.
  5. For table attributes 0-29 click the View browse button to create table attribute entries. These entries appear in the drop-down menu for the specific attribute when assigning attributes to a single facility.
  1. To add a table entry click Configure.
  2. Click Add.
  3. Type an Entry for the table up to 10 characters long.
  4. Type a Description for the entry up to 40 characters long.
  5. Click OK.

- To remove an entry click to highlight it then click Delete.

- To edit an existing entry click to highlight it then click Edit.

  1. Repeat steps 5a-5e until all table entries have been added.
  2. When all table entries have been added for the attribute table, click Close.
  3. Click Close to exit the Select Table Entry dialog box.
  1. If desired type to enter a Category. This entry can be used later to filter attributes in the Properties of a facility.
  2. Repeat steps 3-5 until all the desired attributes have been defined.
  3. Click Cancel to save the attribute configuration.

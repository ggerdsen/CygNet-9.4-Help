Applying the Same Attributes to Multiple FAC Services

If you want to use the same attributes in two different FAC services, you can copy the MetaData.dat file from one FAC folder to the other. Be aware, however, that you still will need to create the table entries for any table attributes. You can use the utility TRSLoad.exe to download the tables from one TRS and import them into another.

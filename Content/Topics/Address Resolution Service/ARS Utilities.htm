ARS Utilities

CygNet Software provides the following utilities for use with the Address Resolution Service (ARS). All CygNet utility files are found in the CygNet\Utilities folder on your host server.

CygNet Utility Name Utility File Name Description Supported Checks Security

ARS Diagnostic Utility

ArsDiags.exe

Provides diagnostic information for the ARS including service validation, iterator test, site consistency, cycle times, and CVS metadata file synchronization.

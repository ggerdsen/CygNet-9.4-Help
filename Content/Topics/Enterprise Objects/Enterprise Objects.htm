Enterprise Objects

Enterprise Objects is a featured component of the CygNet Enterprise Integration Suite (EIS).

Enterprise Objects provides Enterprise-friendly XML-based interfaces between the Business System (Enterprise) Data model and the CygNet Native (SCADA) Data model. These interfaces facilitate the exchange of data between the CygNet Software platform, third-party applications, and other business systems.

The following topics describe how to retrieve and transmit CygNet native data using CygNet Enterprise Objects.

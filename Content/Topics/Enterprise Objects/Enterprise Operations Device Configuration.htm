Enterprise Operations Device Configuration

Each Enterprise Operations EIE driver instance (device) is configurable to retrieve request XML files from the Requests folder on the file system. The XML response is configurable to post to the Responses folder on the file system.

To Configure an Enterprise Operations Device

  1. Using CygNet Explorer, click on the DDS and select Import/Export Devices.
  2. Right-click in the white space and click New Import/Export Device.
  3. In the Select a Device Type… list select Enterprise Operations and click OK. The Device Properties dialog box will be displayed.
  4. On the Device page define the device instance, including the operational security.
  5. On the File I/O page note the locations of the Input Requests and Output Responses directories.

    6. Note: If you are using the Enterprise Gateway, these locations must be specified in the GatewaySettings.xml file.

  6. On the Native Operations page enable desired CygNet Native operations.
  7. On the Business Objects page enable desired Business Object operations.
  8. On the Options page configure desired options and parameters.
  9. On the Advanced page export device settings to an XML file location, or load device settings from an existing XML file, as desired.

See Enterprise Operations Device Properties for more information about configuration settings.

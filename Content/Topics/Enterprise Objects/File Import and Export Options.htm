File Import and Export Options

You can use the Business Object Builder to import and export Business Object Package files using options on the Package page.

Outside sources of interface and point definition mapping files can be utilized in new or existing packages by using the CygBOB Import options. Interface and point definitions files created for one package can be utilized in related tasks by using the CygBOB Export options.

File Import and Export options can also be used to copy mappings within the same package, by exporting the files with a new name and then importing them back into the package.

To Import an Interface File or Mapping

  1. On the Package page, click Import to display the Import Package Items pane.
  2. Click Load Package to browse to the directory containing the file or point definition you want to import.
  3. Select the desired file and click Open to display a tree view of the selected package.
  4. Click to select the aspects of the file, objects, point definitions, or mappings, that you want to import. Dialog boxes indicating dependencies will be displayed to aid in your selection/de-selection process.
  5. When all selections have been made click Import.
  6. If there is a name collision, a dialog box will be displayed indicating the duplication, and providing options to Rename, Skip, or Cancel the import.
  7. Name and save the imported object or point definitions files.
  8. A dialog box will be displayed indicating that the Import succeeded
  9. Click OK.

To Export an Interface File or Mapping

  1. On the Package page, click Export to display the Export Package Items pane including a tree view of the selected package.
  2. Click to select the aspects of the file, objects, point definitions, or mappings, that you want to export. Dialog boxes indicating dependencies will be displayed to aid in your selection/de-selection process.

    3. Note: During this process you can select specific package items to export.

  3. When all selections have been made click Export. A Save As dialog box will be displayed.
  4. Browse to and select the file name and directory location for the export, and click Save.
  5. A dialog box will be displayed indicating that Your package was exported successfully
  6. Click OK.

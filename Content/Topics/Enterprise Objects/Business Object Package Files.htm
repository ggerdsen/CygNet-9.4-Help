Business Object Package Files

Business Object Package file formats encapsulate the following:

  • Business Object Interfaces
  • Business Object Mappings
  • Point Interfaces
  • Point Mappings

Each of these elements can be created specifically for the Package as it is built. Alternatively, an existing Package file can be copied, Interface files and Point definition files can be imported for use from outside sources, or files can be exported for use in related tasks.

Creating Package Files

To Create a Business Object Package File

  1. On the CygBOB menu, click the Package tab.
  2. Click New.
  3. Type in a name for the new package and click Save.
  4. Click the Edit tab.
  5. Create a Business Object Interface.
  6. Create a Business Object Mapping.
  7. Create Point Interfaces.
  8. Create Point Mappings.
  9. On the Package page, click Open. Navigate to the Package you created and select it. Click Open.
  10. Verify that the newly created elements are displayed. Click Save.
  11. If the file has not previously been named, the Save As dialog box will open. Enter the desired file name in the File Name field field and click Save.

Opening, Editing, Copying, and Deleting Package Files

To Open a Business Object Package File

  1. On the CygBOB menu, click Package to open the Package window.
  2. On the Package menu, click Open.
  3. On the Open dialog, navigate to the desired file.
  4. Click on the file name to select it and click Open, or double-click on the file name, to open it.
  5. Click OK.

To Edit a Business Object Package File

The elements of an existing Business Object Package File can be edited on the Edit page after opening the package on the Package tab.

To Copy a Business Object Package File

  1. On the CygBOB menu, click Package to open the Package window.
  2. On the Package menu, click Open.
  3. On the Open dialog, navigate to the desired file.
  4. Click on the file name to select it and click Open, or double-click on the file name, to open it.
  5. Click Save As.
  6. Browse to the desired location, and enter the desired name for the new package file.
  7. Click Save. You can now open and edit the package created.

To Delete a Business Object Package File

  1. On the CygBOB menu, click Package to open the Package window.
  2. Click Open, navigate to the package file desired, and right-click to select it.
  3. Click to select Delete from the drop-down menu. A message will be displayed asking "Are you sure you want to move this file to the Recycle Bin?"
  4. Click Yes.

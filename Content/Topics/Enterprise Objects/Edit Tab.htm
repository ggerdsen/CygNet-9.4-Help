Edit Page

Business Object Package Files, objects, point definitions, and their associated mappings can be edited on the Edit page.

Options available on the Edit page include the following.

Objects

Option Description

Objects

 Lists the business Objects belonging to the currently selected package.

Interface for Object

 Displays the list of Interface Attributes associated with the selected Object. Allows addition and/or removal of attributes, and designation of attributes as key, optional or writable. See Business Object Interfaces for more information.

Primary Mapping

Displays the Primary Mapping for the selected Object, and allows editing of Attributes and configuration of History Rollup. See Business Object Mappings for more information.
Object Definition Displays the definition of the Business Object in terms of native CygNet constructs, in order to evaluate the state of SCADA entities (facilities, points, etc.) before representing them as valid instances of Business Objects.
Current Value Services Displays Current Value Services targeted for data retrieval by the Business Object operations. New CVSs can be added or removed in this field. See Services for more information.
Attribute Name

Lists the name of the Business Object Attribute being mapped to the native CygNet Attribute. See Filter Attributes for more information about the native CygNet attributes.
Mapped To Lists the name of the native CygNet Attribute to which the Business Object Attribute is being mapped.
UDC Lists the UDC of the native CygNet Attribute.
Time Adjustment Allows configuration of History rollups and date/time adjustments. See Time Adjustment for more information.
Filter Displays Object Mapping Filter options.

Point Definitions

Option Description

Point Definitions

 Lists the Point Definitions belonging to the currently selected package.
Add New Point Definition Adds a new Point Definition to the list and opens a corresponding new point definition Interface.

Interface for Point Definition

Displays the Interface Attributes for the selected Point Definition, and allows addition and/or removal of attributes. See Point Interfaces for more information.

PtMapping

Displays Point Mappings and Filters for the selected Point Definition. Every Point Definition Attribute must have an associated Point Mapping to a CygNet native attribute. See Point Mappings for more information.
Filter Displays Point Mapping Filter options.

General

Option Description
Pins the Navigation Menu to the sidebar.
Unpins the Navigation Menu from the sidebar.
Expands the list (for example, of Objects or Point Definitions) displayed.
Collapses the list (for example, of Objects or Point Definitions) displayed.
X Removes the associated Object, Point Definition, or Mapping. Displays a confirmation dialog box prior to removal.
+ Adds new Mapping(s).

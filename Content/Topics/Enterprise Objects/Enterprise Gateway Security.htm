Enterprise Gateway Security

The Enterprise Gateway maintains a list of permissible users. When an authenticated user connects to the Gateway, the list of users is checked before allowing the request to be processed.

The Enterprise Gateway enforces the user ID field provided in the request XML. If the user ID in the request XML does not match the authenticated user, the Enterprise Gateway will reject the request, and an error report will be generated.

See Enterprise Objects Security for further information.

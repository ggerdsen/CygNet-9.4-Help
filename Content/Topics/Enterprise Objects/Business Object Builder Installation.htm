Business Object Builder Installation

The CygNet Business Object Builder (CygBOB) can be installed using the CygNet Client Installer (CInstall.exe) for installation on workstations using CygNet, or using the CygNet Business Object Builder Installation application for installation on workstations not using CygNet.

  • Install CygBOB on Workstations Using CygNet
  • Install CygBOB on Workstations Not Using CygNet

Before You Install

For successful installation of the CygNet Business Object Builder (CygBOB), minimum hardware and software requirements must be met. See the CygNet System Requirements document for more information. The following applications must be installed prior to installing CygBOB:

  • Internet Explorer 6.0+
  • MSXML 6.0
  • Microsoft Visual C++ 2010 Redistributable Package (x86)
  • Microsoft .NET Framework, in accordance with the CygNet System Requirements document
  • ODBC must be installed on the server running the UIS and the Enterprise Operations EIE.

To Install CygNet Business Object Builder on Workstations Using CygNet

  1. Open the CygNet Client Installer (Start Menu > All Programs > CygNet > CygNet Client Installer). The CygNet Client Installer/Launcher dialog box will be displayed.

  1. On the BLOB Service: Admin drop-down menu, select the APPS Service.
  2. In the BLOB Folder list select UTILS.
  3. In the Select CygNet Applications to Install list select CygNet Business Object Builder.
  4. Click the check box adjacent to Add shortcut to desktop to select/clear this feature.
  5. Click Install. A confirmation message will be displayed asking if you want to start the utility.
  6. Select Yes to start CygNet Business Object Builder.

To Install CygNet Business Object Builder on Workstations Not Using CygNet

  1. In the Setup Folder, double-click to select the CygNetBusinessObjectBuilderSetup application. The CygNet Business Object Builder installation dialog box will be displayed.
  2. Follow instructions provided by the Installation application to guide you through installation of the CygNet Business Object Builder.

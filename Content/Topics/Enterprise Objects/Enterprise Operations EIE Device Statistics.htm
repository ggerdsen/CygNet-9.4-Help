Enterprise Operations EIE Device Statistics

System UDCs are provided for monitoring Enterprise Operations device statistics. Points for these UDCs are automatically created in the UIS when an Enterprise Operations device is created.

The Service of the point is the UIS, the Facility is the name of the Enterprise Operations device.

UDC Device Info Description Point Type

SYEDFOP

 Failed Ops

The number of failed operations.

Analog Input

SYEDIPOP

 Pending Ops

The number of pending operations in the dispatch object. 

Analog Input

SYEDSOP

 Successful Ops

The number of successful operations.

Analog Input

SYEOCHCT

 Last file char count

The character count of the last response XML exported.

Analog Input

SYEODURN

 Last op duration

The duration in milliseconds of the last operation.

Analog Input

SYEOFLOP

 Last failed op

The name of the last failed operation.

String Input

SYEOLTOP

 Last op name

The name of the last completed requested operation, regardless of its status.

String Input

SYEOPCCT

 Processed files

The total number of request files processed.

Note: A request that failed to be imported by the EIE is considered to be processed.

Analog Input

SYEOSCOP

 Last successful op

The name of the last successful operation.

String Input

SYEOSTAT

 Last op status

The status of the last operation (0 = No Error; 1 = Error).

Analog Input

Using Notes

Once a Note Service is installed and configured, you may begin to use it in a production setting. The following topics focus on using your notes feature.

Notes Editor

Note configuration is made in the Note Properties dialog box, accessible from CygNet Explorer.

  • Right-click in the service pane and select New to configure a new note.
  • Right-click a note and select Properties (or double-click) to open an existing note record. By default the Note Properties dialog box opens in View (or read-only) mode. Use the Previous and Next buttons to scroll through the configured notes in View mode.
  • Click Edit to modify a note. The Previous and Next buttons are disabled in Edit mode.
  • If changes are made to a note and the Note editor is canceled, a prompt dialog will appear allowing you to Save, Discard, or Cancel and return to the editor.

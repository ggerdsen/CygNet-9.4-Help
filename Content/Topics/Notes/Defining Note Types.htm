Defining Note Types

Notes may be categorized by user-defined type. Types are accessed from the Note Service pane within CygNet Explorer. Categorizing notes by type provides a way to define the kinds of notes appropriate to your organization, to more easily filter existing notes, and to comprise an entire note when only a simple label adequately describes a note.

For example, you could create the note types ADMIN and SHIFT to categorize administrative notes as separate from those made by a shift operator. You could then filter a search for only ADMIN notes or only SHIFT notes. You could also create the note types ARRIVAL and DEPARTURE to mark the arrival and departure of a plunger. In the case of ARRIVAL and DEPARTURE, no other note text may be needed to signify the event.

To Create a Type Definition

  1. Using CygNet Explorer, right-click in the Note service pane, then click Configure Attributes. A dialog box like the following appears:

  1. Click the Type Definitions page, then click Edit.
  2. Click Add to add a new type definition.
  3. Enter the name of your new type in the active text box.
  4. Continue to click Add and define types until you've added all required note types. Your dialog might look something like this:

  1. When you are done adding note type definitions, click Save. The newly added types become available for assignment.

Type Definitions Properties

The following table lists and describes the properties of the Type Definitions page.

Property Description
Add Enables the addition of a new note type.
Rename Enables you to rename an existing type definition and push out that change to all notes that currently use the changed type.
Delete

Enables the removal of an existing note type from the Types list.

If a note type is removed but has already been assigned to one or more notes, the note type value remains unchanged on the affected note(s). However, the removed note type no longer appears as a type option for all notes. To change affected notes to a different type, you must select each individually and update their type.
Edit Enables the editing of an existing note type.

BSSC Bulk Processing

You can list BSSC command lines in a text file for bulk file processing.

Syntax

Operation Folder FileName

Example

An example text file is shown below.

# MyFiles.txt

 

# Reports

U REPORTS Reports\Area1_Report.xls

U REPORTS Reports\Area2_Report.xls

U REPORTS Reports\Area3_Report.xls

 

# Support Files

U SUPPORT ..\SUPPORT\CXVHS.dll

 

# Screens

R SCREENS C:\MYSCREENS\METERDETAIL.dll

Notes:

The file can contain any combination of operations (read, write, update, delete, add, copy).

Each file must have its own line.

Each element in a line must be separated by a single space.

When downloading files from the BLOB the files will be written to the folder from which you run the utility unless you specify a path.

The pound sign (#) remarks out a line.

Bulk Processing Syntax

BSSC SITE.SERVICE File name.txt

Example

BSSC MYSITE.BSS MyFiles.txt

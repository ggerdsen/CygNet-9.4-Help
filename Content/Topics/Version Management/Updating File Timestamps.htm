Updating File Timestamps

TouchBSS is stored in the CygNet\Utilities directory (TouchBss.exe) on the host server. To start the utility, browse to the directory using Windows Explorer and click the program icon.

Field Description

Blob Site.Service

Use … to select the Site.Service of the BSS.

Blob File Type

Type the file extension. Use * for all file types.

Application

Type the BSS application group. In the APPS BSS, typically groups are Clients, Support, Samples. Use * for all groups.

Blob Name

Type the file name. Use * for all files.

Once the parameters have been specified, click OK to update the timestamps. A command window will show the progress. Once the process completes, click Cancel to exit the utility.

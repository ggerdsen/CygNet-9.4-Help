EFM Data Items: FMS History

The flow measurement history data item strings that follow describe FMS History data items supported by CygNet.

See CygNet Measurement for more information.

FMS History Data Items

History data item strings are listed in the following table.

The table lists history data item strings in sections according to type of usage.

  • Periodic Metering
  • shared gas/liquid
  • shared gas/liquid custom
  • gas-specific
  • gas-specific custom
  • liquid-specific
  • liquid-specific custom
  • Periodic Quality
  • shared gas/liquid
  • shared gas/liquid custom
  • gas-specific
  • liquid-specific

See FMS Data Items Units for information about valid units for history data items requiring units.

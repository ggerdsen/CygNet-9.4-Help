BSAP EIE

The BSAP EIE supports data communication between CygNet and BSAP devices, whether networked or stand-alone. Data groups can be configured to reflect data assignments in the ACCOL load file for BSAP 33xx devices, which can be loaded through the EIE editor. Device-level data groups can be configured specifically for device instances.

Electronic Flow Measurement Data Items

Note: At this time, electronic flow measurement (EFM) data item support refers only to the CygNet Measurement (FMS) application, whereas previously it included the now discontinued Gas Measurement Repository (GMR) application. Equivalent GMR data support is now provided by FMS, using "FMS Legacy" data items in cases where only GMR support was previously available.

All CygNet electronic flow measurement (EFM) services use a system of mapping data group elements to EFM data items. In order for field devices to successfully pass data through a remote device to an EFM service, the field device data group elements must be accurately mapped.

The following topics provide information about mapping data group elements to EFM data items within a device template file. Doing so relies on having the correct EFM data item names. These topics list all of the potentially available EFM data items for each data item class.

Mapping relationships and supported EFM data items vary significantly by remote device EIE type. See the Remote Devices topic for your specific remote device type(s) for information about which EFM data items map to which data groups or data group elements.

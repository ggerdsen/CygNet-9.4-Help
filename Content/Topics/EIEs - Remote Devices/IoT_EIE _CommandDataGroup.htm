Command Data Group

The IoT EIE allows a user to write a payload using a command manually through the "Command" (Command) data group using the data group editor or via a UIS command. Sample commands (NPublish and DPublish) are defined in the template file.

See UIS Command Component Parameters for more information.

DPublish

This command sends a device command. Commands are defined in the template file in the "commands" section for a data group.

NPublish

This command sends an edge-node command. Commands are defined in the template file in the "commands" section for a data group.

Configuring the Command Data Group

The Command data group is configured within the View Data dialog box of the data group. To configure the Topic section, see the IoT EIE Data Group Properties table. The following is an example of a View Data dialog box for a Command data group.

 

To configure the command for a Command data group, make selections from the fields below the Topic section.

Property Description

Command

Select an option from the drop-down menu. Commands are defined in the template file.

Description

Displays the description of the command from the template file.

Value

Displays the value of the command from the template file.

Facility ID.UDC

Enter a facility and UDC in the field. The Facility ID is used only in point processing.

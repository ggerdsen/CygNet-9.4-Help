Measurement Units

Emerson ROCPlus field devices can be configured to display values in defined sets of imperial/US or metric measurement units. Once you have defined a unit set, the field device tells CygNet what unit set it is running and CygNet displays the appropriate fields on remote device editor dialog boxes. For Emerson ROCPlus devices, this information is viewed on the Station Parameters data group Advanced page.

To Set Unit Type

For Emerson ROCPlus remote devices, set the unit type by doing the following:

  1. In CygNet Explorer, open your Device Definition Service (DDS).
  2. Click the remote device you want to configure.
  3. In the remote device editor, click the Data Group page.
  4. Double-click Station Parameters and click the Advanced page.
  5. Click Edit, then select the US or Metric option under Units.
  6. When you are done, click Send to RTU.

*For Emerson ROCPlus remote devices, users can control the type of viscosity units used. Viscosity values in the range 0.005 to 0.5 are displayed as metric (i.e., cp) by default and no conversion occurs. If a viscosity value is outside of this range, the units are set to imperial or metric, depending on how the device is configured, and no conversion occurs. If you want to pass through units for this value range, set the "FMS Configuration" (FmsConfig) data group attribute PassVisc to "true". If you do not want to pass through units for this value range and instead want them to be set by CygNet (the default behavior), set the attribute value PassVisc to "false" or remove PassVisc.

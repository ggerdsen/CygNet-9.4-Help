FMS Legacy Events Data Group

Note: Only configuration change events are supported.

The "FMS Legacy Events Data" (GmrEvent) data group is used to send specific historical data to the FMS. It gets its data from the "Change Log" data group, which is hard-coded to support "FMS Legacy Events."

Mapping between event data group elements and FMS occurs within the eventItems section of gmrMaps in the device template file.

Specify date ranges on a per-data group basis within the device editor. For more information, see Selecting Date and Time.

EFM Data Items: FMS Configuration

The flow measurement configuration data item strings that follow describe FMS Configuration data items supported by CygNet.

Note: The eventItems section of your device template file does not contain configuration data items. However, changes to any data item in the configItem section results in an event being sent to FMS. Configuration data items are omitted from the eventItems section of the device template file to avoid data item duplication within the device template file.

See CygNet Measurement and Using the Configuration Control for more information about configuration data item usage in CygNet Measurement.

Configuration data item strings are listed in the following table along with their corresponding value options.

The table lists configuration data item strings in sections according to type of usage.

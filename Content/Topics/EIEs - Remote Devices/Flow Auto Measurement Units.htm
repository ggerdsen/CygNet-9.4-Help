Measurement Units

Flow Automation field devices can be configured to display values in defined sets of measurement units. Once you have defined a unit set, the field device tells CygNet what unit set it is running; CygNet data groups might or might not display those units on remote device editor dialog boxes. The configured unit set type can be viewed at the EngUnits data group element of the "System Parms (native)" data group. Possible EngUnits values are as follows: 0 - English, 1 - Canadian Metric, 2 - SI Metric, and 3 - GOST Metric.

See also Defining Measurement Units and Configuring Unit Set Definitions.

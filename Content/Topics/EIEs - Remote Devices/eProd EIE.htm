eProd EIE

The eProd EIE supports interoperation with several common rod pump control (RPC) field devices that use the (Baker) 8500 protocol, most notably those now offered by Weatherford.

The eProd EIE supports the ability to view surface and downhole dynagraph cards from the remote device editor or by means of the CygNet Dynagraph Viewer client application. In addition to graphing pump cycle data, this functionality enables you to set a pump-off setpoint that protects machinery while optimizing recovery.

Another useful feature of the eProd EIE is the "Command Message (Configurable)" data group. This is an ordinalizable data group that provides a means to add command messages for a number of Command Action (CA) parameters. While instances of this data group may be added on an ad hoc basis from the remote device editor, they can also be added at the device template file level for simultaneous implementation in numerous field devices.

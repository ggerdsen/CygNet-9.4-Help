EFM Data Items: FMS Legacy Event Data Items

The flow measurement event data item strings that follow describe FMS Legacy Event data items supported by CygNet.

See FMS Legacy Event Types for more information about legacy flow measurement event data types.

Note: FMS Legacy events and alarms use the same set of event items. Whether an event data item is an alarm or not depends upon the value of the status from the remote device.

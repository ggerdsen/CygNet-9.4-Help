Elements and Attributes

This section provides reference details about elements and attributes as they are used in CygNet device template files. These components rely on the structure prescribed and validated by CygNet. For a high-level overview of how elements interrelate, see sample template files in your source and Template File Hierarchy.

Not all of the elements and attributes documented here work with all EIEs. Many of these components are only useful in specific situations or with specific EIEs. Moreover, a small number of attributes sharing the same name do different things in different EIEs. Wherever possible, elements and attributes that are used in specific situations or in specific EIEs are called out.

Configuring a Data Group

Data groups can be added to a remote device in one of two ways:

  • By means of a device editor
  • By means of a device template file (.dtf)

Generally speaking, data groups are first defined in a device template file so that they then become available in the remote device editor for addition, configuration, and ordinalization. The following procedure explains how to configure a data group in the remote device editor after it has been added to a device template file. For information about defining data groups in a device template file, see various topics under Device Template Files.

When you add a remote device to the Device Definition Service (DDS), default data groups are listed on the Data Group page. If other data groups are available, they are listed in the Select Data Group to Add dialog box. Access the Select Data Group to Add dialog box by clicking Add on the Data Group page in the DDS. When you add a new data group, you must define its properties.

To Configure a Data Group

  1. Using CygNet Explorer, open the Device Definition Service (DDS).
  2. Double-click the remote device you want to add a data group to.
  3. Click the Data Group page.
  4. Click in the white space of the Data Groups field and click Add to create a new data group.
  5. Fill in data group properties according to your needs and click OK. Some data group property editors have special options, like advanced configuration buttons. See specific remote device documentation for device-specific requirements.
  6. Select the new data group and click Map UDCs. This enables you to map CygNet uniform data codes (UDCs) to remote device data group elements.
  7. Select a data group element and click Map.
  8. Select the correct UDC and click Assign UDC. Use the UDC Category drop-down filter to narrow possibilities. Click OK.
  9. To remove a UDC assignment, select a data group element and click Remove Mapping, then click OK.
  10. Click Apply.
  11. Click the UIS Commands page to define commands for the data group. For more information, see Configuring a UIS Command.

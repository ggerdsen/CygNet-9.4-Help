FMS Config Data Group Mapping for Gas Meters

The following table aligns "FMS Config" data items for gas meters with the Emerson ROCPlus data groups and elements they map to.

Note: Native data group references to "Station Parameters" in the following table apply to either the data group "Station Parameters" or the data groups "Station Parameters (1 of 2)" and "Station Parameters (2 of 2)."

For more information, see EFM Data Items.

EFM Data Items: FMS Alarm

The flow measurement alarm data item strings that follow describe FMS Alarm data items supported by CygNet.

See CygNet Measurement for more information.

FMS Alarm Data Items

Alarm data item strings are listed in the following tables. See FMS Data Items Units for information about valid units for alarm data items requiring units.

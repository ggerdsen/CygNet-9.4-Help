EFM Data Items: FMS Legacy Configuration Data Items

The flow measurement configuration data item strings that follow describe FMS Legacy Configuration data items supported by CygNet.

Note: The "FMS Legacy Config Send" data group has no configuration data items unique from those used by the "FMS Legacy Configuration" data group. "FMS Legacy Config Send" data items are a subset of "FMS Legacy Configuration" data items.

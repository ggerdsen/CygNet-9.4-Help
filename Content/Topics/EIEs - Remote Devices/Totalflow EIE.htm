Totalflow EIE

Important: The correct version of the Totalflow API (tcidll.dll or tcidll64.dll) must be installed in order for the Totalflow EIE to operate properly. For more information, see Totalflow API.

The Totalflow EIE supports 4 generations of Totalflow flow computers (FC) and/or remote controllers (RC) as well as other ABB API-supported devices. Generations supported are Totalflow G1, G2, G3, and G4. Totalflow G3 and G4 devices were once known as XSeries devices.

The CygNet interface to the ABB API is templated, which means you can define device template files to suit specific needs. User-defined device template files allow you to match Totalflow device register configurations, to add new register-based data groups, and to retrieve device-level data from an ABB trend. Sample device template files for Totalflow G1, G2, G3, and G4 devices are available from CygNet; they include representative register-based data groups. Many Totalflow device configurations are possible; because of this, create a new device template file for each device model and/or configuration.

The CygNet Totalflow EIE provides several ways to retrieve current volume data. Each current volume data group element serves a specific purpose and retrieves a unique kind of data. Familiarize yourself with each before choosing how you to retrieve current volume data.

EFM Data Groups

Electronic flow measurement (EFM) data groups enable the passing of data from EIEs to the specialized CygNet EFM service, FMS, utilized by CygNet Measurement. Data relevant to each EFM data group type is displayed within each data group's respective remote device editor/viewer. Although FMS and FMS Legacy data groups are similar to one another, they differ in significant ways. Data groups within each EFM service also differ somewhat as used by specific EIEs.

All EFM data groups are dependent upon specific support data groups, consisting of the native data groups from which EFM data groups draw their data. No EFM data group works without the required support data group(s).

Some EIEs expose their support data groups in a device template file where they are available for addition, deletion, and modification. An example of an EFM device EIE that uses a highly configurable device template file is the Modbus EFM EIE. Support data groups are defined within a supportDg element within the EFM data group in its device template file.

Note: In the supportDg list in the device template file for FMS data groups, if you want to include a data group that is not included on the FMS ordinal, that data group must be on ordinal 0.

Other EIEs do not allow alterations of support data groups in a device template file. Support data groups are still required for EFM data groups, but they are hard-coded in the device driver. An example of an EFM device that has pre-defined, hard-coded support data groups is the Emerson ROC EIE.

FMS Legacy Data Groups

Although the GMR application has been discontinued, the following data groups are currently supported in FMS, and represent a subset of FMS support that includes data items previously supported in GMR. See CygNet Measurement for more information.

The following is a list of available FMS Legacy data groups.

  • FMS Legacy Configuration Data
  • FMS Legacy Config Send Data
  • FMS Legacy Events Data (Alarms are a special class of events. No unique Alarms data group exists for FMS Legacy.)
  • FMS Legacy History Data

See Configuring Remote Devices for CygNet Measurement for information about using these data groups.

See EFM Data Items for information about specific data items.

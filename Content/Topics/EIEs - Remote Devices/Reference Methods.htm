Reference Methods

Reference methods enable you to use data group elements within a data group to convert data from a data source into a different format. Reference methods may reference raw device data or data converted by a conversion method. Unlike conversion methods, however, reference methods only perform client-side conversions. Conversion method data is stored to the Device Definition Service (DDS), but reference method data is not.

The following topics list and explain common ways you can use reference methods to perform conversions. You can employ reference methods to customize templates to interface with a field device in a way that best suits your needs. For example, a field device might return a value that is represented in inches, but you want to display the value in feet. A reference method could be defined in the template that converts a value from inches to feet.

The topics below include examples of basic methods and advanced methods. It is useful to first look at a basic example before progressing to more advanced examples. Your template design and needs could further complicate the use of available methods. Whenever possible, minimize template complication by using the most straightforward method available to meet your needs.

For information about device template file elements and attributes, see Elements and Attributes.

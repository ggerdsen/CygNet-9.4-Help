Mapping EFM Data Items for Flow-Cal Export — FMS Only

Data is mapped differently for export to Flow-Cal, when sending data from the CygNet Measurement application (FMS).

Mapping FMS Data Items to Flow-Cal for Export

FMS data items can be mapped to Flow-Cal fields, and can be edited and customized, in the command definition file (.cdf) for the corresponding command type. See the following topics for more information.

  • Configuring Export: Flow-Cal Data CFX Commands
  • Configuring Export: Flow-Cal Ticket Data TFX Commands

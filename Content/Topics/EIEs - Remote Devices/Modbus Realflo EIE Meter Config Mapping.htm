Meter Configuration Mapping

This topic correlates supported meter run types with their respective configuration data groups and input data groups. While the tables provide a "complete" picture of the applicable data groups for a given run type, you might not need to poll all input data groups; you determine which input data is applicable to your needs. In general, meter configuration data groups must be polled, but supporting input data groups can be polled on an ad-hoc basis.

To supply data to CygNet Measurement, ensure that all required support data groups for a given meter type are instantiated in the remote device editor. Locate the FmsConfig data group in the applicable device template file and survey its supportDg element for the list of possible support data groups you must instantiate. Exactly which support data groups varies by meter type.

For information about which data groups supply input data, see Data Groups.

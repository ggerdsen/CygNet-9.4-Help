EFM Data Items: FMS Send Gas Quality

The flow measurement send gas quality data item strings that follow describe FMS Send Gas Quality data items supported by CygNet. These data items can be sent to the device via CygNet Measurement utilizing a Send Gas Quality command. See CygNet Measurement and Send Gas Quality for more information.

Note: "FMS Send Gas Quality" data items are a subset of "FMS Configuration" data items; none of the following data items is unique from "FMS Configuration" (Quality) data items. See FMS Configuration Items for more information.

RSM Diagnostic Tool

The RSM Diagnostic Tool (RsmDiags.exe) is an administrative utility available to assist users configuring CygNet Redundancy. The tool verifies the consistency of redundancy definitions and services and their owners across RSMs in a redundancy environment. The tool allows you to see the domain and site service of every service owned by every redundant RSM, and to quickly compare them across RSMs. The utility scans the current or specified domain for any redundant RSMs, retrieves each redundancy definition, gets all the RSMs in those definitions, and displays each in a separate column.

The RSM Diagnostic Tool (RsmDiags.exe) is stored in the CygNet\Utilities folder on the host server. Click the following image to see an example RSM Diagnostic Tool:

Redundancy Definitions

The grid has four fixed rows at the top: Zones, Domains, Networks, and Triggers. Each cell provides a count of the number of definitions for each property and is color coded to indicate if they match. You can open the Redundancy Editor by double-clicking any cell in these rows.

Services and Owners

The remainder of the grid lists the services and owner(s) for each RSM. The row header displays the domain site.service (for example, [5410]SITE.SERVICE) and the cell value in each column displays the owning RSM (for example, [5411]SITE.RSM). If the service and owner doesn't match between the various RSMs, the inconsistency is color coded.

Redundancy Monitor Properties

The following table lists and describes the RSM Diagnostics Tool configuration options.

Parameter Description

RSM service

Specify the domain and RSM of the main service. Optional.

RSM filter

Specify a domain and RSM to limit the services to a single owner.
Load button Click to load all services in the redundancy definition.
Refresh button Click to refresh all services in the redundancy definition.

Update automatically

Select to automatically update the grid at the defined refresh rate.
Refresh rate Specify the refresh frequency in seconds.

Last refresh

Indicates the last time the grid was refreshed.

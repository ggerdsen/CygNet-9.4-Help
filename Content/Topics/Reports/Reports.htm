Reports

CygNet Software provides the following options for accessing, exporting, and formatting of CygNet data into reports.

  • CygNet Excel Export — CygNet Excel Export (CxExport.dll or CxExport64.dll) can be used to create reports in a variety of applications, including Microsoft Excel, VBA, C++, Microsoft Word. A sample use of the .dll with Excel is included, which can be used to generate a History Values Report, a Current Values Report, and a Current Values by Group Report directly with Microsoft Excel.
  • CygNet Measurement Reports — Run and email a variety of industry-standard CygNet Measurement Reports.
  • CygNet Reports — Export CygNet data using CygNet Reports. Reports include Alarm Activity, Communications Efficiency, GNS LOG, Morning Report, and Daily Summary.
  • Enterprise Operations — Enterprise Objects provides an interface between an Enterprise Data model and the CygNet Native (SCADA) Data model. This interface facilitates the exchange of data between the CygNet Software platform, third-party applications, and other business systems.
  • ODBC — Data can be retrieved from CygNet using the CygNet ODBC driver for use with an ODBC application such as Microsoft Access, Microsoft Excel, or Crystal Reports. The CygNet application that uses ODBC to retrieve data is CygNet Reports.
  • Export Tools — In addition to the reporting options listed above, CygNet Software provides many other export options for getting data and configuration settings out of the system. These tools, utilities, and applications are listed here.

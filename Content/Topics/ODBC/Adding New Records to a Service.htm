Adding New Records to a Service

Do not use ODBC to add new records to a service.

Use the following utilities to add various record types.

  • To add points to the system - Copy existing points using the Copy Points for Facility option in the PNT, or import points using the Point Configuration Manager utility or the Text Import EIE.
  • To import devices - Use the DDS Import/Export option, or copy existing devices singularly or in bulk.
  • To add data groups or commands to devices - Use the utilities CygNet DDS Data Group Copy and CygNet DDS Command Copy, respectively.
  • To add facilities or facility attribute values - Use the FacLoad utility.
  • To add records to the Table Reference Service - Use the utility TRSLoad.

Available Point Properties

Some point properties are available through the CygNet OPC HDA Server. These are either historized, and therefore available to be accessed based on a time range, or current values. The available properties are listed in the tables below.

Historized Properties

OPC Property Description
POINTSTATEDESC Point State Description
POINTSTATENAME Point State Name
STATUS CygNet status of the point

TIMESTAMP

Item Timestamp
USERSTATUS User-defined status

VALUE

Item Value

Current Value Properties

OPC Property Description
COMMENTS The point’s comments (PNT property)
DESCRIPTION Point Description
FACILITY The point’s facility (PNT property)

FULLTAG

 CygNet Full Tag (SITE.SERVICE.POINTID:LONGID)
LONGPOINTID Long Point ID
LONGTAG CygNet Long Tag (SITE.SERVICE:LONGID)
POINTID Point ID
SCANRATE Server Scan Rate
SERVICE The point’s current value service
SITE The point’s site
SITESERVICE The point’s site and current value service (MYSITE.UIS)
SYSTEM The point’s system (PNT property)
TAG CygNet Tag (SITE.SERVICE.POINTID)
UDC The point’s UDC (PNT property)
UNITS Engineering Units
VALUETYPE Item Canonical Data Type

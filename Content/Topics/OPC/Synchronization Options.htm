Synchronization Options

OPCIS data points can be configured with a synchronization option. Two OPCIS point options are OPC Auto Sync Config and OPC Manual Config. These properties are applicable only if the point was created using the utility OPCSync.

  • OPC Auto Sync Config enables the OPCIS to change configuration settings based on the OPC server settings of the point.
  • OPC Manual Configure disables the auto configuration.

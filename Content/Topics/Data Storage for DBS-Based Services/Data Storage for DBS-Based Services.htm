Data Storage for DBS-Based Services

This section describes the data storage technology used for DBS-based services in CygNet Software.

DBS-Based Services

A DBS-based service is a generic term for a CygNet service that stores data used by other services. All DBS-based services use the same database engine, and have similar configuration parameters, security administration, backup and restoration settings, recovery settings, and maintenance practices and recommendations. See DBS-Based Services for more information.

The following CygNet services are DBS-based services:

  • Access Control Service (ACS)
  • Application Blob Service (APPS)
  • Audit Service (AUD)
  • Blob Storage Service (BSS)
  • Database Service (DBS) Note: Custom service only. See Generic DBS Service for more information.
  • Device Definition Service (DDS)
  • Event Logging Service (ELS)
  • Alarm Event Logging Service (ELSALM)
  • Facility Service (FAC)
  • General Notification Service (GNS)
  • Group Service (GRP)
  • Master Scheduling Service (MSS)
  • Note Service (NOTE)
  • Point Service (PNT)
  • Table Reference Service (TRS)

Utilities for DBS-Based Services

CygNet provides a set of utilities for managing DBS-based services. See Database Utilities for more information.

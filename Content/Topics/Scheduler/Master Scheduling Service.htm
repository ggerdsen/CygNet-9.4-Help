Master Scheduling Service

The Master Scheduling Service (MSS) is the CygNet scheduler. It can be used to schedule UIS command tasks, FMS command tasks, set info keyword tasks, and setpoint tasks. Tasks can be scheduled to be recurring, for a duration, based on a condition, or continuous. A task can be applied to a single device or multiple devices. Blackout periods can be applied to tasks.

Context Menu Options

The following table lists and describes the context menu options that display when you right-click in the Note service pane.

Option Description
Tasks Tab
New

Click this option to create a new tasks. There are four options:

  • UIS Command Task
  • FMS Command Task
  • Set Info Keyword Task
  • Set Point Task
Delete Tasks Highlight an existing record and click this option to permanently delete a task.
Properties Highlight an existing record and click this option to view its properties in read mode. You can also just double-click an existing record.
Execute Now

Select this option to execute an MSS task immediately. For one instance only, Execute Now issues the selected task to its respective service. The task's priority determines when the task will initiate an actual poll. After the task executes, polling returns to the task's scheduled routine.

Note: You must either have administrative access to the service or have "update" level access to the security event EXECNOW to run this task.
Activate

Select this option to activate a highlighted task without opening its property sheet.
Deactivate

Select this option to deactivate a highlighted task without opening its property sheet.
View Blackouts

Select this option to view the Blackouts page of the MSS.
Blackouts Tab
New Blackout

Click this option to create a new blackout period.
Delete Blackout Highlight an existing record and click this option to permanently delete a blackout.
Properties Highlight an existing record and click this option to view its properties in read mode. You can also just double-click an existing record.
Activate

Select this option to activate a highlighted blackout without opening its property sheet.
Deactivate

Select this option to deactivate a highlighted blackout without opening its property sheet.
View Tasks

Select this option to view the Tasks page of the MSS.

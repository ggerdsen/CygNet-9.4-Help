Set Info Keyword Task

A set info keyword task sends an info keyword to a service to trigger the service to perform an action. Currently, the set info keyword task can be used to perform service backups at a scheduled interval or to trigger the VHS to create a new journal file.

For more information, see Scheduling a Backup with the MSS, Service Backups on Demand, and/or VHS Journal Files.

Set Info Keyword Task Properties

The following table lists and describes properties of the Set Info Keyword Task property sheet.

Parameter Description

Service

Name of the service to which the Info Keyword value applies.
Load Keywords Click to refresh the list of keywords for the specified service.

Info Keyword

Specifies the info keyword to send to the specified service. Options are as follows:

  • BACKUP_NOW — Triggers the creation of new service backups.
  • VHS_START_JOURNAL — Triggers the creation of new VHS journal files.
Incremental Select to schedule an incremental backup, which writes transaction log files to the back-up directory and stores them there until a full backup is performed.

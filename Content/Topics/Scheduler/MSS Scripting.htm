MSS Scripting

The following scripting options are available for data interface functions related to scheduling and the Master Scheduling Service.

COM API Description and Link

CxMssBlackoutEditor

The CxMssBlackoutEditor object is a dialog box-based editor for Blackout records of the Master Scheduling Service (MSS).

See CxEditors.CxMssBlackoutEditor Object for more information.

CxMssCmdEditor

The CxMssCmdEditor object is a dialog box-based editor for Command records of the Master Scheduling Service (MSS).

See CxEditors.CxMssCmdEditor Object for more information.

CxMssInfoKeywordEditor

The CxMssInfoKeywordEditor object is a dialog box-based editor for Info Keyword Command records of the Master Scheduling Service (MSS).

See CxEditors.CxMssInfoKeywordEditor Object for more information.

CxMssSetPointEditor

The CxMssSetPointEditor object is a dialog box-based editor for SetPoint Command records of the Master Scheduling Service (MSS).

See CxEditors.CxMssSetPointEditor Object for more information.

See Scripting for general information about scripting in CygNet Software.

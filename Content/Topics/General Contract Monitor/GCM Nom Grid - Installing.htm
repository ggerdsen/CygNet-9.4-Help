Installing a CygNet GCM Nomination Grid

Perform the following steps to install the CygNet GCM Nomination Grid:

To Install a CygNet GCM Nomination Grid

  1. Copy the ActiveX file (CxHmiGcm.ocx) to the CygNet\Support folder on the host server.
  2. Register the file in one of two ways:
  1. Register the file using CVersionInfo.exe. It is located in the CygNet\Utilities folder.
  2. Register the file using regsvr32.exe (located in the Windows\System32 folder, or use Windows\SysWOW64\regsvr32.exe on a 64-bit operating system).
  1. Using CygNet Explorer, upload the file (CxHmiGcm.ocx) to the SUPPORT group in the Application Blob Service (APPS).
  2. Download the StudSupp.rsp file from the SUPPORT group in APPS.
  3. Copy the following line to the "CygNet ActiveX and Automation Files" section of the StudSupp.rsp file:

[SUPPORT]..\SUPPORT\CxHmiGcm.ocx <REGISTER>
  1. Upload the edited StudSupp.rsp to the SUPPORT group in APPS.
  2. Use the TouchBSS utility found in the CygNet\Utilities folder to update the timestamp for all .rsp and .rsq files in APPS.
  3. Close and reopen CygNet Explorer to perform version management.

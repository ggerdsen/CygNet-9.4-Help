Command Parameters

Command parameters limit the value of a calculation for a UIS command.

Currently, command parameters only exist for the Estimate Hourly Accumulation calculation, where default parameters are used to delimit a specified time frame.

For related information, see Configuring UIS Command Properties.

To Edit Command Parameters

  1. Open the Properties dialog box for the GcmCalc device you want to alter.
  2. Click UIS Commands and either double-click an existing UIS command or click in the white space of the UIS Commands field and click Add.
  3. On the UIS Command Properties dialog box, either double-click an existing Calculation or if you want to add a new calculation click Add.
  4. On the Edit Command Component dialog box, open the Calculation drop-down menu and view an existing calculation or select a new one. Available parameters appear in the Parameters table.

    5. Note: At this time, the only calculation that displays parameters is Estimate Hourly Accumulation. Parameters include MaxDaysBack and MinDaysBack.

  1. In the Parameters table, highlight and double-click a parameter to add a new value.
  2. On the Modify Parm dialog box, enter a new value in the New Value field and click OK. The value must be a wildcard-based date; absolute values will result in a failed calculation.

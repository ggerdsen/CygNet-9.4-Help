Configuring a CygNet GCM Nomination Grid

The CygNet GCM Nomination Grid is an ActiveX control.

Before mapping GCM data from the DDS to the CygNet GCM Nomination Grid for display in a Studio screen, you must configure the control's basic properties on the Property Sheet in Edit mode in CygNet Studio. This involves configuring properties under the Normal, Events, and ActiveX pages on the Property Sheet.

To Configure a CygNet GCM Nomination Grid

  1. Select from the Standard Tools panel by clicking it once and then clicking once on the View, or drag and drop the onto the View.
  2. On the Insert OLE Control dialog box, scroll down to CygNet GCM Nomination Grid and double-click it. This places a CygNet GCM Nomination Grid object on the View.
  3. Right-click on the object to open its Property Sheet.
  4. Configure the Normal, Events, and ActiveX properties according to your specifications.
  5. Click the Active X property page and double-click (OCXProperties). Advanced configuration of the application is done by clicking General, Filters, Display, Columns, and Rows and configuring each set of properties to match the properties configured on the GCM device editor.

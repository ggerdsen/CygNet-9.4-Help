Configuring Filters

Use the Filters page to configure the filters that define the location of each input within a hierarchy.

The control begins at a root node and travels to each node in the hierarchy that meets the navigation node filter rules. Navigation nodes include meter station and meter run nodes. Branches without a meter station node are ignored. You can create custom filters to further define special locations in the hierarchy where inputs can be populated.

The fields configured on this page match fields on the Group Filters page of the GCM device editor. See Using CygNet Filter Rules for a general discussion on using CygNet filter rules.

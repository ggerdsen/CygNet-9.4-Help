System UDCs

Each GCM device has a set of status points that are automatically generated when the device is created. The values in these points always pertain to the most recent execution. To see a history of the results, enable trending on the points.

UDC UDC Description Description Point Type
SYEDFOP Failed calcs The total number of failed calculations. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYEDIPOP

 Pending calcs

The number of calculations pending or in progress. Time period suffixes (CD, CH, CM, LD, LH) apply.

 Analog Input
SYEDSOP Successful calcs The number of successful calculations. Time period suffixes (CD, CH, CM, LD, LH) apply. Analog Input

SYLPBRT

Last op hier bld tm

Total runtime (in seconds) of the hierarchy build time of last operation. Time period suffixes (CD, CH, CM, LD, LH) apply.

Analog Input

SYLPERR

 Last op error count

The number of errors during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply.

 Analog Input

SYLPRT

 Last op runtime

Total runtime (in seconds) of the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply.

 Analog Input

SYLPWARN

 Last op warning count

The number of warnings during the last operation. Time period suffixes (CD, CH, CM, LD, LH) apply.

 Analog Input

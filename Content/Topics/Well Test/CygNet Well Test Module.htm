CygNet Well Test Module

The CygNet Well Test Module:

  • leverages the CygNet infrastructure stack: UIS, DDS, PNT, MSS, OPCIS, and VHS
  • accesses building blocks for replicating device logic (defined in the XML configuration)
  • uses triggers (manual or scheduled) to affect the well test process (Canvas HMI Screens, MSS)
  • interfaces with external systems such as ForeSite to retrieve the well test records via the CygNet.API.WellTest

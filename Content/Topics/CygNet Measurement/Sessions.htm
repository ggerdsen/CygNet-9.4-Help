Session

The concept of a Session operates in concert with the Node organizational structure in CygNet Measurement to provide an effective and expedient approach to measurement data analysis. Use the Session settings to define a set of parameters that collectively describe a working session. Nodes of your choice can then be set to simultaneously use those Session settings to provide a consistent view of desired measurement data. Use of the Session parameters is optional, and can be selected and cleared at will to help you access appropriate data views.

By defining a Session, you can switch back and forth between controls using the same settings, without resetting your options. You can also use a defined Session to quickly switch out of using the Session settings, to view data for other parameters, and then quickly switch back to the Session settings again.

Defining the Session

A Session in CygNet Measurement is defined by configuring a Session Node and Session options. Session options consist of a Time Window and Time Format. These parameters are configured on the Session menu. Once settings are configured, the Session menu will display the name of the selected Session Node, the time ranges and shift/expand increments for the selected Time Window, and the selected Time Format in which to view data. Current Session parameters can therefore be verified at any time by viewing the Session menu.

See Using the Session Menu for more information.

Using the Session Settings

Session settings can be selected for use in association with controls of your choice. Open desired controls from the Controls menu in FMS Explorer, and select the Options pane to access options to use Session settings. Options include Use Session Node and Use Session options.

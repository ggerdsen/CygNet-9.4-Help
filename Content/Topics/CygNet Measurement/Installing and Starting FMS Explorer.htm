Installing and Starting FMS Explorer

Use the CygNet Client Installer (CygNet\Clients\CInstall.exe) to install the CygNet FMS Explorer client application. The CygNet Client Installer sets up integrated version management to ensure that you have the latest version of the application files. See CygNet Client Installer for more information.

Installing FMS Explorer

To Install and Start CygNet FMS Explorer Using the CygNet Client Installer

  1. Open the CygNet Client Installer (Start Menu > All Programs > CygNet > CygNet Client Installer). The CygNet Client Installer/Launcher dialog box will be displayed.

  2. On the BLOB Service: Admin drop-down menu, select your APPS Site.Service.
  3. In the BLOB Folder: list, select Clients to access the list of client applications available for installation.
  4. In the Select CygNet Applications to Install list, select CygNet FMS Explorer.
  5. Click the check box adjacent to Add shortcut to desktop to select/clear this feature. Selecting this feature ensures that you have version management settings.
  6. Click Install. A confirmation message will be displayed asking if you want to start the application.
  7. Select Yes to start FMS Explorer.

Starting FMS Explorer

Once FMS Explorer has been installed using the CygNet Client Installer, you can open the application using either of the following methods. Both of these methods provide version management for the FMS Explorer application.

  • Start the application using the CygNet Client Installer. Follow the installation procedure above to step 6 and then click Launch.
  • Start the application using the Start menu shortcut. From the Start menu, browse to Programs > CygNet > CygNet FMS Explorer and click the application icon .

Alternatively, you can open the application using either of the following methods. These methods do not provide version management.

  • Start the application using Windows Explorer.

    Browse to the CygNet\Clients\FMSEXPLORE directory and double-click the FMS Explorer executable (FmsExplore.exe).

  • Start the application from a command line.

    In the command window, specify the location of the FMS Explorer application, C:\CygNet\Clients\FMSEXPLORE\FmsExplore.exe.

    Command-line Parameters

    Available command-line parameter options are described in the following table.

    ParameterDescription

    /Domain:nnnn

    To run the application on another domain, specify the domain number using the parameter /Domain:nnnn where nnnn is the port number of the desired domain. Acceptable values are 5001 to 32767.

    /noload

    To override the workspace file auto-load feature, include the parameter /noload.

    Note: There are consequences to using this parameter; use only if necessary. See Troubleshooting CygNet Measurement for more information.

FMS Toolbox Utility

CygNet FMS Toolbox is a CygNet Software utility provided to streamline the process of performing various device and data processes interacting with the CygNet Measurement (FMS) service. Use the utility to migrate files from GMR to FMS, import device and group-level data and process variables from within or outside of the CygNet Measurement (FMS) system into FMS, and manage device-level data.

Each tool contained in the FMS Toolbox contains component tasks that can be completed in sections or at different times, without repeating previously completed portions of the process. FMS Toolbox tasks utilize comma-separated value (.csv) files.

The FMS Toolbox utility file (FmsToolbox.exe) is stored in the CygNet\Utilities directory on your host server.

You must have proper security authorization to perform various aspects of file migrations. Required security authorization levels are noted in each of the applicable task steps.

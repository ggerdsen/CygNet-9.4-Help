Accessing Help

Use the following Help options to access current information related to your FMS Explorer client application.

Option Description
Support Click Help to access the Help.
Options

Click Settings to access the Client Settings dialog box.

Either view the current settings, or choose to do one of the following.

  • Remove any custom settings and click Restore to Default to apply the change. Click Close.
  • Select new settings and click Use New Settings to apply the change(s). Click Close.
About FMS Explorer

View About FMS Explorer to see information about your FMS Explorer client application.

Support

Click the Help button to access the CygNet Help.

Options

Click Settings to access the Client Settings dialog box. Network Traffic information, Application Log Settings, and Network Log Settings information can be viewed, restored to default settings, or edited on this dialog box. See Client Settings for more information.

About FMS Explorer

View the following information about your FMS Explorer client application from the Help page.

  • The File Version of the application executable you are running
  • The Product Version of CygNet Measurement you are running
  • The File location of the application's file executable
  • The Log file location, including a hyperlink to the log file
  • CygNet Software copyright information

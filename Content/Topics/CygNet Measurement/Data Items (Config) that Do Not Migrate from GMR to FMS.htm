Configuration Data Items that Do Not Migrate from GMR to FMS

When you use FMS Toolbox to migrate GMR events, alarms, and device configuration information to FMS using the Migrate Events task in the GMR Migration tool, some items may not migrate into FMS. These items will be ignored in the import file because they do not apply in FMS.

Select Verbose status messages before you click Migrate, to generate messages in the migration results text box if this occurs. The migration results would then list the migration status as "ignored" and explain that the item is GMR specific.

CygNet FMS History Graph Control

The CygNet FMS History Graph Control is an ActiveX control used to view liquid (periodic) or gas metering/gas quality history graphically for multiple selected Nodes, in order to get a visual overview of the data and to facilitate the identification of trends and features on a larger scale. The History Graph can display data for up to 16 process variables at one time. Data cannot be edited with this control.

See Using the History Graph Control for more information about configuring and using this control.

See CxFmsHistoryGraphCtrl Object in the Scripting section for more information about scripting this control.

Adding a History Graph Control to a Screen

To Add and Configure a CygNet FMS History Graph Control in CygNet Studio

  1. Create a new CygNet Studio screen.
  2. Click the ActiveX tool on the Standard Tools palette.
  3. Click and drag the outline of a box on the screen. When the mouse button is released, an Insert OLE Control dialog box appears.
  4. In the dialog box, double-click on CygNet FMS History Graph Control to place a history Graph control on your screen.
  5. Right-click the control to open its property sheet.
  6. On the ActiveX page, click ... or double-click (OCXProperties) to access the CygNet FMS History Graph Control Properties dialog box.
  7. The CygNet FMS History Graph control contains a General tab, a Graph tab, a Colors tab, and a Commands tab.
  8. Configure properties on each page as desired, click Apply, and then click OK.
  9. Close the property sheet, save, and then switch to Run mode to use the control.

Selecting and Modifying Process Variables for Display

Select up to 16 process variables for display in the graph. Attempting to add more than the maximum number of process variables produces an error message. When a selected process variable has no data for the entire time window selected for the control, it will not appear on the graph or in the Value Table.

To Configure History Graph Process Variables for Display

The list of available process variables is subdivided into gas and liquid items for ease of selection. Modify the list as follows.

  1. On the Options pane, click Modify to access the Process Variables dialog box to select the process variables to display in the graph.

  1. Select the items to display.
    • Click a process variable name from the list of available items (left pane), and then click >> and << to edit the list of process variables to display graphically (right pane).
    • To select multiple PVs at one time, use Ctrl+Click or Shift+Click.
  2. Select the order of the items in the display.
    • The order of items in the Process Variables list will be the order of display in the graph.
    • To change the order, first click an item to select it in the list and then click Move up / Move down until the desired order is achieved. Each click moves the selected item one position further up/down the list.
  3. Save your selections.
    • Once the list contains the desired items in the desired order, click Save to view the results in the process variables list shown on the Options pane.

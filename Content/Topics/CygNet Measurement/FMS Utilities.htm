FMS Utilities

 CygNet Software provides utilities to help you streamline the accomplishment of various processes with the FMS service, and to manage the FMS internal database. All FMS utility files are located in the CygNet\Utilities directory on your host server.

FMS Toolbox

FmsToolbox.exe

 GMR Migration Migrates devices, groups, events, alarms, and/or configuration data from a GMR service to an FMS service

Import Creates templates and imports devices from another source into an FMS database
Device Data Updates multiple FMS Node device configuration values

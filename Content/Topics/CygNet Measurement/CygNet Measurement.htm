CygNet Measurement

CygNet Measurement provides timely retrieval, storage, validation, estimation, analysis, modification, and delivery of accurate electronic flow measurement data. CygNet Dispatch is an optionally available companion application to CygNet Measurement, used to provide job information from the field that could potentially adjust FMS data records. See CygNet Dispatch for more information.

CygNet Measurement Application

Run CygNet Measurement on CygNet Software v8.5.1 or greater. See the CygNet Release Documents for more information about current software version requirements and considerations appropriate for your installation and usage.

CygNet Measurement and its calculations comply with the following industry standards, and meet calculation method requirements as noted.

  • American Petroleum Institute (API) standards for gas measurement as published in Chapter 21.1
  • Bureau of Land Management General Order Number 5
  • American Gas Association (AGA) algorithms for volumetric gas flow (AGA 3 for 1985/1992/2013, AGA 7, AGA 11), and compressibility (AGA 8, NX-19)
  • Gas Processors Association (GPA) standards for heating value, relative density, and hydrocarbon liquid content as GPM (GPA 2172-09) and referencing the standard table of physical properties (GPA 2145-16)

See CygNet Measurement Data Calculation for more information.

The CygNet Measurement system requires licensing, and can be run in one of two different licensed modes, offering either complete (FULL) or limited (REPOSITORY) feature sets. See CygNet Measurement Licensing Requirements for more information.

Note: If your system operates in REPOSITORY mode, unavailable features do not appear, and some options appear as unavailable (dimmed). See Operational Mode in the CygNet Measurement Overview topic for more information.

Warning: Run trial versions of CygNet Measurement in a test environment only. Running your system in a full feature set mode precludes the possibility of running it in a limited feature set mode in the future. You cannot go back to a limited feature set CygNet Measurement system from a full feature set, due to database implications.

CygNet Dispatch

CygNet Dispatch is an optionally available, separately licensed tool that adds functionality to integrate the results of field technician activities into your CygNet Measurement system. This additional feature set includes job scheduling, tracking, and reporting functionality for calibration and inspection tasks for gas meters and chromatographs. See CygNet Dispatch for more information.

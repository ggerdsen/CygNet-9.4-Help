Creating CygNet Measurement Nodes

A Node is the basic element used within CygNet Measurement to functionally organize your data. You can create device-level Nodes and group-level Nodes; each Node will be defined by its own set of properties.

You must have proper security authorization to create Nodes in CygNet Measurement. See FMS Security (and NODE security event) for information about configuring security access for Node creation.

See Nodes in the CygNet Measurement Concepts section for more information about Nodes.

Node Types

The following Node types can be created in CygNet Measurement.

Device Nodes:

 

Gas Meter Device Node
  Station Meter Node (a Gas Meter Node set to also function as a station)
 

Chromatograph Device Node
 

Liquid Device Node
 

Accessory Node

Group Nodes:

 

General Group Node

 

Station Group Node

Node Creation Methods

FMS Nodes can be created manually or automatically.

  • Manually via FMS Explorer - Create FMS Nodes via the Nodes menu in FMS Explorer.

    See Using the Nodes Menu for more information.

  • Automatically via FMS commands - Create FMS Nodes via two system commands that can automatically create FMS device or group Nodes in your CygNet Measurement system.

    See Using the Commands menu and the following topics for more information about automatically creating FMS Nodes.

  • Detect New Devices
  • Generate Smart Groups
  • Automatically via FMS Toolbox - Create FMS Nodes when migrating or importing files into the CygNet Measurement system, using tools provided in the FMS Toolbox utility.

    See FMS Toolbox Utility and the following topics for more information about automatically creating FMS Nodes.

  • Migrating Files from GMR into FMS
  • Importing Files into FMS

Viewing Audit History

FMS Explorer provides a detailed audit history trail for several externally audited FMS records, providing insight into the user workflow process for the selected record.

The Audit History dialog box can be accessed via the Audit History button found in the following locations.

Click the Audit History button to access the Audit History dialog box for the item selected.

The following fields appear in the Audit History dialog box. In addition, the bottom left portion of the dialog box status bar displays the audit history results (e.g. "Got 3 changes" in the example above).

Field Description

Refresh Audit Trail

Click to refresh the audit trail for the selected FMS record.

[Audit History]

Expand/contract the record entries to view/hide audit trail details for the FMS Node and time window selected.

Each entry will display the following information on the first line.

  • Date of the record, in Device time (D)
  • Time of the record, in Device time (D)
  • Name of the Site and User that performed the action, enclosed in parentheses (Site\User)

Each entry will display the following information on additional lines

  • User action completed, as applicable - expand/contract the entries to view desired audit trail details, e.g. Add FMS record, Update FMS record, Delete FMS record, etc.
  • For each user action, the New Value

New Value

Displays the new value for the record that has changed.

Close

Click to close the dialog box.

Using the Commands Menu

Use the Commands menu to access the following options to customize FMS commands for quick execution. Run and manage Node commands, run system commands, configure extended options, and view command logs from this menu.

See FMS Commands for a complete list of FMS commands and access details, as not all commands are executed via this menu.

See FMS Security (CMDDEV and CMSSYS security events) for information about configuring security access for executing FMS commands.

See Scheduling FMS Command Tasks in the MSS for more information about scheduling commands.

Commands Menu

The FMS Explorer commands menu is arranged in categories to help you quickly perform command-related tasks and run a variety of commands against a selected Session Node. See Session Node for more information.

Node Commands

Run and manage Node data commands (Request... or Send... commands, or optional installed functionality) from this menu, using the following buttons.

  • Run - access and execute a command saved in your system, or a <Quick Command>, against the Session Node
  • Manage - access options to define, edit and delete FMS data commands

System Commands

Run and manage CygNet Measurement system commands of defined types and parameters, using the following buttons.

  • File Import - access and execute a variety of import commands installed in your system
  • File Export - access and execute a variety of export commands available in your system
  • Normalization Views - update normalization views saved in your system
  • Station Estimation - estimate station data using estimation engines associated with each Node
  • Smart Groups - generate smart groups based on configured smart rules and conditions
  • Detect New - detect new devices in your system and auto-generate FMS Nodes based on user-defined parameters
  • Database Maintenance - perform database maintenance tasks

In addition, you can use the following button to access options to customize and support system commands that require additional configuration.

  • Extended Options - access these options to configure additional system command parameters
  • Smart Group Configuration - define smart group definitions and rules for creating new group Nodes, that are used by the Generate Smart Groups command
  • Detect New Devices - configure criteria and parameters for creating new device Nodes, that are used by the Detect New Devices command
  • Database Maintenance - configure log and command retention periods, that are used by the Perform Database Maintenance command

Logs

Use the View button to access views of historical command logs for selected FMS commands, and of pertinent system information, within a specified time frame.

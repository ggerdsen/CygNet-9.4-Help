CygNet Measurement Licensing Requirements

CygNet Measurement must be licensed in addition to existing SCADA components, and is licensed according to the quantity of Measurement devices in your system. This can be done by means of a free trial license or a standard purchased license, acquired either at the time of purchasing CygNet SCADA or after purchasing CygNet SCADA.

CygNet Measurement License File

If you are adding CygNet Measurement to an existing installation, you are shipped a new CygNet.lic file upon closing your purchase of the new products. The new license file is to be placed in your master CygNet\Services\ARS directory, to replace your existing CygNet.lic file, and includes all of your existing licensing information along with the new use permissions for the CygNet Measurement products.

Warning: Run trial versions of CygNet Measurement in a test environment only. Running your system in a full feature set mode precludes the possibility of running it in a limited feature set mode in the future. You cannot go back to a limited feature set CygNet Measurement system from a full feature set, due to database implications.

You can run CygNet Measurement with either a FULL or REPOSITORY mode feature set, corresponding to the product license type purchased. The mode is set by keywords in your FMS configuration file (CygNet\Services\FMS\fms.cfg), and determines which features your system will be able to access.

See CygNet Measurement Overview for more information about FULL and REPOSITORY mode features.

See License Management for more information about how to properly license your new products.

Contact your Account Manager or CygNet Sales for additional licensing information.

CygNet Dispatch

CygNet Dispatch is an optionally available, separately licensed, tool that augments CygNet Measurement usage by providing integrated job scheduling, tracking, and reporting functionality for calibration and inspection tasks. For more information about the CygNet Dispatch product, contact your Account Manager or CygNet Sales.

Using FMS Toolbox

Use the CygNet FMS Toolbox utility to migrate, import, or manage device, group, and Node level data files used for managing your CygNet Measurement system. FMS Toolbox contains three primary tools, each of which is comprised of component tasks that can be completed incrementally. The following tools are provided in the FMS Toolbox.

  • GMR Migration
  • Import
  • Device Data

You must have proper security authorization to perform various aspects of file migrations used in this utility. Required security authorization levels are noted in each of the applicable tasks.

Component Tasks

The following is an overview of the component tasks that can be performed utilizing each tool provided in the FMS Toolbox.

GMR Migration

The following tasks help you migrate files from GMR into CygNet Measurement using the GMR Migration tool from the CygNet FMS Toolbox.

  • Export Devices - Use the utility to export GMR device data and process variables from GMR to .csv files.
  • Export Groups - Use the utility to export GMR group data from GMR to .csv files.
  • Configure and edit device .csv files. - Prepare exported device .csv files for import.
  • Configure and edit group .csv files. - Prepare exported group .csv files for import.
  • Import Devices - Use the utility to import device .csv files into FMS.
  • Import Groups - Use the utility to import group definition and/or group entries .csv files into FMS.
  • Migrate Events - Use the utility to migrate events, alarms, and configuration information from GMR to FMS.
  • Migrate Contract Hour - Use the utility to migrate contract hour information from GMR to FMS.
  • Migrate History - Use the utility to migrate history information from GMR to FMS.
  • Migrate Data Groups - Use the utility to migrate data group mappings from GMR to FMS.

See Migrating Files from GMR into FMS for more information.

Import

The following tasks help you import and/or export FMS .csv files using the Import tool from the CygNet FMS Toolbox.

  • Create Templates - Use the utility to create .csv template files containing the necessary file header information for your import files.
  • Export Devices - Use the utility to export devices from FMS to a device definition .csv file for editing.
  • Export Groups - Use the utility to export groups from FMS to a device definition .csv file for editing.
  • Configure and edit device .csv files. - Prepare exported device .csv files or templates for import.
  • Configure and edit group .csv files. - Prepare group .csv file templates for import.
  • Import Devices - Use the utility to import device .csv files into FMS.
  • Import Groups - Use the utility to import group definition and/or group entries .csv files into FMS.

See Importing Files into FMS for more information.

Device Data

The following task helps you make modifications to FMS Nodes using the Device Data tool from the CygNet FMS Toolbox.

  • Update Configuration - Use the utility to make multiple configuration changes to FMS Nodes all at once.
  • Update Frozen Configuration - Use the utility to freeze or unfreeze configuration items for multiple FMS Nodes.

See Managing FMS Device Data for more information.

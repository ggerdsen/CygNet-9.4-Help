FMS Explorer

FMS Explorer is the primary user interface for the CygNet Measurement application.

The user interface contains a number of menus to access both administrative and user functions for managing and handling measurement data.

The FMS Explorer executable is stored in the CygNet\Clients\FMSEXPLORE directory (FmsExplore.exe) on your host server.

On-Demand or Scheduled Requests for Data

The FMS Explorer client supports only on-demand requests for electronic flow measurement (EFM) data retrieval.

Scheduled requests for EFM data retrieval for CygNet Measurement can be set up by scheduling FMS command tasks in the Master Scheduling Service (MSS). The MSS is part of the CygNet Software platform, and is accessed through CygNet Explorer. See CygNet Explorer and Scheduling FMS Command Tasks in the MSS for more information.

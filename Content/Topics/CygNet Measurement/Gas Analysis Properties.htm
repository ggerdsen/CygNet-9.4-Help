Gas Analysis Properties

Gas analysis item properties for each entry in the Gas analysis data view can be accessed via the context menu. You can then view or edit records using a series of tabbed property pages.

You must have proper security authorization to edit gas analysis entries. See FMS Security (and DEVCFG security event) for information about configuring security access.

Gas Analysis Property Pages

The following gas analysis properties are available when creating, editing, or viewing gas analysis entries.

  • Common Properties
  • Quality Property Page
  • Composition Property Page
  • Liquid Content Property Page
  • Custom Property Page
  • Sample Details Property Page

Use the Record details button to access contiguous records.

Record Details Button

When editing records, each property page provides access to a pair of navigation buttons (<<, >>) and a Record details button at the bottom of the page. You can use this feature to verify auditing information and, if there are multiple records displayed within the current time window shown in the control, to quickly access contiguous gas analysis records.

When there are multiple records available within the time window, use the << and >> navigation buttons to navigate between contiguous records, and find one for which you want to view more concise information about the gas analysis record change(s). As you navigate between records, the current record properties are reflected in the common properties area in the top pane of the dialog box.

Once the desired record is found, click Record details to access the Record Details dialog box where you can view details such as the date and time of the gas analysis record change, the ID of the user who made the change, and optionally the audit category assigned and/or any comments entered regarding the change.

See Viewing Record Details for more information.

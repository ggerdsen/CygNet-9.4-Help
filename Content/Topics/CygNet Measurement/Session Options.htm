Session Options

The concept of Session options works hand-in-hand with the Session Node to define a Session in CygNet Measurement. Usage of these organizational concepts facilitates effective and expedient management of measurement data. Selecting to use the Session options unifies the display of measurement data views, and synchronizes the selected controls to display data using the same time window and/or time format, so that you can quickly navigate between compatible data views while using multiple controls.

Session options are defined on the Session menu, and consist of a defined time window and a selected time format that you designate as the options set for the current working Session. Once these Session options are set, they are available for uniform application to all controls associated with the Session settings, across FMS Explorer. This is accomplished by selecting to Use Session options for multiple controls of your choice. You can share Session options in order to switch back and forth between controls using the same time window and time format settings, without resetting options each time. You can also quickly switch out of using the Session options, to view data for another time range or format, and then quickly switch back.

Setting Session Options

Defining a Time Window and/or Time Format and then setting them as the Session options will apply those settings to all controls that have selected to Use Session options. Once set, the Session options settings can be verified on the Session menu at any time.

The only location from which you can set Session options is the Session menu.

Set the Time Window and Time Format for the Session options in FMS Explorer, using the following method.

To Set the Session Options

On the Session menu, do the following to set the Session options.

  • Type the Session Time Window parameters directly into the Node text box. Press Enter.
  • Click the Session Time Format desired

See Using the Session Menu for more information.

Using Session Options

Most controls that allow you to select Nodes can be configured to run using the Session options (all except Close Period or Reports controls). Data for the time window and time format that are set as the Session options will then automatically be displayed in any FMS Explorer controls that have selected the Use Session options option, once you Refresh or Apply the control settings. This option can be selected or cleared at will, to allow easy navigation between data views using the Session options and data views for other time ranges of interest.

You can select the option to use the session options from several locations in the Controls menu (Summary, Data, or Admin controls). To enable use of the Session options for a selected control, select the Use Session options check box in the control's Options pane. When this option is selected, the embedded Node chooser for the control will be hidden. Then, once you initially Refresh or Apply the control settings, data displayed for the control will be automatically updated to the new Session options data whenever the Session options changes.

To Select Using the Session Options from Controls

On the Controls menu, there are several locations from which you can select/clear use of the Session options.

  • On the Options pane of a Data (Alarm, Configuration, History Graph, History Grid, and Raw Data) control, click to select/clear the check box to Use Session options. The Session options are selected by default for all Data controls. If you do not want to use the Session options, click the check box to clear it, and then enter the time window and time format information desired.
  • On the Options pane of a Summary (Dashboard, Exceptions) or Admin (Prior Period Adjustment, Balance; not applicable to Close Period) control, click to select/clear the check box to Use Session options. The Session options are cleared by default for Summary and Admin controls. Enter the time window and time format information desired, or click the check box to use the Session options.

When Use Session options is selected, the Start date, End date, and Time format fields on the Options pane for the control will be hidden. Then, once you initially Refresh or Apply the control settings, data displayed for the control will be automatically updated to the new Session options data view whenever the Session options change.

See Using the Controls Menu for more information.

Viewing and Editing CygNet Measurement SCADA Properties

A Node is the basic organizational element within CygNet Measurement and each FMS Node is associated with specified SCADA devices and facilities, depending upon how the Node is configured. SCADA device or facility properties dialog boxes can be accessed from FMS Explorer or CygNet Explorer.

SCADA properties can be viewed or edited by users with the required authorization level.

SCADA Properties

The following SCADA properties are applicable for each Node type.

 

Gas Meter Device Node - polling device, polling FAC
  Station Meter Node - polling device, polling FAC, reporting FAC
 

Chromatograph Device Node - polling device, polling FAC
 

Liquid Device Node - polling device, polling FAC
 

Accessory Node - N/A
 

General Group Node - N/A
 

Station Group Node - reporting FAC

Viewing and Editing SCADA Properties Using FMS Explorer

In FMS Explorer, the following SCADA properties options can be accessed from various locations in the application.

  • View polling device
  • View polling FAC
  • View reporting FAC

See Viewing and Editing SCADA Properties in FMS Explorer for more information.

Viewing and Editing SCADA Properties Using CygNet Explorer

Viewing SCADA Properties

In CygNet Explorer, the following SCADA properties options can be accessed through the Flow Measurement Service (FMS).

  • View polling device
  • View polling FAC
  • View reporting FAC

To View an FMS Node's SCADA Polling Device Details

  1. Open CygNet Explorer.
  2. In the service hierarchy for your site, click FMS (Flow Measurement Service) to display your FMS Nodes in the Contents pane.
  3. In the Contents pane, use the filters to narrow your Node selection as desired, and then click Retrieve Nodes to display a list of FMS Nodes meeting the filter criteria.
  4. Right-click on a Node name from the list to access the context menu, and then click View polling device to access the FMS Polling Device dialog box listing the data groups configured for the selected Node. See Remote Device Editors in the Remote Devices section for more information about the information contained.

To View an FMS Node's SCADA Polling Facility Details

  1. Open CygNet Explorer.
  2. In the service hierarchy for your site, click FMS (Flow Measurement Service) to display your FMS Nodes in the Contents pane.
  3. In the Contents pane, use the filters to narrow your Node selection as desired, and then click Retrieve Nodes to display a list of FMS Nodes meeting the filter criteria.
  4. Right-click on a Node name from the list to access the context menu, and then click View polling FAC to access the Properties dialog box listing the polling facility attributes configured for the selected Node.

To View an FMS Node's SCADA Reporting Facility Details

  1. Open CygNet Explorer.
  2. In the service hierarchy for your site, click FMS (Flow Measurement Service) to display your FMS Nodes in the Contents pane.
  3. In the Contents pane, use the filters to narrow your Node selection as desired, and then click Retrieve Nodes to display a list of FMS Nodes meeting the filter criteria.
  4. Right-click on a Node name from the list to access the context menu, and then click View reporting FAC to access the Properties dialog box listing the reporting facility attributes configured for the selected Node.

Editing SCADA Properties

You can edit the device or facility Properties in the dialog box that is displayed, if you have proper security authorization.

Note: Before editing any data groups, recognize that EFM (electronic flow measurement, e.g. FMS and FMS Legacy) data groups may have dependencies on non-EFM data groups. See EFM Data Items and the documents in the Remote Devices section for the specific EIE used by your device for more information about its EFM data groups, data group dependencies, and usage notes. Once you have verified the information, you can optionally perform edits to the device.

See the corresponding SCADA properties topic in the FMS Explorer section for more information.

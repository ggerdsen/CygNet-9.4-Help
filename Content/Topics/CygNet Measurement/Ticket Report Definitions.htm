Ticket Report Definitions

The following parameters apply to Ticket reports, when creating or editing report definitions through FMS Explorer.

See Ticket Report Templates for information about the referenced template files.

Parameter Value

Start date

Select the starting date of the time range to include in the report from the drop-down menu.

Options are as follows.

  • Use Session date
  • User override

If User override is selected, type the start date into the text box or click ... to Select Date.

Notes:

The report will display data in Device time (D), although the report command parameters are configured relative to Contract time (C).

The report will only include records that exist entirely within the specified date range.

See Selecting Date and Time for more information.

This parameter is required.

Note: This parameter will not appear if it has been specified in the report template.

End date

Select the ending date of the time range to include in the report from the drop-down menu.

Options are as follows.

  • Use Session date
  • User override

If User override is selected, type the end date into the text box or click ... to Select Date.

Notes:

The report will display data in Device time (D), although the report command parameters are configured relative to Contract time (C).

The report will only include records that exist entirely within the specified date range.

See Selecting Date and Time for more information.

This parameter is required.

Note: This parameter will not appear if it has been specified in the report template.

Unit set

Select a unit set in which to view the report data. Options appear in the drop-down menu, which is populated by the unit sets defined in the service.

See Configuring Unit Set Definitions for more information.

This parameter is required.

Note: This parameter will not appear if it has been specified in the report template.

Node name

Valid Node name values include Liquid Device Nodes. The report will display data for those with a history type of Ticket.

Select the Liquid Device Nodes for which to display ticket data in the report you are building.

Options are as follows.

  • Use Reports control Node - use the Node selected in the embedded Chooser of the reports control
  • User override - type the desired Node ID into the text box provided
  • Use Session Node - use the Node selected as the Session Node in FMS Explorer

This parameter is required.

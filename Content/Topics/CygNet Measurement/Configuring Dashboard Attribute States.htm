Configuring Dashboard Display Properties

Data displayed on the Dashboard can be customized using the Display editor. Edit color options in the display editor to customize the appearance of data shown in the grid, contributing to a clearer initial understanding of the data at a glance. You can configure the display of options for FG Color (foreground color), BG Color (background color), and selected categories of Cell Value, or use the default values provided.

If the selected property does not apply for a specific Node, the dashboard will display [NS] (not supported) in the record cells.

On the Options pane, click Configure colors to access the Display Properties dialog box.

Set cell color and text values as you want them to be displayed in the grid. Configurable properties available in the display editor include the following.

Configuring Display Properties

Configure the way you want various properties to appear in your Dashboard control. The settings will be saved to your workspace when you save the control.

To Configure Dashboard Display Properties

  1. In the Options pane, click Configure display to access the Display Properties dialog box.
  2. In the Display editor, click on a color field cell to edit FG Color or BG Color. See Configuring Color for more information.
  3. Edit applicable text value fields, as desired. Text in some cell value fields (analyst overview, collection status, configuration poll, edit states, and raw data export) can be customized. Click to select the Cell Value field, type in the cell value display desired, and then click OK to save the setting.
  4. Select which State to use when coloring Data Quality cell(s) from the drop-down menu. Options are for the color of the data quality cell, to reflect either the Lower or the Higher available value. The Lower value is shown by default.
  5. Click OK to save the settings.

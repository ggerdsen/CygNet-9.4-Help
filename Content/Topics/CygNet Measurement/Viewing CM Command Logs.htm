Viewing CygNet Measurement Command Logs

Commands in CygNet Measurement are executed by the FMS service, and a history of command actions is logged by the system.

You can access the command logs through FMS Explorer via the View button on the Commands menu. Use the View option to retrieve and view historical logs of selected FMS commands, and of certain pertinent user input, within a specified time frame. The user input actions are made available for viewing in conjunction with the command logs because they display useful related information regarding user submitted/deleted records.

See Viewing Command Logs in the FMS Explorer section for more information.

FMS Data Items Units

The following EFM (electronic flow measurement) data item units are supported by CygNet Measurement, and describe FMS data item units required to be specified in the system. Supported unit types for FMS unit sets are listed in the tables below. Some gas configuration items inherit their units from associated gas metering items and are listed in an additional table. Some values in the system may be slightly affected by a recent change to how pressure units are converted, as described here.

Changes to Pressure Unit Conversion

A recent modification was made to the CygNet unit conversion process to comply with API standards concerning the base temperature associated with pressure unit conversions. See Converting Pressure Units in the Points section for more information about this change.

This modification may affect certain unit conversions in FMS, for which the associated API recommendations apply. Affected devices are those that return differential pressure in units other than inches of water (inH2O), for conversions to/from inH2O pressure units from/to any other pressure units.

Depending on your licensed FMS mode and version, pressure unit conversions may affect FMS values in the following manner.

REPOSITORY Mode

In cases where raw data is being exported to another measurement system from a CygNet Measurement system in REPOSITORY mode, the pressure unit conversion could have the following impact.

  • For CygNet versions 8.1.2 or later, FMS maintains the raw data's native units for export whenever possible, so in most cases no conversion will occur.

    Note: There may be specific circumstances that require data conversion affecting pressure values. For instance, devices may report raw data in unsupported metric units, therefore conversion to supported pressure units may still occur and may involve inH2O units. In such cases, you may notice a slight difference in pressure values using inH2O units.

  • For CygNet versions 8.1.1 or prior, FMS normalized the raw data to inH2O units prior to export. Because of this, you may notice a slight difference in pressure values using inH2O units.

FULL Mode

In cases where raw data is being exported to another measurement system from a CygNet Measurement system in FULL mode, the pressure unit conversion could have the following impact.

  • For any CygNet version, FMS values for the Volume Correction Factor (VCF) will be slightly different for devices that return data in metric units. VCF is a ratio of recalculated volume to device volume, that is stored in the gas device record and is applied when recalculations occur, to ensure that the recalculated volume is consistent with the device’s reported volume.

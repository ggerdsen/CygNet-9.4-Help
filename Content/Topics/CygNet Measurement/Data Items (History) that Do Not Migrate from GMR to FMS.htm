History Data Items that Do Not Migrate from GMR to FMS

When you use FMS Toolbox to migrate GMR history information to FMS using the Migrate History task in the GMR Migration tool, certain data items will not migrate successfully. These items will fail to be imported because they are not supported for FMS.

Select Verbose status messages before you click Migrate to generate messages in the migration results text box if failure occurs. The migration results would then list the migration status as "failed" and explain that the item is not supported.

CygNet FMS Report Control – Properties, Methods, and Events

The following tables list properties, events, and methods for the CygNet FMS Report Control.

  • The first table lists normal properties, events, and ActiveX properties as they appear on the control's Property Sheet.
  • The second table lists normal properties, normal methods, ActiveX properties, and ActiveX methods as they appear in the control's Script Editor.

See CxFmsReportCtrl Object in the Scripting section for additional details about Properties, Methods, and Events specific to this control.

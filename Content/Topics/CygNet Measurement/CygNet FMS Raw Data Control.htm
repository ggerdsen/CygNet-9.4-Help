CygNet FMS Raw Data Control

The CygNet FMS Raw Data Control is an ActiveX control used to view data in its original form for auditing or debugging purposes. Each row represents a record, and each column represents a data field.

See Using the Raw Data Control for more information about configuring and using this control.

See CxFmsRawDeviceDataCtrl Object in the Scripting section for more information about scripting this control.

Adding a Raw Data Control to a Screen

To Add and Configure a CygNet FMS Raw Data Control in CygNet Studio

  1. Create a new CygNet Studio screen.
  2. Click the ActiveX tool on the Standard Tools palette.
  3. Click and drag the outline of a box on the screen. When the mouse button is released, an Insert OLE Control dialog box appears.
  4. In the dialog box, double-click on CygNet FMS Raw Data Control to place a Raw Data control on your screen.
  5. Right-click on the new control to open its property sheet.
  6. On the ActiveX page, click ... or double-click (OCXProperties) to access the CygNet FMS Raw Data Control Properties dialog box.
  7. The CygNet FMS Raw Data control contains a General tab, a Raw data tab, and a Commands tab.
  8. Configure properties on each page as desired, click Apply, and then click OK.
  9. Close the property sheet, save, and then switch to Run mode to use the control.

General Properties

The General page of the CygNet FMS Raw Data Control Properties dialog box contains the following properties.

Property Description
General

FMS service

 Type the name of the Site.Service to which to connect in the control's Chooser, or click ... to access the Select Service dialog box.

Use Point in Time mode

Click the check box to select/clear the option to use the Point in Time mode.

PIT (C)

If Use Point in Time mode is selected, do one of the following.

  • Type the desired date/time value into the text box.
  • Click ... to access the Select Date and Time dialog box, then enter the date/time, in Contract time (C), from which you want to view data.

Default value is T, date and time today.

See Using the Home Menu for more information about PIT mode.

Use Session Node

Click the check box to select/clear the option to Use Session Node instead of using the chooser.

Default value is cleared.

Use Session options

Click the check box to select/clear the option to Use Session options instead of entering a time window and time format.

Default value is cleared.

Show Chooser pane

Click the check box to select/clear the option to display the Chooser pane in the control.

Default value is selected.

Show Options pane

Click the check box to select/clear the option to display the Options pane in the control.

Default value is selected.
Node Chooser

Allow FMS service selection

 Click the check box to select/clear the option to show the FMS service selector in the control's Chooser.

Allow Node category selection

 Click the check box to select/clear the option to show the Node category selector in the control's Chooser.

Category

Select the desired Node Category to display in the control's Chooser from the drop-down menu.

Options are as follows.

  • Device
  • Gas Device
  • Liquid Device

Group

Select the desired Node Group type to display in the control's Chooser from the drop-down menu.

Options are as follows.

  • All Nodes
  • General
  • Station
  • By Category

Filter

Select a filter.

First type a partial string of known Node name/description/ID characters to populate the filter text box in the control's Chooser.

Wildcard values * (multiple character) and ? (single character) are supported.

Default value is * (multiple character wildcard) - all.

Note: Wildcards can be used only when filtering by Name or Desc, not when filtering by ID.

Then select a Node option to filter for results.

Options are as follows.

  • Name
  • Desc
  • ID

Primary column

Select the desired primary sort column in the control's Chooser from the drop-down menu.

Options are as follows.

  • Type ascending
  • Name ascending
  • Description ascending
  • Type descending
  • Name descending
  • Description descending

Default value is Name ascending
Date

Start

Date selections are in Contract days (C).

Enter the starting date (absolute or relative date) in the text box or click ... to access the Select Date dialog box.

See Selecting Date and Time for more information.

End

Date selections are in Contract days (C).

Enter the ending date (absolute or relative date) in the text box or click ... to access the Select Date dialog box.

See Selecting Date and Time for more information.

Format

Select the desired time format from the drop-down menu.

Options are as follows.

  • Device
  • Contract
  • Local

See Using the Raw Data Control for more information about configuring these properties.

Raw Data Properties

The Raw data page of the CygNet FMS Raw Data Control Properties dialog box contains the following properties.

Property Description
Show units

Click the check box to select/clear display of data units in the grid, where applicable.

Default value is cleared (units not shown).
Unit set

Select a Unit set in which to view the data.

Options appear in the drop-down menu, which is populated by the unit sets defined in the service, and includes a default unit set.

Refresh the available unit set list
Columns

Click ... to access the Raw Device Data Control Columns dialog box to select columns for display in the control.

The data fields available to be viewed in the columns correspond to the data view selected, and you can select the sets of columns to appear for each data view.

Options include data items corresponding to the following.

  • Alarm
  • Event
  • Gas Metering Raw
  • Gas Quality Raw
  • (Gas) Config Log Raw
  • Liquid Config Log Raw
  • Liquid Metering Raw
  • Batch Raw
  • Ticket Raw

See Using the Raw Data Control for more information about columns.
Display mode Click to select a Data view mode from the drop-down menu.

Options are as follows.

  • Alarm Entry
  • Config Log Raw
  • Event Entry
  • Gas Quality Raw
  • Metering Raw
  • Batch Raw
  • Ticket Raw
Decimal precision

Click to select a Decimal precision for the display from the drop-down menu.

Options are from 0 - 10 decimal places.

Default value is 7.

Note: Although values are shown in the Raw Data control with a decimal precision of up to 10 places, device Node values are shown in the Batch, Ticket, Liquid Device QTR, and Liquid Device Raw QTR reports with a decimal precision of 15 places.

See Using the Raw Data Control for more information about configuring these properties.

Exporting History Graphs

Use the History Graph control to export history graph data to your clipboard, a file directory location, or a printer.

To Export a History Graph

  1. From the History Graph control, right-click to access the context menu, and then click Export Dialog to access the Exporting dialog box to specify the file type, destination, and size for your graph export.

  2. In the Export box, click a button to select the desired export file type. Options are EMF (Enhanced Metafile Format), WMF (Windows MetaFile format), BMP (Bitmap Image File format), JPG (JPEG - Joint Photographic Experts Group), PNG (Portable Network Graphics), or Text/Data.
  3. In the Export Destination box, click a button to select the destination for your export. Options are Clipboard, File, or Printer (although Text/Data exports do not support the Printer option.)
  4. Do one of the following to complete your selected export.
  1. Export a history graph image file to the clipboard.
  1. In the Export Size box, enter parameter values.
  2. Click Export.
  1. Export a history graph image file to a file directory location.
  1. In the Export Destination box, click Browse to access the Save As dialog box. Navigate to the desired location for the file, and then click Save.
  2. In the Export Size box, enter parameter values.
  3. Click Export.
  1. Export a history graph image file to a printer.
  1. In the Export Size box, enter parameter values.
  2. Click Print to access the Printing Chart dialog box to configure print parameters.
  3. Select the desired values for printer, orientation, paper, printing style, and DPI. Optionally click Setup to configure additional properties for the selected printer and then click OK.
  4. Click OK.
  1. Export a history graph text/data file to the clipboard.
  1. Click Export to access the Export dialog box to configure text/data parameters.
  2. Select the desired options for data subsets and points, data to export, export style, and decimal precision.
  3. Click Export.
  1. Export a history graph text/data file to a file directory location.
  1. In the Export Destination box, click Browse to access the Save As dialog box. Navigate to the desired location for the file, and then click Save.
  2. Click Export to access the Export dialog box to configure text/data parameters.
  3. Select the desired options for data subsets and points, data to export, export style, and decimal precision.
  4. Click Export.

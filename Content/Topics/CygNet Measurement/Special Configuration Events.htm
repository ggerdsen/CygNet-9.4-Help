Special Configuration Events

A Special Configuration Event is returned whenever one of the following special configuration properties is changed on an Emerson ROC or Emerson ROCPlus field device.

  • Primary Calculation Method
  • Primary Calculation Version
  • Heating Value (Volume or Mass)
  • Heating Basic/Units (Mass to Volume, or Volume to Mass)

These events require special handling because the event by itself does not have enough information to be able to record the configuration change, so a little extra processing is required. For example, the event tells us that the heating value changed types, but it does not tell us what the heating value is. Another event change may tell us that the calculation method changed to AGA 3, but it does not tell us what version of AGA 3.

FMS Explorer records and displays these special configuration events in the following way.

  • The Raw Data control (Event Entry data), Category column will display the raw events as a special type Special Configuration Events.
  • The CygNet Measurement system will create a new configuration record for each special event with a distinct timestamp in the event set that was polled. The Configuration control will display the record as a regular configuration record by normalizing the new raw value to the system default unit set.

See FMS Event Items for more information about how these events are handled in the DDS.

Updating FMS Frozen Configuration Items

The Update Frozen Configuration Items task helps you update frozen configuration item states for multiple CygNet Measurement (FMS) Nodes all at once. The following steps describe how to accomplish mass frozen configuration update tasks utilizing the Update Frozen Config task from the Device Data tool in the FMS Toolbox.

You must have proper security authorization to perform frozen configuration item updates. FMS event authorization level 3 is required to complete this process. See FMS Security (and DEVCFG security event) for information about configuring security access for Nodes.

Updating Frozen Configuration Items

The following list provides an overview of the steps to update frozen configuration items for multiple Nodes simultaneously.

  • Select the Nodes - Define the FMS service and Nodes you want to update.
  • Specify the frozen configuration items - Select the specific configuration items you want to update (freeze or unfreeze).
  • Provide audit information - Select an audit category and enter comments to help you track your changes for future reference.
  • Update frozen configuration states - Perform the update and view the results.
  • Optionally repeat the process - Perform additional updates, by repeating the process using a different combination of FMS service, Nodes and/or configuration item selections and value states.

Use the following procedure to perform frozen configuration updates for multiple Nodes.

To Update Frozen Configuration Items for Multiple FMS Nodes

  1. Start CygNet FMS Toolbox, and then click Device Data to access associated tasks.
  2. Click Update Frozen Config to access a series of four Update Frozen Configuration Items pages.
  3. To update frozen status, do the following.
  1. On the first page, select Nodes to update.

  1. Click ... to access the Select FMS Service dialog box, and browse to the Site.Service name containing the Nodes you wish to change. Click OK.
  2. Set the parameters to filter your Node selections and then click Retrieve Nodes.
  3. Results for retrieved Nodes are listed in columns by Node type icon, Node name and Node description. If desired, sort displayed results based on any of the available columns.
  4. Select Nodes to change. You may select multiple Nodes by pressing Ctrl+Click or Shift+Click.
  5. Click Next to continue.
  1. On the second page, select the configuration items you wish to freeze or unfreeze for all of the selected Nodes. Only those items with a selected check box in the "Update?" column will be changed.

  1. Click each check box in the Update? column to select/clear the desired items to freeze/unfreeze.
  2. Click Previous to go back to the Node selection page, or click Next to continue.
  1. On the third page, enter audit information to help track your changes in the system.

  1. Select an Audit category from the drop-down menu options. The menu will be populated based on the FMS Site.Service selected. Audit categories are configured on the Admin menu. See Configuring Audit Categories for more information.
  2. Type Audit comments into the text box. Maximum character count is 140. When you type comments about the configuration updates you are making, they will become part of the audit record. Although comments are optional, they can be helpful when auditing or reviewing records later.
  3. Click Previous to go back to the frozen configuration items page, or click Update to update all of the frozen status configuration items you have selected, for all of the FMS Nodes you have selected.
  1. On the final page, view the update results and optionally choose to repeat the process.

  1. View the results log of your frozen configuration updates in the status box.
  2. Optionally click Restart to return to the first page, to make additional frozen configuration updates using different Nodes, update methods, and/or configuration items or states. Repeat step 2 to make additional updates as desired.

Node Selection Parameters

Set the following parameters to begin filtering Node results and make Node selections for your changes.

Parameter Description

Group

Select a Node Group by group type, or by category.

Options are as follows.

  • <All Nodes>
  • General
  • Station
  • By Category

Node category

Select a Node category.

Options are as follows.

  • <Any - Gas>
  • Group
  • Gas Device
  • Station

Filter

Select a filter.

First type a partial string of known Node name/description/ID characters to populate the filter text box in the control's Chooser.

Wildcard values * (multiple character) and ? (single character) are supported.

Default value is * (multiple character wildcard) - all.

Note: Wildcards can be used only when filtering by Name or Desc, not when filtering by ID.

Then select a Node option to filter for results.

Options are as follows.

  • Name
  • Desc
  • ID

Retrieve Nodes

Click Retrieve Nodes to apply the selected filters and retrieve the results. Results will appear in the Node list.

Node Selection Columns

You can sort the Node list results based on any of the following columns, to help with Node selection.

Column Description

[Icon]

Displays the icons indicating the Node type of the Nodes that meet the selected filters

Node icons are as follows.

 

Gas Meter Device Node
  Station Meter Node
 

Chromatograph Device Node
 

General Group Node

 

Station Group Node

Click on the column header above the icons to sort the display (in ascending or descending order) based on Node type.

Node Name

Lists the names of the Nodes that meet the selected filters

Click on the column header to sort the display (in ascending or descending order) based on Node Name.

Node Desc

Lists the descriptions of the Nodes that meet the selected filters

Click on the column header to sort the display (in ascending or descending order) based on Node Desc.

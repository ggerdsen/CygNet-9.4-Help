GMR Data Items Units

FMS assigns an expected unit set to data coming from GMR. On occasion, GMR data could arrive with some data items using unit values other than what is expected. To adjust for this, you can define and specify an optional unit set override to correct the unit value inconsistencies.

The following EFM (electronic flow measurement) data item units list the expected unit values for the GMR unit set applied to data coming into the CygNet Measurement system from GMR.

GMR Unit Set

Expected GMR units for the following data item types are listed in the tables below.

Unit Set Override

Unless you specify otherwise, FMS will apply the expected unit set to data coming into FMS via a GMR migration or via an FMS Legacy data group, even if it contains items with data units other than what is expected. To allow you to correct for this, FMS provides the option to specify a Unit set override to apply to the incoming GMR migration / FMS Legacy data instead.

The unit set override can be set in the following locations.

FMS Explorer

  • Creating Chromatograph Nodes or Viewing and Editing Chromatograph Properties
  • Creating Gas Meter Nodes or Viewing and Editing Gas Meter Properties
  • Creating Liquid Device Nodes or Viewing and Editing Liquid Device Properties

FMS Toolbox

  • Configuring and Editing Device CSV Files

FMS Scripting

  • FmsDevice Properties

Configuring and Editing CSV Files for FMS Import

Once a .csv file has been created, the next step is to configure and edit the file properly to prepare it for import into CygNet Measurement (FMS). Device and Group .csv files each have specific format requirements for minimum and optional columns. Each .csv file can be created and prepared individually, or you can create file templates for each of your file types. See Creating Import File Templates for more information.

Preparing CSV Files for Import

The following steps help you prepare files for successful import into CygNet Measurement.

  • Configure and edit device .csv files. - Prepare device definition .csv files for import.
  • Configure and edit group .csv files. - Prepare group definition and group entries .csv files for import.

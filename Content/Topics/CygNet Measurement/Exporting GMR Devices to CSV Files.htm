Exporting GMR Devices to CSV Files

The Export Devices task helps you to prepare to migrate GMR devices to FMS via .csv files. The first step of the migration process is to export your GMR devices to a .csv file, using this task.

You must have proper security authorization to export GMR devices.

Exporting GMR Devices

To Export GMR Devices to a CSV File

  1. Start CygNet FMS Toolbox, and then click GMR Migration to access associated tasks.
  2. Click Export Devices to access the Export Devices pane.

  3. Enter information into the following fields.
  1. GMR text box.
    • To export devices from a Site.Service located on the same domain, type the name of the GMR Site.Service from which you wish to export devices, or click ... to access the Select Export GMR dialog box to browse to your selection and then click OK.
    • To export devices from a Site.Service located on a different domain, type [XXXX], where XXXX is the name of the domain, followed by the name of the GMR Site.Service from which you wish to export devices.
  1. Groups box. When you select your GMR service, this box will be populated with all eligible groups. Click to select the GMR group name(s) containing devices you wish to export. You may select multiple groups by pressing Ctrl+Click or Shift+Click. Default selection is <ALL_METERS>.

    Note: A group specified for GMR export will only contain devices from the top level of the selected group. Select your group accordingly.

  2. Definition file text box. Type the file name of the new device definition output file to export, or click ... to access the Select Location to Save Template dialog box to browse to an existing .csv file.

    Note: If you select an existing .csv file, it will be overwritten by the .csv created.

  1. Click Export to generate the .csv file to the specified location.

Preparing Device CSV Files for Import

Once your device .csv file has been created, the next step of the migration process is to configure and edit the file to prepare it for the subsequent step, where you will import the file into FMS.

See Configuring and Editing Device CSV Files for more information.

Sending Gas Quality to a Device from the Gas Analysis Data View

You can use the Gas Analysis data view to send gas quality information from a selected gas analysis record to the device by executing a Send Gas Quality command. The request can be initiated via the Gas analysis data view context menu or via the Node's Gas Analysis Entry property sheet.

Gas analysis records for the selected Node can also be sent via the Send Gas Quality command in other ways. When this is the case, the FMS service does a check and then sends the appropriate records, from gas analysis or gas quality data or both, based on the device configuration at the time. See Send Gas Quality for more information.

You must have proper security authorization to send GQ data from gas analysis information to a device. See FMS Security (and CMDDEV security event) for information about configuring security access.

See Configuring Send Gas Quality Commands for more information about the command that is sent.

See Gas Analysis and Gas Quality Data in the CygNet Measurement Concepts topic for a discussion about gas quality and gas analysis data.

Sending Gas Quality to a Device

Gas quality information contained in a gas analysis record for a selected Node can be sent directly to the device using one of the following procedures.

To Send Gas Quality to a Device from the Gas Analysis Context Menu

  1. Right-click on the gas analysis entry containing the desired gas quality information to access the context menu.
  2. On the context menu, click Send gas quality to device. A dialog box will appear, to verify that you want to send all supported GQ data from the gas analysis record directly to the selected device Node. Click Yes to execute the command.
  3. The FMS Command dialog box will appear, displaying the command status and results. Click in the status row to view detailed results in the Command Results box below. See Viewing Command Logs for information about viewing historical command logs.

To Send Gas Quality to a Device from the Gas Analysis Property Sheet

  1. Right-click on the gas analysis entry containing the desired gas quality information to access the context menu, and then click Edit to access the Edit Gas Analysis Entry dialog box for the selected record.
  1. Optionally click the appropriate tab(s) to access the desired gas analysis property pages, and make any necessary adjustments to the properties. See Creating or Editing Gas Analysis Entries for more information.

    Note: See Gas Analysis Properties for more information about property page content.

  2. Click Send to device to initiate the command to send all supported GQ data from the gas analysis record directly to the device.
  3. The FMS Command dialog box will appear, containing the command status and results. Click in the status row to view detailed results in the Command Results box below. See Viewing Command Logs for information about viewing historical command logs.
  4. The Status field and Send button text will be updated to reflect the results of the send action.

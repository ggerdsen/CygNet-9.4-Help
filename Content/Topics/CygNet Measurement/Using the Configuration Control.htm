Using the Configuration Control

Use the Configuration control to view, create, and edit* device configuration logs or gas analysis records for a single selected gas device Node (gas meter, station meter, or chromatograph) at a time, selected from the filtered Node Chooser results. Liquid device Nodes are not supported. The control helps you quickly review information, distinguish changes between configuration log or gas analysis entries, and optionally send gas quality information to the device. Two data views, configuration and gas analysis, are available for viewing. Gas quality information can be sent to the device from either data view. You can create up to eight instances of this control.

*Note: Editing functionality is only available in systems licensed for FULL mode.

Configuration data for Gas Device Nodes is available for viewing in this control. (Liquid Device Nodes are not supported.) You can view data for one Node at a time, selected from the filtered Node Chooser results.

Each gas device Node in CygNet Measurement is configured with a variety of settings that impact the calculation of its measurement data, depending on the Node type and its specific circumstances. These device configuration details, and any changes made to them, are tracked and audited by the FMS service.

You must have proper security authorization to perform these actions. See FMS Security (and DEVCFG security event) for information about configuring security access for the Configuration control.

See Using the Configuration Data View and Using the Gas Analysis Data View for more information about each data view.

See CygNet Measurement Data Calculation for more information about recalculation implications of editing base condition values (base temperature or base pressure) or heating value basis.

See Gas Analysis and Gas Quality Data in the CygNet Measurement Concepts topic for a discussion about gas quality and gas analysis data.

See FMS Explorer User Interface for more information about the FMS Explorer user interface.

Viewing Configuration Data

Use the Configuration control Chooser pane (unless the Use Session Node option is selected in the Options pane) and Options pane to filter the data to show in the grid. Either pane can optionally be displayed or pinned to the left side.

Chooser Pane

In the embedded Node Chooser for the Configuration control, select the Nodes you want to view data for from the Gas Device (including Station Meter) Node category. Further filter the Nodes by name, description, or ID as desired. See Using the Node Chooser Control for more information.

Options Pane

In the Session section of the Options pane, define the values for the data you would like to view.

Configuration control options are as follows.

Option Description

Session

Use Session Node

Click the check box to select/clear the option to Use Session Node instead of using the chooser.

Notes:

The Use Session Node check box is selected by default for Data controls (Alarm, Configuration, History Graph, History Grid, and Raw Data).

The Use Session Node check box is cleared by default for Summary controls (Dashboard, Exceptions), Reports controls, and Admin controls (Balance, Jobs; not applicable to Close Period, PPA).

Use Session options

Click the check box to select/clear the option to Use Session options instead of entering a time window and time format.

Notes:

The Use Session options check box is selected by default for Data controls (Alarm, Configuration, History Graph, History Grid, and Raw Data).

The Use Session options check box is cleared by default for Summary controls (Dashboard, Exceptions), Reports controls, and Admin controls (PPA, Balance, Jobs; not applicable to Close Period).

Start date

Date selections are in the selected time format.

Select the starting date, unless Use Session options is selected. Type the date into the text box or click ... to Select Date.

See Selecting Date and Time for more information.

End date

Date selections are in the selected time format.

Select the ending date, unless Use Session options is selected. Type the date into the text box or click ... to Select Date.

See Selecting Date and Time for more information.

Time format

Click to select the desired time format from the drop-down menu, unless Use Session options is selected.

Options are as follows.

  • Device
  • Contract
  • Local

Default value is Device time.

Unit set

Select a unit set in which to view the data. Options appear in the drop-down menu, which is populated by the unit sets defined in the service, and includes a default unit set.

Refresh the available unit set list.

Apply

Click to Apply the changes made through the Options pane.

Note: When you click Apply on the Options pane, any Node selection that has been made but not applied on the Chooser pane will also be applied.

Managing Configuration Data

Toolbar

Use the Configuration control toolbar to access options to view and manage configuration logs for the device you have selected.  See Menus and Toolbars for more information.

The following options appear on the toolbar; unavailable buttons appear dimmed.

Option Description

Click to Refresh the control view.

Note: The Refresh button will not take into account changes made through the Options pane that have been selected but not yet applied.

See Refresh Button for more information.

Click to Save data that has been edited.

Edited configuration records are not changed until they are committed using the Save action.

See Creating or Editing Configuration Log Entries or Viewing or Editing Gas Analysis Records for more information.

Click to Cancel uncommitted edits and refresh the control.

Click to Open and view the control as a .csv file.

[The Print button is not available on this control.]

Click to Export the contents of the control to a .csv file.

Data view

Click the Data view drop-down menu to select the type of data you would like displayed, and refresh the grid.

Options are as follows.

  • Configuration
  • Gas analysis

Configure filter

Click to access the Configure Filter dialog box to select the source(s) by which to filter the selected control view.

Configuration data view - Options are as follows.

  • Device - display data that was polled from a device
  • Device file - display data that was imported from a device-generated file
  • Event generated - display data that changed due to an event
  • Imported - display data that was imported from other measurement system files, e.g. migrated from GMR
  • Linked gas quality - display data from linked gas quality sources
  • System generated - display data that was system-generated for non-history records
  • User entered - display data that was user entered or edited, but which did not replace calculated values
  • User file - display data that was imported from user-generated files

Gas analysis data view - Options are as follows.

  • Linked gas quality - display data from linked gas quality sources
  • System generated - display data that was system-generated for non-history records
  • User entered - display data that was user entered or edited, but which did not replace calculated values
  • User file - display data that was imported from user-generated files

Default value is all sources selected.

Use filter

Click the check box to select/clear the Use filter option to filter your data view by the selected source(s).

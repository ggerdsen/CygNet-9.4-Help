Required Device Configuration Data Items for CygNet Measurement

Devices must be configured to contain certain minimum required data items to function successfully in CygNet Measurement. Various device and calculation types have specific configuration data requirements in order to operate with CygNet Measurement to achieve the desired calculation functionality in your system. For instance, devices that were previously configured for Gas Measurement Repository (GMR) systems can be utilized by CygNet Measurement (FMS) systems if they are configured to contain the required "FMS" or "FMS Legacy" data items.

Note: If you are preparing to migrate data from GMR to FMS, certain data items do not migrate. See Configuration Data Items that Do Not Migrate from GMR to FMS and History Data Items that Do Not Migrate from GMR to FMS for more information.

Required Data Items

The terms "output" and "input" used in the sections below are relative to the calculation.

Output Data Items

The following (output) FMS data items will be calculated by the system if not provided by the device. In cases where volume is provided, the system will calculate a volume value for comparison with the device-provided volume and create a VCF value. See VCF for more information.

  • Output FMS data items, by calculation (device) type
  • AGA 3 1985 (Orifice) - Volume, Energy
  • AGA 3 1992 (Orifice) - Volume, Energy
  • AGA 3 2013 (Orifice) - Volume, Energy
  • AGA 7 (Turbine/Ultrasonic) - Volume, Energy
  • AGA 11 (Coriolis) - Volume, Energy, Mass
  • V-Cone/Wafer Cone (Cone) - Volume, Energy

See corresponding data items lists in the EFM Data Items section for more information.

Input Data Items

The following tables list historical and configuration (input) FMS data items that are required to be present in your devices in order to successfully use FMS. Data items that are required to be present, but for which values are optional, are listed as "Optional" in the Input Required? column. The tables are arranged according to calculation/device types, and contain the data item description, data item input requirements, the FMS data item string, the corresponding FMS Legacy (eGMR_...) data item string where applicable, and links to additional helpful information about specific device data items and their possible values.

  • Primary Calculations, by calculation (device) type
  • AGA 3 1985 (Orifice)
  • AGA 3 1992 (Orifice)
  • AGA 3 2013 (Orifice)
  • AGA 7 (Turbine/Ultrasonic)
  • AGA 11 (Coriolis)
  • V-Cone/Wafer Cone (Cone)
  • Compressibility Calculations, by calculation type
  • AGA 8 Detailed Method
  • AGA 8 Gross Methods 1 & 2
  • NX 19 Complete Method
  • NX 19 Gravity/Carbon/Nitrogen (GCN) Method

Using the FMS Explorer User Interface

FMS Explorer is the primary graphical user interface for CygNet Measurement. This interface is capable of displaying a large variety of measurement data simultaneously, in grid and graphical views.

The interface contains several related menus with which to view, edit, and manage electronic flow measurement (EFM) data, and a common status bar for monitoring what is happening in your system. In addition, the application interface makes use of several features that allow you to customize the presentation of data in your system. These features include a customizable view window, a field chooser to help customize data grid views, a historization navigator to allow quick access to sequential views of records, and color configuration opportunities in many areas to help clarify data status at a glance.

See CygNet Measurement Workflow for more information about generic workflow processes in CygNet Measurement.

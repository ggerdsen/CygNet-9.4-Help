Audit Service View

The Audit Service pane in CygNet Explorer lists audit records. Audit records properties are displayed in the resizable Audit Trail Transaction Properties dialog box and cannot be altered aside from editing comments.

You must have permission to edit audit records in order to add or change comments. Comments may be changed or removed by opening the Audit Transaction Trail Properties dialog box again and making edits. Updating the comment for one record will also affect other transactions that are grouped with the record (if any), that is, the other records listed in the properties dialog box.

To Filter Records

  1. Right-click in the AUD service pane, click Select Search from the pop-up menu.
  2. Select the items to be displayed in the AUD service pane, and click OK.

To View Record Properties

  1. In the AUD service pane, locate and right-click the record.
  2. Select Properties from the pop-up menu (or double click the record) to open the Audit Trail Transaction Properties dialog box.

Among the properties shown are the item IDs (including Super Item IDs) and the old and new values of the attribute, if applicable. All transactions considered to be part of the same action are listed.

To Add a Comment to a Record

  1. In the AUD service pane, right-click on the record.
  2. Select Properties from the pop-up menu (or double-click the record).
  3. Click Edit to change to Edit mode.
  4. Type the comment in the Comment dialog box, and click Save.

Adding an Address Record to a Group Record

To Add an Address Record to an Existing Group

  1. Using CygNet Explorer, show the properties of the Group Record.
  2. Click on the Addresses page.
  3. Click New.
  4. In the Type box, select Reference Address.
  5. Click the Address browse button and select the Address Record.
  6. Click OK to save the recipient information.
  7. Click OK to save the notification record.

To Add an Address Record to a New Group

  1. Using CygNet Explorer, right-click in the GNS pane and select New.
  2. Type the ID and Description for the record.
  3. In the Type box, select Group.
  4. Click on the Addresses page.
  5. Click New.
  6. In the Type box, select Reference Address.
  7. Click the Address browse button and select the Address Record.
  8. Click OK to save the recipient information.
  9. Click OK to save the notification record.

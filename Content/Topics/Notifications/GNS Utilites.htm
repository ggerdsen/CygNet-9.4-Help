GNS Utilities

CygNet Software provides the following utilities for managing notifications. All CygNet utility files are found in the CygNet\Utilities folder on your host server.

CygNet Utility Name Utility File Name Description Supported Checks Security

GNS Configuration Tester

GNSConfig.exe

This utility is used to define, encrypt, and test configuration properties for GNS email (Email tab) and voice messaging (VoIP tab) in the GNS configuration file.

  

GNS Responses Editor

ResponsesEditor.exe

This utility is used to modify the standard responses from the GNS voice and WAV address notifications. (For example, a user can add a phrase such as "Press 1 to acknowledge.")

 

Text-to-Speech Registry Changer

GNSTextToSpeechFixRegistry.exe

The CygNet Text-To-Speech Registry Changer is used to grant permission to the CygNet server user to use the Microsoft Text-to-Speech technology on Windows Server 2016 only. It is not required on Windows Server 2019. The utility must be run on the CygNet host server.

Defining an Event Record and Configuring Messages

To Define the Event Record and Configure Messages

  1. Using CygNet Explorer, right-click in the GNS pane and select New.
  2. Type the ID and Description for the record.
  3. In the Type box, select Event.
  4. Click on the Messages page.
  5. In the Set area, type the message to be sent when the alarm is set. Click … to insert tokens into string wherever applicable.
  1. Use the Email Subject field if the recipient is to receive an email message via SMTP.
  2. Use the Message field if a recipient is to receive a message of the following type:
  1. alphanumeric message
  2. email message
  3. voice message via the Dialogic board or SIP Server.
  1. Use the Numeric Msg field if a recipient is to receive a numeric page.
  2. Use the Wave Msg field if a recipient is to be played a WAV file via the Dialogic board or SIP Server.
  1. In the Clear area, type the message to be sent when the alarm is cleared. Click … to insert tokens into string wherever applicable.
  1. Use the Email Subject field if the recipient is to receive an email message via SMTP.
  2. Use the Message field if a recipient is to receive a message of the following type:
  1. alphanumeric message
  2. email message
  3. voice message via the Dialogic board or SIP Server.
  1. Use the Numeric Msg field if a recipient is to receive a numeric page.
  2. Use the Wave Msg field if a recipient is to be played a WAV file via the Dialogic board or SIP Server.

The properties on the Messages page are described below.

Property Description

Set/Clear

Email Subject

Specifies the subject line for email messages. Tokens can be used. See SMTP Mail Messages for more information.

Set/Clear

Message

Specifies the message for alphanumeric notifications. Tokens can be used.

Click Play to preview the message for Voice Address notifications. See Alphanumeric Pager and Voice Address for more information.

Note: Messages are limited to 1800 characters.

Set/Clear

Numeric Msg

Specifies the message for Numeric Pager notifications. See Numeric Pager for more information.

Set/Clear

Wave Msg

Specifies the Wave file(s) for WAV Address notifications. Multiple files can be concatenated using a semicolon. Tokens can be used. See WAV Address for more information.

Send Test Notification

Click Send Test Notification to send a test notification to all recipients. The notification is identified as "test."

A Note About SMS Messages

Some SMS gateway providers limit text messages to 160 characters. Others do not. Using tokens to compose your outgoing notification message may increase the size of the message beyond the 160 character limit. Since the Ack ID is appended to the end of the message, it may be truncated, making acknowledgment difficult.

Configuring Notifications

Setting up a notification is a multi-step process:

  1) Creating a GNS record ID and description.
  2) Determining the record type: Event, Group or Address.
  3) Define the address type (Alphanumeric Pager, COM Port Address, Net Send Address, Numeric Pager, Reference Address, SMTP Mail Message (includes SMS), SNPP Message, Voice Address, or WAV Address) and record of the recipient.
  4) Configure the address of the recipient and any messages handling rules (acknowledgment, resend, blackouts, rule propagation, report alarm clear).
  5) Add the address record to a group record.
  6) Define the event record and configure the notification message.
  7) Associate the event record with the alarm.
  8) Configure the users who are permitted to acknowledge notifications via email. This requires additional security.

Once your notifications are configured, use the GNS queue to manage notifications, including viewing, acknowledging, and deleting notifications.

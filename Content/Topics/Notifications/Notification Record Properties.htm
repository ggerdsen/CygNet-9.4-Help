Notification Record Properties

A GNS record contains three pages:

  • The Notifications page contains general information regarding the record.
  • The Addresses page contains the recipient information.
  • The Messages page contains the messages to be sent. The Messages page is shown only if the record type is Event or Unspecified.

Each page of the GNS record is described below.

Notifications Page

The Notifications page contains general information regarding the record. See Configuring a GNS Record for more information about this page.

Addresses Page

The Addresses page lists the recipient(s) of a notification message. Each recipient can be any notification type. Columns include Type, Address, Address Description, Next to Notify, Retries, Retry Delay, Numeric Delay, Resend Wait Time, Ack Reqd, Report RTN, and Resend If Not Cleared. See Configuring an Address Record for more information about configuring recipients on this page.

Messages Page

The Messages page is displayed only if the record type is Event or Unspecified. The Set area contains the message that is sent when the alarm bit is set. The Clear area contains the message that is sent when the alarm bit is cleared. See Defining an Event Record and Configuring Messages for more information about this page.

Note: The Report Alarm Clear property in an Address record must be enabled for the recipient to receive the Clear message.
